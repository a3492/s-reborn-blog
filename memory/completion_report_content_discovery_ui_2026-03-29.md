## 🎓 학습 내용

### 마주쳤던 문제와 해결

| 문제 | 원인 | 해결 |
|------|------|------|
| 검색 페이지 데이터 미임베딩 | 빌드 스크립트에 치환 로직 미포함 | build-article-pages.cjs에서 JSON.stringify로 직접 임베딩 |
| 클라이언트 사이드 렌더링 성능 | 데이터가 클라이언트로 전송되어야 함 | HTML 파일에 직접 JSON 데이터 포함 (Template literal) |
| 이전/다음 글 네비게이션 | 시리즈 구조가 필요 | 메타데이터의 series + series_order 활용 |

### Best Practices

```
✅ 메타데이터 기반 자동 생성
✅ 정적 파일로 배포 (빠른 로드)
✅ 클라이언트 사이드 렌더링 (백엔드 불필요)
✅ 데이터와 UI 분리 (유지보수 용이)
✅ 자동화 스크립트로 운영 간소화
```
