- **심각도**: 🟡 중간

### [BUG-04] 홈 히어로 카드 — 본문이 한 줄로 연결 ✅ 수정완료
- **심각도**: 🔴 심각
- **위치**: 홈페이지 히어로 카드
- **증상**: 단락이 구분 없이 이어져 가독성 최악

- **수정**: 해당 줄 제거 → `\n{3,}→\n\n` 정규화로 단락 보존. `<p>` 태그 단락 분리 렌더링.
