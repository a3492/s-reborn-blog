## 프로젝트 개요

### 비전
의료 AI에 관심은 있지만 어렵게 느끼는 초보자들을 위한 **진입 장벽 없는 AI 학습 공간** 제공

### 미션
- ✅ AI 개념을 5-7분 안에 이해할 수 있게 단순화
- ✅ 실제 의료 현장에서 바로 써먹을 수 있는 팁 제공
- ✅ AI에 대한 두려움과 오해 해소

### 슬로건
> **"AI, 어렵지 않아요. 우리 함께 시작합시다."**

## 목표 및 KPI

### 정성적 목표
1. **접근성**: 의료진, 학생, AI 초보자도 쉽게 이해할 수 있는 콘텐츠
2. **신뢰성**: 정확하지만 단순한 설명으로 AI에 대한 신뢰 구축
3. **실용성**: 이론만이 아닌 실제 사용 사례 제공

### 정량적 KPI (6개월 기준)
| 지표 | 목표 | 측정 기준 |
|------|------|---------|
| 애린이 글 수 | 50개+ | 카테고리별 글 개수 |
| 초보자 방문 | 월 1,000명+ | Google Analytics |
| 평균 체류 시간 | 3분+ | 페이지 분석 |
| 클릭율 (CTR) | 15%+ | 애린이 섹션 링크 클릭 |
| 공유율 | SNS 100회+/월 | SNS 트래킹 |

## 타겟 오디언스

### Primary (주 타겟)
- **의료 전공자**: 의사, 간호사, 약사, 의료 행정가
  - 특성: AI에 대해 알고 싶지만 시간 부족
  - Pain Point: "너무 어렵고 길다"

- **의료 학생**: 의대생, 간호 학생, 약학 대학생
  - 특성: 호기심 많음, 빠른 학습 원함
  - Pain Point: "기초가 없어서 겁난다"

### Secondary (부 타겟)
- **일반인**: AI에 관심 있는 의료 분야 비종사자
- **정책 입안자**: 의료 AI 규제를 다루는 공무원

## 콘텐츠 전략

### 콘텐츠 3가지 유형

#### Type 1️⃣: 개념 설명 (Concept)
**목표**: "이게 뭐예요?"에 답하기

**특징**:
- 길이: 3-5분 (700-1,000단어)
- 구조: 정의 → 예시 → 의료 활용
- 난이도: ⭐☆☆☆☆ (매우 쉬움)

**예제**:
```
제목: "LLM(거대언어모델)이 뭐예요?"
├─ 한줄 요약: 엄청 많은 텍스트로 학습해서
│            사람처럼 말하는 AI
├─ 예시: ChatGPT, Claude, Gemini
├─ 의료에선: 진료기록 작성, 환자 설명
└─ 주의점: "의사 판단을 완전히 대체 불가"
```

**포함될 글들**:
- Token이 뭐예요?
- Prompt가 뭐예요?
- Embedding이란?
- Hallucination(환각)이란?
- RAG가 뭐예요?
- Fine-tuning과 Pre-training 차이?
- 등 (총 15개 목표)

---

#### Type 2️⃣: 실전 팁 (How-to)
**목표**: "내가 바로 써먹을 수 있다"

**특징**:
- 길이: 4-7분 (1,000-1,500단어)
- 구조: 상황 → 단계별 방법 → 스크린샷/예제 → 결과
- 난이도: ⭐⭐☆☆☆ (쉬움)

**예제**:
```
제목: "ChatGPT로 진료기록 5분만에 작성하는 법"
├─ 상황: 외래 끝나고 기록 100건 남음
├─ Step 1: 음성 녹음하기
├─ Step 2: 프롬프트 입력 (예시 제공)
├─ Step 3: 결과 다듬기
└─ 팁: "절대 그대로 쓰면 안 되고..."
```

**포함될 글들**:
- ChatGPT로 진료기록 작성하기
- Claude로 의학 논문 5분만에 요약하기
- AI로 환자 상담 스크립트 만들기
- 프롬프트 한 줄 바꿔서 결과 향상시키기
- 등 (총 20개 목표)

---

#### Type 3️⃣: 자주 묻는 질문 (FAQ)
**목표**: "불안감 해소"

**특징**:
- 길이: 2-4분 (500-800단어)
- 구조: 질문 → 간단한 답 → 상세 설명 → 의료법 검토
- 난이도: ⭐⭐☆☆☆ (쉬움)

**예제**:
```
Q: "AI를 환자 진료에 쓰면 불법 아닌가요?"
├─ 간단한 답: 아니요, 의사의 판단 보조면 OK
├─ 상세 설명: 의료법 제21조...
├─ 예: OOO 병원의 사례
└─ 주의점: "최종 판단은 의사"
```

**포함될 글들**:
- AI를 환자 진료에 써도 법적으로 괜찮나요?
- AI가 내 일을 빼앗아가나요?
- 의료 AI 환각이 위험한가요?
- 환자한테 AI를 썼다고 말해야 하나요?
- 등 (총 15개 목표)

## 기술 구현 계획

### 옵션 비교

| 항목 | 옵션 A: 새 카테고리 | 옵션 B: 태그 시스템 ⭐ |
|------|-----------------|-------------------|
| **구현 복잡도** | 낮음 | 중간 |
| **유연성** | 낮음 | 높음 |
| **기존 글 재활용** | 불가 | 가능 |
| **관리 난이도** | 쉬움 | 보통 |
| **미래 확장성** | 낮음 | 높음 |
| **선택 이유** | 빠른 출시 | 장기 지속성 |

### 최종 선택: **옵션 B - 태그 + 특수 섹션**

#### 구현 단계

**Phase 1: 콘텐츠 준비 (2주)**
- [ ] 기존 글 중 "초보자 친화적" 글 태그 지정
- [ ] 15개 신규 개념 설명 글 작성
- [ ] 20개 신규 실전 팁 글 작성
- [ ] 15개 신규 FAQ 글 작성

**Phase 2: 프론트엔드 개발 (2주)**
- [ ] 홈페이지에 "🌱 애린이 코너" 섹션 추가

- [ ] 애린이 전용 페이지 `/blog/beginner` 생성
- [ ] 사이드바에 "애린이 추천" 위젯 추가

**Phase 3: 데이터 구조 수정 (1주)**
```yaml

```

## 글 작성 가이드라인

### ✅ 애린이 글 작성 체크리스트

#### 제목 작성
- [ ] 질문 형식으로 (예: "뭐예요?", "뭐가 다르죠?")
- [ ] 애매함 없이 명확한 주제
- [ ] 한국어 평문 (영문 약자 최소화)
  ```
  ✅ "LLM이 뭐예요?"
  ❌ "Large Language Model (LLM) 개념"
  ```

#### 구조 설계
- [ ] 한 글 한 주제 (다중 주제 금지)
- [ ] 3-7분 읽을 분량 (700-1,500 단어)
- [ ] 문장은 15단어 이내 (길 수 있음)

#### 필수 섹션
```markdown
---
title: "..."
difficulty: beginner
reading-time: 3
---

## 한줄 요약
[가장 핵심을 15단어 이내로]

---

## 이게 뭐예요?
[간단한 정의 + 비유 1개]

## 예시
[실제 의료 사례 또는 일상 사례]

## 의료에선 이렇게 써요
[병원/의료현장에서의 실제 활용]

## 주의할 점
[초보자가 놓칠 수 있는 부분]

## 더 깊이 알고 싶다면
[다음 단계로 읽을 글 추천]
```

#### 표현 가이드
| 피해야 할 표현 | 사용할 표현 | 예시 |
|-------------|---------|-----|
| 복잡한 수식 | 비유/시각화 | "계수"를 "손잡이"에 비유 |
| 학술 용어 | 일상 용어 | "거대언어모델" 대신 "엄청 많이 배운 AI" |
| 기술 가정 | 기초 설명 | "API"를 먼저 설명 후 사용 |
| 한번에 여러 개념 | 한 글 한 개념 | 나눠서 작성 |
| 무미건조한 톤 | 친근한 톤 | "🌟", "💡" 이모지 활용 |

#### 이모지 사용 기준
- 🌟: 중요한 개념
- 💡: 팁이나 주의
- ✅: 좋은 사례
- ⚠️: 위험한 사례
- 🎯: 목표/핵심
- 📝: 예시/기록
- ❓: 질문/의문
- 🔗: 다른 글 연결

#### 의료 콘텐츠 검수
- [ ] 의료법 준수 (비의료인 진료 권장 X)
- [ ] 출처 명시 (주장하는 경우)
- [ ] "의사 판단 필수" 명시
- [ ] 윤리 고려 (개인정보, 동의)

## 기대 효과

### 1. 사용자 관점
✅ **접근성 증대**
- AI 배우는 데 드는 시간 90% 단축
- 이해도 증가: 40% → 80%
- 실전 활용 시작: 회피 → 시작

✅ **신뢰 구축**
- "이 블로그는 내 눈높이에 맞춘다"
- 의료 AI에 대한 불안감 → 자신감
- "의사들도 이거 읽는다"는 입소문

### 2. 비즈니스 관점
✅ **트래픽 증가**
- 신규 방문자 유입: 초보자층
- 체류 시간 증가: 페이지뷰 증가
- 검색 순위 개선: SEO 효과

✅ **브랜드 강화**
- "초보자 친화적 의료 AI 블로그" 위치 확립
- 신뢰도 증가: 상담 문의 증가
- 커뮤니티 형성: 충성 독자 확보

### 3. 콘텐츠 관점
✅ **콘텐츠 생태계 확장**
- 기초부터 심화까지 연결된 학습 경로
- 글들 간 내부 링크 증가: SEO ↑
- 콘텐츠 재사용성: 다양한 포맷 가능
