ChatGPT vs Claude — 의사 입장에서 비교

둘 다 써봤습니다. 진료실에서 쓸 AI를 고를 때 실제로 차이를 만드는 부분만 정리합니다.

대상 전공의, 전문의, 개원의 난이도 입문 주제 AI 도구, 선택 기준
핵심 요약: ChatGPT는 범용성·플러그인·이미지 인식에서 앞서고, Claude는 긴 문서 처리·지시 이행·안전한 답변 경향에서 앞선다. 의료 업무별로 맞는 도구가 다르다.

비교 전에 알아야 할 것

ChatGPT(OpenAI)와 Claude(Anthropic)는 둘 다 대형 언어 모델(LLM) 기반 AI 챗봇입니다. 둘 다 유료 플랜이 있으며, 무료 버전도 제공됩니다.

두 도구 모두 실제 환자의 개인정보를 입력해서는 안 됩니다. 특정 가이드라인이나 최신 데이터를 묻기 전에 학습 날짜(knowledge cutoff)를 확인하세요.

직접 써보니

  • 긴 퇴원요약서(3장)를 요약했을 때 — Claude가 더 일관되게 구조를 유지했다
  • ECG 이미지를 붙여넣었을 때 — ChatGPT가 더 구체적인 소견을 언급했다 (단, 검증 필수)
  • "다음 조건을 모두 지켜서 써줘" 같은 다단계 지시 — Claude가 조건을 더 잘 지켰다
  • 최신 가이드라인(2025년 발표) 내용 — ChatGPT 웹 검색 연동이 더 유용했다
두 도구 모두 의료 정보에 대해 '그럴듯한 오류'(환각)를 낼 수 있습니다. 용량, 금기사항, 최신 가이드라인은 반드시 원문 확인 후 쓰세요.

처음 시작하는 방법

  1. ChatGPT: chat.openai.com — 구글 계정으로 가입, 무료 플랜으로 시작
  2. Claude: claude.ai — 이메일 가입, 무료 플랜으로 시작
  3. 처음에는 민감하지 않은 문서(익명화된 예시, 가이드라인 텍스트)로 먼저 연습
  4. 프롬프트는 이전 편(Prompts 1편)을 참고: 임상 프롬프트 입문 →
