## 한줄 요약

논문 전문을 Claude에 붙여넣고 질문하면,
5분 만에 "이 논문이 왜 중요한지"를 파악할 수 있어요.

---

## 당신의 상황

🎯 학회 발표가 다음 주인데, 읽어야 할 논문이 10편입니다.

전부 영어이고, 통계 부분은 이해하기도 벅차요.

💡 Claude를 쓰면 논문 1편당 5-10분이면 충분해요.

---

## 이렇게 하면 5분에 끝나요

### Step 1: Claude 접속
claude.ai 접속 → 무료 계정 생성 (또는 ChatGPT도 가능)

💡 Claude가 긴 텍스트 처리에 특히 강해요.
무료 버전 Claude Sonnet도 충분합니다.

---

### Step 2: 논문 텍스트 복사

PDF 논문을 열고:
1. Ctrl+A (전체 선택)
2. Ctrl+C (복사)
3. Claude 입력창에 붙여넣기

또는 Abstract + Methods + Results만 복사해도 충분해요.

---

### Step 3: 핵심 질문 프롬프트

```
다음 의학 논문을 읽고 아래 항목을 한국어로 요약해줘.

1. 연구 목적 (한 문장)
2. 연구 방법 (환자 수, 설계, 기간)
3. 주요 결과 (수치 포함, 3줄 이내)
4. 결론 및 임상적 의의
5. 한계점
6. 내 진료에 어떻게 적용할 수 있는지

[논문 내용 붙여넣기]
```

---

### Step 4: 결과 확인

Claude 출력 예시:
```
1. 목적: 제2형 당뇨 환자에서 GLP-1 제제의
   심혈관 예후 개선 효과를 RCT로 평가

2. 방법: 고위험 당뇨 환자 9,340명,
   평균 추적 기간 3.8년, 이중 맹검

3. 결과:
   - MACE 발생: 위약군 14.9% vs 치료군 11.4%
   - 상대 위험 감소 23% (HR 0.74, p<0.001)
   - 심부전 입원 32% 감소

4. 결론: GLP-1 제제가 심혈관 고위험 당뇨 환자에서
   MACE를 유의하게 줄임. 심혈관 위험이 높은
   당뇨 환자에서 1차 약제로 고려 권장.

5. 한계: 단일 약제 비교, 특정 인종 과대표

6. 적용: 심혈관 기저질환 동반 당뇨 환자에서
   GLP-1 제제를 초기부터 적극 고려.
```

## 더 잘 쓰는 팁

### 💡 팁 1: 통계가 이해 안 될 때

```
이 논문의 통계 방법 (HR, CI, NNT 등)을
의사 아닌 사람도 이해할 수 있게 쉽게 설명해줘.
```

### 💡 팁 2: 비판적으로 읽기

```
이 논문의 약점과 비판할 수 있는 부분을 3가지만 알려줘.
```

### 💡 팁 3: 다른 논문과 비교

```
이 논문의 결과와 기존 메타분석 결과가
어떻게 다른지 비교해서 설명해줘.
```

### 💡 팁 4: 발표 준비

```
이 논문을 5분 저널 클럽에서 발표한다면
슬라이드 구성을 어떻게 하면 좋을지 제안해줘.
```

## 실수하기 쉬운 부분

⚠️ 실수 1: AI 요약을 논문 내용으로 인용하면 안 돼요
AI가 잘못 요약할 수 있어요.
인용할 때는 원문을 직접 확인하세요.

⚠️ 실수 2: 최신 논문은 AI가 모를 수 있어요
AI 학습 시점 이후 발표된 논문은
내용을 모르거나 엉뚱하게 답할 수 있어요.
반드시 논문 원문을 직접 붙여넣으세요.

⚠️ 실수 3: 수치 확인은 필수예요
결과의 수치(HR, p값, 비율)는 AI가 틀릴 수 있어요.
중요한 수치는 원문과 대조하세요.
