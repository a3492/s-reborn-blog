## 한줄 요약

NLP(자연어처리)는 컴퓨터가 사람이 쓰고 말하는 언어를 이해하고 분석하는 기술이에요. ChatGPT부터 EMR 자동 분류까지 모두 NLP가 기반이에요.

---

## 왜 알아야 할까?

- 💡 ChatGPT, Claude 같은 AI의 근본 기술이에요
- 💡 EMR 비정형 텍스트를 자동으로 분석하는 원리를 이해해요
- 💡 의료 빅데이터 연구에서 NLP 활용이 빠르게 늘고 있어요
- 💡 의무기록에서 의미 있는 정보를 자동 추출하는 기반이에요

## 이게 뭐예요?

NLP(Natural Language Processing, 자연어처리)는 컴퓨터가 인간의 언어(자연어)를 처리하는 AI 기술이에요. '자연어'는 프로그래밍 언어처럼 규칙적이지 않고, 사람이 자연스럽게 쓰는 언어를 말해요.

NLP는 언어를 이해하는 것부터 생성하는 것까지 폭넓은 기술을 포함해요. 번역, 요약, 감정 분석, 텍스트 분류, 질의응답 등이 모두 NLP의 범주예요.

🏥 비유 1: 의무기록 텍스트에서 진단명 자동 추출

수천 건의 퇴원 요약지에서 "당뇨병 환자"를 모두 찾으려면 예전엔 사람이 일일이 읽어야 했어요. NLP는 "당뇨", "DM", "Diabetes", "혈당 조절 불량" 같은 다양한 표현을 자동으로 인식하고 추출해줘요.

🏥 비유 2: 외국어 의학 논문 자동 번역

영어로 된 최신 가이드라인을 읽기 위해 사전을 찾아가며 번역하는 건 시간이 너무 걸려요. NLP 기반 번역 AI는 의학 용어까지 정확하게 번역해줘요. Google 번역도, DeepL도 NLP 기술이에요.

한줄 정리: NLP = 컴퓨터가 사람 말을 이해하는 기술. 의료 텍스트 분석의 핵심이에요.

## 예시를 들어볼게

예시 1 — 감정 분석

> 환자 리뷰: "오래 기다렸지만 의사 선생님이 친절하게 설명해줘서 만족했어요"
> NLP 분석: 불만 요소(대기 시간) + 만족 요소(친절도) 자동 분류

대량의 환자 피드백을 자동 분석할 수 있어요.

예시 2 — 정보 추출

> EMR 텍스트: "환자는 3일 전부터 38.5도 발열과 기침이 있었으며 PCR 검사에서 양성 반응"
> NLP 추출: 증상(발열 38.5도, 기침), 기간(3일), 검사결과(PCR 양성)

비정형 텍스트에서 구조화된 데이터를 자동으로 만들어요.

예시 3 — 의료 QA 시스템

> 질문: "혈압약 복용 중에 자몽을 먹으면 안 되는 이유가 뭐예요?"
> NLP 기반 시스템: 의학 문헌에서 관련 내용 검색 → 이해하기 쉽게 답변 생성

질문의 의도를 파악하고 관련 정보를 찾아 답변해요.

## 의료에서는 이렇게 써요

활용 1: EMR 비정형 데이터 구조화

❌ 기존 방식: 의사가 자유롭게 작성한 메모는 검색·분석 불가

✅ NLP 적용: "흉통 호소", "chest pain", "가슴 답답함" → 모두 같은 증상으로 자동 분류

수십만 건 EMR에서 연구 대상 환자를 자동으로 선별할 수 있어요.

활용 2: 임상 의사결정 지원

❌ 키워드 검색: 정확한 단어 일치만 검색 가능

✅ NLP 기반 검색: 증상 맥락을 파악해서 관련 진료지침 자동 추천

의사가 진료 중 실시간으로 필요한 정보를 얻을 수 있어요.

활용 3: 퇴원 요약 자동 생성

❌ 기존 방식: 입원 기간 모든 기록을 직접 읽고 요약 작성 → 시간 과다 소요

✅ NLP 자동화: EMR 입원 기록 자동 분석 → 퇴원 요약 초안 생성

의사가 검토·수정만 하면 되어서 문서 작업 시간이 줄어요.

## 주의할 점 ⚠️

⚠️ 의료 NLP는 일반 NLP보다 어렵고 복잡해요

의료 용어는 약어가 많고, 맥락에 따라 의미가 달라요. ("Discharge"는 퇴원일 수도, 분비물일 수도 있어요.) 일반 NLP 모델을 의료에 그대로 쓰면 오류가 많아요.

⚠️ 부정문 처리가 중요해요

"두통 없음"과 "두통 있음"을 잘못 분류하면 큰 문제예요. NLP는 부정문, 가정문, 과거 이력 등 맥락을 정확히 처리해야 해요.

⚠️ 언어별 성능 차이가 커요

영어 의료 NLP 모델은 발전했지만, 한국어 의료 NLP는 아직 발전 중이에요. 한국어 EMR에 영어 기반 모델을 적용하면 성능이 크게 떨어져요.
