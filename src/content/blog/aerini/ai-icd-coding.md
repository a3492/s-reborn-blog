## 한줄 요약
진단명이나 증상을 AI에 입력하면 ICD-10 코드 후보를 주진단·관련 코드로 나눠서 받을 수 있습니다.
최종 코딩은 공식 코드북이나 의료 코딩 전문가로 반드시 확인하세요.

---

## 당신의 상황
🎯 퇴원 코딩을 해야 하는데, 적절한 ICD-10 코드가 바로 생각나지 않습니다. 코드북을 뒤지면 시간이 걸리고, 비슷한 코드가 여러 개라 어떤 걸 써야 할지 헷갈립니다.

💡 AI에 진단명을 주면 주진단 코드와 관련 코드 후보를 한 번에 정리해 줍니다. 검색의 시작점을 잡는 데 유용합니다.

## 이렇게 하면 됩니다

### Step 1: 진단명 또는 증상 정리하기
코딩할 진단명을 정리합니다. 영문이나 한국어 모두 괜찮습니다.

```
예시:
- 주진단: 지역사회 획득 폐렴, 원인균 불명
- 동반 진단: 2형 당뇨병 (인슐린 비사용), 고혈압
- 시술: 기관지 내시경 (진단 목적)
```

### Step 2: AI에 프롬프트 입력하기
주진단과 동반 진단을 구분해서 주면 더 정확한 결과가 나옵니다.

```
다음 진단에 적합한 ICD-10 코드를 알려줘.
주진단 코드와 관련(동반) 코드를 구분해서 정리해줘.
비슷한 코드가 여러 개라면 차이점도 설명해줘.

진단 목록:
- 주진단: 지역사회 획득 폐렴, 원인균 불명
- 동반: 2형 당뇨병 (인슐린 비사용), 고혈압
- 시술: 기관지 내시경 (진단 목적)
```

### Step 3: 결과 확인하기
AI가 제안하는 코드 예시입니다.

```
주진단:
- J18.9 — 상세불명의 폐렴
  (J18.1 엽성 폐렴, J18.0 기관지폐렴과 혼동 주의)

동반 진단:
- E11.9 — 합병증 없는 인슐린 비의존성 당뇨병
- I10 — 본태성(원발성) 고혈압

시술:
- 0BJ08ZZ — 기관지 내시경, 진단 목적 (ICD-10-PCS 기준)

유사 코드 차이:
- J18.9 vs J18.1: J18.9는 폐엽 분포가 불명확할 때 사용.
  흉부 X선에서 특정 엽에 침윤이 명확하면 J18.1 고려.
```

### Step 4 (선택): 특정 코드 상세 확인하기
코드 선택에 확신이 없을 때 추가 질문을 하세요.

```
J18.9와 J18.1 중 어떤 상황에서 각각을 선택해야 하는지
임상 기준을 설명해줘.
```

## 더 잘 쓰는 팁
💡 **팁 1**: "유사 코드와 차이점도 설명해줘"라고 추가하면 헷갈리는 코드 구분에 큰 도움이 됩니다.
💡 **팁 2**: 한국에서는 ICD-10이 기본이지만, 기관에 따라 KCD(한국 표준 질병 사인 분류)를 사용합니다. 프롬프트에 "KCD 기준으로"라고 명시하세요.
💡 **팁 3**: AI가 제안한 코드를 공식 WHO ICD-10 브라우저(icd.who.int)에서 빠르게 검증할 수 있습니다.

## 실수하기 쉬운 부분
⚠️ **실수 1**: AI가 존재하지 않는 코드를 만들어 냄 → 해결: 제안된 코드는 반드시 공식 코드북이나 ICD-10 브라우저에서 직접 확인하세요.
⚠️ **실수 2**: 주진단과 동반 진단 코드를 혼동 → 해결: 프롬프트에서 명확히 구분해 입력하고 결과도 구분해서 받으세요.

## 다른 방법도 있어요
- **ICD-10 공식 브라우저**: icd.who.int에서 검색어로 코드를 직접 찾을 수 있습니다. AI 결과 검증용으로 사용하세요.
- **건강보험심사평가원 의약품 및 치료재료 코드 조회**: 한국 보험 청구용 코드는 심평원 포털에서 직접 확인하는 게 가장 정확합니다.
