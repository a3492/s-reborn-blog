AI를 가르치는 방법은 여러 가지입니다. 의대 교육에도 강의, 실습, 피드백이 있듯이 AI도 학습 방식이 다양합니다. 핵심 용어를 의대 교육에 비유해서 살펴봅시다.

## AI 학습 방법 핵심 용어

| 용어 | 설명 | 의대 교육 비유 |
|------|------|--------------|
| Supervised Learning (지도학습) | 정답 레이블이 있는 데이터로 학습 | 정답지 있는 문제집 풀기 |
| Unsupervised Learning (비지도학습) | 정답 없이 패턴을 스스로 찾아 학습 | 교수 없이 증례 스스로 분류하기 |
| Reinforcement Learning (강화학습) | 보상·벌칙을 통해 행동을 최적화 | 시험 점수 피드백으로 공부 방향 수정 |
| Fine-tuning (파인튜닝) | 기존 모델을 특정 분야에 맞게 추가 학습 | 의대 졸업 후 전공의 수련 |
| Transfer Learning (전이학습) | 다른 분야에서 학습한 지식을 활용 | 기초과학 지식을 임상에 적용 |
| Data Augmentation (데이터 증강) | 기존 데이터를 변형해 학습 데이터 늘리기 | 같은 X-ray를 회전·밝기 조정해 여러 장처럼 활용 |
| Regularization (정규화) | 과적합을 막기 위한 학습 제약 기법 | 너무 한 케이스에 집중하지 말라는 교수의 조언 |

## 의료 AI에서 학습 방법 적용 예시

| 학습 방법 | 의료 AI 활용 사례 |
|----------|----------------|
| 지도학습 | 영상 AI: X-ray에 "폐렴/정상" 레이블을 달아 학습 |
| 비지도학습 | 환자 군집 분석: 비슷한 임상 패턴끼리 자동 분류 |
| 강화학습 | 치료 최적화: 결과가 좋은 처방 전략을 강화 |
| 파인튜닝 | 범용 GPT-4를 방사선 보고서용으로 추가 학습 |
| 전이학습 | 자연어 이해 능력을 의료 기록 분석에 전용 |
| 데이터 증강 | 희귀 질환 케이스 부족 시 데이터 다양화 |

## 왜 파인튜닝이 중요한가

범용 AI는 의학을 "알긴" 하지만 전문의 수준은 아닙니다. 의대 졸업생이 모든 과를 두루 아는 것과 같습니다. 파인튜닝은 이 졸업생을 흉부외과 전문의로 특화 훈련시키는 과정입니다.

✅ 파인튜닝 후 → 방사선 보고서 요약이 훨씬 정확해집니다.

❌ 파인튜닝 없이 → 범용 모델은 의료 특수 용어나 형식을 잘 모릅니다.

⚠️ 파인튜닝도 잘못된 데이터로 하면 오히려 성능이 떨어집니다.
