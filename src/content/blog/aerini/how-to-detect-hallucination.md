## 한줄 요약

AI 결과를 믿기 전, 수치·출처·논문을 직접 확인하세요.
'모르면 모른다고 해줘'를 프롬프트에 항상 추가하세요.

---

## 왜 이게 중요한가요?

🎯 AI 환각의 위험성을 과소평가하기 쉬워요.

답변이 너무 자연스럽고 자신 있어 보이기 때문이에요.
마치 경험 많은 선배가 확신에 차서 말하는 것처럼요.

💡 이 5가지 방법으로 환각을 80% 이상 걸러낼 수 있어요.

---

## 방법 1: '모른다고 해줘' 추가하기

모든 프롬프트 뒤에 이 문장을 붙이세요:

```
확실하지 않은 내용은 '확실하지 않음'이라고 표시해줘.
모르는 내용은 모른다고 말해줘.
추측이면 '추측'이라고 표시해줘.
```

→ AI가 아는 척을 덜 하게 돼요.

예시 적용:
```
다음 약물의 신기능 저하 시 용량 조절 방법을 알려줘.
확실하지 않은 경우 반드시 표시하고,
확인이 필요한 내용은 어디서 찾아봐야 하는지 말해줘.
```

---

## 방법 2: 출처를 요구하기

```
이 내용의 출처(학회명, 가이드라인명, 발표 연도)를
함께 제시해줘.
```

→ 출처를 대야 하니 지어내기 어려워요.

⚠️ 단, AI가 제시한 출처도 100% 신뢰하면 안 돼요.
PubMed나 학회 홈페이지에서 실제 존재하는지 확인하세요.

---

## 방법 3: 수치는 항상 원문 확인

AI가 제시하는 수치 — 특히 이것은 꼭 확인하세요:

⚠️ 약 용량: 용량 오류는 치명적
⚠️ 금식 시간: 수술 전후 중요
⚠️ 검사 정상 범위: 병원마다, 검사실마다 다름
⚠️ 가이드라인 수치: 업데이트됐을 수 있음

확인 방법:
- 의약품: 식약처 의약품 정보 사이트
- 가이드라인: 해당 학회 공식 홈페이지
- 검사 수치: 우리 병원 검사실 기준값

---

## 방법 4: 같은 질문을 다시 물어보기

AI에게 같은 질문을 다시 하면 다른 답이 나올 수 있어요.

두 답변이 다르다 → 어느 쪽이 맞는지 모름 → 확인 필요

또는:
```
방금 답변한 내용에서 틀릴 가능성이 있는 부분이 있나요?
스스로 검토해줘.
```

→ AI 스스로 오류를 잡아내기도 해요.

---

## 방법 5: 의심스러운 논문은 PubMed 확인

AI가 논문을 인용하면:

확인 절차:
1. PubMed (pubmed.ncbi.nlm.nih.gov) 접속
2. 저자명 + 연도로 검색
3. 논문이 실제 존재하는지 확인
4. 인용된 내용이 실제 논문 내용과 일치하는지 확인

30초면 됩니다.

---

## 의료에서 특히 조심할 상황

### 절대 AI만 믿으면 안 되는 상황

🚨 처음 보는 희귀 질환
AI의 학습 데이터가 부족할 수 있어요.

🚨 최신 치료 프로토콜
AI 학습 시점 이후 변경된 내용은 모를 수 있어요.

🚨 환자 개인 특수 상황
알레르기, 다약제 복용, 임신 등 복합 상황에선 더 조심하세요.

🚨 자신 있어 보이는 수치
오히려 더 의심하세요. 환각일 때 더 확신에 차 보이는 경우가 있어요.

---

## 핵심 체크리스트

진료에 AI 답변을 쓰기 전에:

- [ ] 약 용량/수치를 공식 자료로 확인했나?
- [ ] 인용된 논문이 PubMed에 실제로 있나?
- [ ] 최신 가이드라인과 일치하나? (학회 사이트 확인)
- [ ] AI가 "확실하지 않다"고 표시한 부분은 없나?
- [ ] 중요한 결정이라면 다른 출처로 한 번 더 확인했나?
