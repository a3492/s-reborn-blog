## 한줄 요약

API는 서로 다른 소프트웨어가 정해진 방식으로 기능을 주고받게 해주는 연결 인터페이스입니다.
ChatGPT API를 사용하면 병원 EMR 안에서 직접 AI 기능을 구현할 수 있습니다.

---

## 왜 알아야 할까?

🎯 **Pain Point**
ChatGPT는 편리하지만, 매번 브라우저를 열어서 복붙하는 건 번거롭습니다. 우리 병원 시스템 안에서 바로 AI를 쓸 수는 없을까요?

✅ **API를 알면**
- AI를 기존 병원 시스템에 통합하는 방법을 이해할 수 있습니다
- 개발자와 의미 있는 대화를 할 수 있습니다
- 어떤 AI 도구가 API를 제공하는지 파악할 수 있습니다
- 병원 디지털 헬스케어 프로젝트에서 역할을 할 수 있습니다

---

## 이게 뭐예요?

**API(Application Programming Interface)**는 프로그램들이 서로 소통하는 방법을 정의한 약속입니다.

A 시스템이 B 시스템에게 "이런 형식으로 요청하면, 이런 형식으로 응답해줄게"라고 규칙을 정해놓은 것입니다.

**비유 1 — 병원 간 의뢰 시스템**
1차 의원에서 종합병원으로 환자를 의뢰할 때, 정해진 양식에 맞게 의뢰서를 씁니다. 진단명, 검사 결과, 현재 투약 정보를 정해진 형식에 맞춰 전달하면, 종합병원이 그 정보를 받아 처리합니다. API도 마찬가지입니다. 정해진 형식으로 요청을 보내면, 정해진 형식으로 응답이 돌아옵니다.

**비유 2 — 혈액검사 기기와 EMR 연동**
혈액검사 기기가 결과를 자동으로 EMR에 입력하는 것이 API 연동의 좋은 예입니다. 검사기기(A 시스템)가 EMR(B 시스템)에 정해진 형식으로 데이터를 전송합니다. 의사가 수동 입력하지 않아도 결과가 자동으로 나타납니다.

💡 **한줄 정리**
"서로 다른 시스템이 대화하는 방법을 정해놓은 규칙"

---

## 예시를 들어볼게

**예시 1 — ChatGPT API 기본 사용**
개발자가 ChatGPT API를 호출하면 이렇게 됩니다. "환자 A는 당뇨 10년, 고혈압 5년, 현재 복용 약은..."라는 텍스트를 보내면, ChatGPT가 "주요 약물 상호작용 위험 항목은..."이라는 답변을 돌려줍니다. 이 과정이 EMR 화면 안에서 버튼 하나로 동작합니다.

**예시 2 — 날씨 API**
날씨 앱이 직접 기상 데이터를 수집하지 않습니다. 기상청 API에 "서울 내일 날씨 주세요"라고 요청하면, 기상청 서버가 데이터를 돌려줍니다. API 덕분에 모든 앱이 기상청 데이터를 활용할 수 있습니다.

**예시 3 — 의료 AI API 통합**
방사선 AI 회사가 흉부 X-ray 분석 API를 제공합니다. 병원 PACS(의료영상저장전송시스템)가 영상을 API로 보내면, 분석 결과(폐결절 위치, 크기, 의심도)가 자동으로 판독문 초안에 추가됩니다.

---

## 의료에서는 이렇게 써요

**1. EMR에 AI 기능 추가**
❌ 기존 방식: 의사가 ChatGPT 웹사이트를 열고, 정보를 복붙하고, 결과를 다시 EMR에 입력
✅ API 연동: EMR 화면 안에서 버튼 하나로 AI 초안 생성 → 의사가 확인 후 저장

**2. 자동 퇴원 요약 생성**
입원 기간 데이터(진단, 처방, 검사 결과)를 AI API로 보내면, 구조화된 퇴원 요약 초안이 자동 생성됩니다. 의사가 검토·수정 후 확정합니다.

**3. 의약품 상호작용 API**
처방 시 약물 DB API를 실시간으로 호출해서 상호작용을 자동 체크합니다. 의사가 처방을 입력하는 순간 경고가 뜨는 시스템이 이 방식으로 작동합니다.

---

## 주의할 점 ⚠️

**1. 환자 데이터를 외부 API에 보낼 때 보안 검토가 필요합니다**
ChatGPT API에 환자 정보를 그대로 보내면 개인정보 문제가 생길 수 있습니다. 비식별화 처리나 온프레미스(원내 서버) 배포를 검토해야 합니다.

**2. API 비용이 발생합니다**
ChatGPT API는 사용량에 따라 비용이 청구됩니다. 사용 빈도와 데이터 크기에 따른 비용 계획이 필요합니다.

**3. API 버전 관리가 필요합니다**
AI 회사가 API를 업데이트하면 기존 연동이 깨질 수 있습니다. 지속적인 유지보수 체계가 있어야 합니다.
