## 한줄 요약
환자 핵심 정보를 붙여 넣으면 AI가 퇴원 요약지 형식의 초안을 5분 안에 만들어 줍니다.
수치와 날짜는 반드시 직접 검토 후 서명하세요.

---

## 당신의 상황
🎯 퇴원 요약지 하나 쓰는 데 30분이 걸립니다. 입원 경과, 약물, 외래 계획까지 일일이 정리하다 보면 밀린 다른 업무가 쌓입니다.

💡 AI에 핵심 정보만 던져 주면 형식을 갖춘 초안을 뽑아 줍니다. 빈칸 채우기보다 교정하는 게 훨씬 빠릅니다.

## 이렇게 하면 됩니다

### Step 1: 핵심 정보 메모하기
EMR을 보면서 아래 항목을 간단히 메모합니다. 문장 형태가 아니어도 됩니다.

```
예시 메모:
- 환자: 68세 남성
- 진단: 지역사회 획득 폐렴 (CAP)
- 입원 기간: 2026-03-24 ~ 2026-03-31
- 경과: 입원 시 발열 39도·CRP 15. 항생제 치료 후 3일째 해열.
         흉부 X선 호전 확인. 경구 항생제로 전환 후 퇴원.
- 치료: ceftriaxone → amoxicillin/clavulanate 전환
- 퇴원 약물: 아목시실린/클라불라네이트 625mg 하루 3회 5일, 아세트아미노펜 500mg 필요 시
- 외래 계획: 1주 후 외래, 흉부 X선 추적
```

### Step 2: AI에 프롬프트 입력하기
형식을 명확히 지정하면 바로 쓸 수 있는 초안이 나옵니다.

```
다음 정보로 퇴원 요약지를 작성해줘.
형식: 진단명 / 입원 경과 / 시행 치료 / 퇴원 약물 / 외래 계획
각 항목은 3~5문장 이내로 간결하게. 한국어 공식 문서 문체로.

환자 정보:
- 68세 남성
- 진단: 지역사회 획득 폐렴 (CAP)
- 입원: 2026-03-24 ~ 2026-03-31
- 경과: 입원 시 발열 39도·CRP 15. 항생제 치료 후 3일째 해열.
         흉부 X선 호전. 경구 전환 후 퇴원.
- 치료: ceftriaxone → amoxicillin/clavulanate
- 퇴원 약: 아목시실린/클라불라네이트 625mg tid 5일, 아세트아미노펜 500mg prn
- 외래: 1주 후, 흉부 X선 추적
```

### Step 3: 초안 검토하고 수정하기
AI가 생성한 초안을 EMR에 붙여 넣고 다음 항목을 직접 확인합니다.

```
체크리스트:
[ ] 날짜가 실제 입원 기간과 일치하는가?
[ ] 약물 용량과 복용 기간이 정확한가?
[ ] 외래 날짜나 추적 계획이 처방과 일치하는가?
[ ] 누락된 주요 사건(시술, 합병증)이 없는가?
```

### Step 4 (선택): 특정 섹션 보강 요청하기
입원 경과가 복잡했다면 해당 섹션만 따로 요청하세요.

```
입원 경과 부분을 조금 더 자세하게 다시 써줘.
항생제 변경 이유와 시점을 명확히 포함해줘.
```

## 더 잘 쓰는 팁
💡 **팁 1**: 메모가 짧아도 됩니다. AI는 구조를 잡아주는 역할을 하고, 내용은 당신이 검토합니다.
💡 **팁 2**: "공식 문서 문체로"라고 명시하면 구어체 없이 깔끔한 초안이 나옵니다.
💡 **팁 3**: 같은 과에서 반복적으로 쓰는 진단이 있다면 프롬프트 템플릿을 메모장에 저장해 두세요.

## 실수하기 쉬운 부분
⚠️ **실수 1**: AI가 날짜를 만들어 냄 → 해결: 날짜는 프롬프트에 직접 명시하고 결과물에서 반드시 검토하세요.
⚠️ **실수 2**: 약물 용량을 AI가 임의로 변경 → 해결: 약물명과 용량은 메모에 정확히 써주고, 결과물을 처방과 대조하세요.

## 다른 방법도 있어요
- **음성 받아쓰기 + AI 정리**: 회진 중 음성으로 메모한 뒤 AI에 넘기면 더 빠르게 초안을 만들 수 있습니다.
- **병원 EMR 자동완성 기능**: 일부 EMR에는 이전 요약지 템플릿이 내장되어 있습니다. AI와 병행하면 효율이 올라갑니다.
