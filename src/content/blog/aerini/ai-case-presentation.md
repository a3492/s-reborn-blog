## 한줄 요약
케이스 정보를 AI에 주면 발표 구조, 핵심 학습 포인트, 예상 질문까지 한번에 받을 수 있습니다.
실제 발표 전에는 반드시 충분히 검토하세요.

---

## 당신의 상황
🎯 내일 컨퍼런스 케이스 발표인데 준비 시간이 2시간도 안 됩니다. 어떤 순서로 발표해야 할지, 어떤 포인트를 강조해야 할지 막막합니다.

💡 AI에 케이스 요약을 주면 발표 구조를 잡아주고, 질의응답 예상 질문까지 미리 준비할 수 있게 도와줍니다.

## 이렇게 하면 됩니다

### Step 1: 케이스 핵심 정보 정리하기
발표에 필요한 정보를 간단히 적습니다. 완벽한 문장이 아니어도 됩니다.

```
예시 케이스 메모:
- 환자: 45세 남성, 비흡연자
- 주소: 3주간 지속되는 건성 기침, 호흡곤란 점진 악화
- 검사: 흉부 X선 양측 간질성 침윤, 산소포화도 88%
- LDH 상승, β-D-glucan 양성
- HIV 검사 양성, CD4 45/μL
- 기관지 폐포 세척액(BAL): Pneumocystis jirovecii 확인
- 치료: TMP-SMX 고용량, prednisolone 병용
- 경과: 치료 5일째 점진적 호전
```

### Step 2: AI에 프롬프트 입력하기
발표 시간과 청중 수준을 명시하면 더 맞춤화된 구성이 나옵니다.

```
다음 케이스를 5분 컨퍼런스 발표 형식으로 구성해줘.
청중은 내과 레지던트와 전임의.
다음을 포함해줘:
1. 발표 순서 (각 섹션별 권장 시간)
2. 각 섹션에서 강조할 핵심 포인트
3. 핵심 학습 포인트 3가지
4. 예상 질문 2가지와 간단한 답변

케이스:
- 45세 남성 비흡연자, 3주간 건성 기침·호흡곤란 악화
- 흉부 X선 양측 간질성 침윤, SpO2 88%
- LDH 상승, β-D-glucan 양성
- HIV 양성, CD4 45/μL
- BAL: PCP 확인
- 치료: TMP-SMX 고용량 + prednisolone
- 5일째 점진 호전
```

### Step 3: 발표 구조 확인하기
AI가 제안하는 발표 구조 예시입니다.

```
발표 구조 (5분):

1. 환자 소개 (1분)
   - 주증상·경과를 간결하게
   - "3주간의 건성 기침"으로 시작해 청중의 관심 잡기

2. 검사 결과 및 진단 과정 (1.5분)
   - 간질성 폐렴 패턴 → 감별 진단 접근
   - HIV 검사 결정 배경 강조

3. 진단 및 치료 (1분)
   - PCP 확진 과정 및 치료 선택 이유

4. 학습 포인트 (1분)
   - 포인트 1: 비흡연 성인의 간질성 폐렴 — HIV를 항상 감별하라
   - 포인트 2: CD4 <200에서 PCP 예방 요법의 중요성
   - 포인트 3: 중등도 이상 PCP에서 스테로이드 병용의 근거

5. 질의응답 준비 (0.5분)

예상 질문:
Q1. 스테로이드를 언제 추가하나요?
→ PaO2 <70mmHg 또는 A-a gradient >35인 중등도 이상 PCP에서 투여.

Q2. 퇴원 후 TMP-SMX 예방 요법은 언제까지?
→ HAART 후 CD4 >200을 3개월 이상 유지하면 중단 고려.
```

### Step 4 (선택): 슬라이드 구성 요청하기
발표 슬라이드 구성도 부탁할 수 있습니다.

```
위 발표 구조를 바탕으로 파워포인트 슬라이드 5장 구성을 제안해줘.
각 슬라이드의 제목과 핵심 내용 불릿 포인트를 알려줘.
```

## 더 잘 쓰는 팁
💡 팁 1: "청중은 [레지던트/전임의/교수]"라고 지정하면 설명의 깊이가 달라집니다.
💡 팁 2: 예상 질문을 받은 뒤 "이 질문에 대해 더 자세히 설명해줘"라고 이어 물으면 개인 공부도 됩니다.
💡 팁 3: 발표 후 피드백을 AI에 입력하면 다음 발표를 위한 개선 포인트도 잡을 수 있습니다.

## 실수하기 쉬운 부분
⚠️ 실수 1: AI가 제안한 학습 포인트를 검증 없이 발표 → 해결: 학습 포인트는 UpToDate나 가이드라인에서 근거를 확인하세요.
⚠️ 실수 2: 발표 시간을 초과 → 해결: AI가 제안한 섹션별 시간을 보고 실제로 소리 내어 타이머를 재며 연습하세요.

## 다른 방법도 있어요
- ChatGPT의 Canvas 기능: 발표 스크립트를 초안으로 만들고 실시간 수정하기에 편리합니다.
- NotebookLM: 논문이나 증례 파일을 올리면 요약과 핵심 포인트를 자동으로 정리해 줍니다.
