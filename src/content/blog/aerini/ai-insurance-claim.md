## 한줄 요약

진료 내용을 AI에게 주면 주진단·부진단 코드 후보와 청구 주의사항을 정리해 줍니다.
최종 청구는 반드시 심사 기준을 직접 확인하세요.

---

## 당신의 상황

🎯 복잡한 환자의 주진단 코드를 어떻게 잡아야 할지, 어떤 부진단을 함께 넣어야 하는지 헷갈린 적 있으신가요?

💡 AI는 진료 내용을 보고 ICD 코드 후보를 제안하고, 청구 시 자주 발생하는 오류 포인트도 알려줍니다. 처음 배우는 분들에게 특히 유용합니다.

---

## 이렇게 하면 됩니다

### Step 1: 진료 내용을 간결하게 정리하기

AI에게 줄 정보는 다음 항목이면 충분합니다.

```
주 호소, 진단명, 시행한 검사와 처치,
처방 약물, 입원/외래 여부
```

환자 이름과 주민번호는 반드시 제거하세요. 진료 내용만으로도 코드 추천이 가능합니다.

### Step 2: AI에게 코드 추천 요청하기

```
다음 진료 내용으로 건강보험 청구에 적합한
주진단, 부진단 코드와 청구 시 주의사항을 알려줘.

[진료 내용]
```

구체적인 예시:

```
다음 진료 내용으로 건강보험 청구에 적합한
주진단, 부진단 코드와 청구 시 주의사항을 알려줘.

65세 남성 외래 환자.
주 호소: 숨참, 하지 부종.
진단: 심부전 악화 의심, 당뇨 동반.
시행: 흉부 X선, BNP 검사, 이뇨제 추가 처방.
처방: furosemide 40mg 추가
```

### Step 3: AI 추천 코드 검토하기

AI가 제안한 코드를 받으면 다음을 확인하세요.

- 제안된 ICD 코드가 실제 진단명과 일치하는지
- 해당 코드가 현재 건강보험 급여 기준에 포함되는지
- 청구 주의사항 중 해당되는 항목 체크
- 이전에 같은 코드로 삭감된 경험이 있는지

### Step 4: 소견서 작성에도 활용하기

보험사 제출 소견서 작성에도 AI를 활용할 수 있습니다.

```
다음 진료 내용으로 보험사 제출용 소견서 초안을 작성해줘.
객관적 소견 중심으로, 의학적 필요성이 드러나도록 써줘.

[진료 내용]
```

초안 후 반드시 의사가 검토하고 서명해야 합니다.

## 더 잘 쓰는 팁

💡 **팁 1.** "이 코드로 청구할 때 자주 삭감되는 사유가 뭔가요?"라고 후속 질문하면 삭감 예방에 도움이 됩니다.

💡 **팁 2.** 심평원 고시 내용을 함께 붙여넣으면 AI가 더 정확한 판단을 해줍니다. "아래 급여 기준에 맞게 코드를 확인해줘"라고 하세요.

💡 **팁 3.** 자주 쓰는 진료과 코드 조합을 한 번 정리해 두면 다음에 참고 자료로 쓸 수 있어요. AI에게 "이 과에서 자주 쓰는 코드 조합 20개를 표로 만들어줘"라고 요청하세요.

---

## 실수하기 쉬운 부분

⚠️ **실수 1.** AI 추천 코드를 심사 기준 확인 없이 바로 청구하는 것. 급여 기준은 자주 바뀌고 AI 정보가 최신이 아닐 수 있습니다. 반드시 건강보험심사평가원 기준을 직접 확인하세요.

⚠️ **실수 2.** AI가 제안한 소견서 문구를 수정 없이 그대로 쓰는 것. 환자 개인 맥락과 다를 수 있고, 법적 서류인 만큼 의사가 직접 내용을 확인해야 합니다.
