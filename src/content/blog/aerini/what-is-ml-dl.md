## 한줄 요약

AI 안에 머신러닝이 있고, 머신러닝 안에 딥러닝이 있습니다.
러시아 인형(마트료시카)처럼 작은 것이 큰 것 안에 들어있어요.

---

## 왜 알아야 할까?

🎯 의료 AI 뉴스를 읽다 보면 이런 단어들이 나와요.

"머신러닝 기반 예측 모델", "딥러닝으로 영상 분석",
"AI 진단 시스템"...

모두 비슷비슷하게 들리는데, 사실 다 다른 거예요.

💡 이 세 가지를 구분하면:
- ✅ 의료 AI 뉴스를 제대로 이해할 수 있어요
- ✅ 어떤 AI가 우리 병원에 적합한지 판단할 수 있어요
- ✅ 의료 AI 개발팀과 대화할 수 있어요
- ✅ 투자 제안서나 보고서를 이해할 수 있어요

---

## 이게 뭐예요?

### 전체 그림: 러시아 인형

```
┌─────────────────────────────────┐
│  AI (인공지능)                   │
│  인간처럼 생각하는 기계           │
│                                  │
│  ┌───────────────────────────┐  │
│  │  머신러닝                  │  │
│  │  데이터에서 스스로 학습     │  │
│  │                           │  │
│  │  ┌─────────────────────┐  │  │
│  │  │  딥러닝              │  │  │
│  │  │  신경망 기반 학습    │  │  │
│  │  │  (LLM, 이미지AI 등) │  │  │
│  │  └─────────────────────┘  │  │
│  └───────────────────────────┘  │
└─────────────────────────────────┘
```

### AI란?
컴퓨터가 인간처럼 판단하고 문제를 푸는 기술 전체를 뜻해요.
규칙 기반 프로그램부터 ChatGPT까지 모두 AI의 범주예요.

### 머신러닝이란?
**데이터를 보고 스스로 규칙을 학습하는 AI**

예전 AI: 사람이 규칙을 직접 입력 ("고열 + 기침 → 폐렴")
머신러닝: 수천 건 케이스를 보고 스스로 규칙을 발견

### 딥러닝이란?
**뇌의 신경망을 모방한 방식으로 학습하는 머신러닝**

딥러닝은 머신러닝의 한 종류예요.
특히 이미지, 음성, 텍스트 같은 복잡한 데이터에 뛰어나요.
ChatGPT, 의료 영상 AI가 모두 딥러닝 기반이에요.

---

## 예시를 들어볼게

### 예시 1: 당뇨 예측 모델

**머신러닝 방식**:
EMR에서 연령, 체중, 혈당, 가족력 등 수치 데이터 분석
→ "이 수치 조합이면 당뇨 발생 확률 78%"

**딥러닝 방식**:
망막 사진을 AI가 직접 분석
→ "망막 혈관 패턴에서 당뇨 초기 징후 발견"

### 예시 2: 의료 영상 분석

**머신러닝**: 간단한 수치 조합으로 분류
"크기 >2cm + 불규칙 경계 → 악성 의심"

**딥러닝**: 이미지 자체를 수천만 픽셀로 분석
"CT 이미지 전체에서 미세한 패턴 인식 → 초기 폐암 발견"

## 의료에서는 이렇게 써요

| 기술 | 의료 활용 예시 | 특징 |
|------|-------------|------|
| 일반 AI | 처방 알림 시스템 | 단순 규칙 기반 |
| 머신러닝 | 재입원 위험도 예측 | 수치 데이터 분석 |
| 딥러닝 | X-ray 폐렴 진단 보조 | 이미지·텍스트 분석 |
| LLM (딥러닝) | ChatGPT 의무기록 요약 | 자연어 이해·생성 |

### 어떤 걸 쓰면 되나요?

수치 데이터(혈압, 혈당, 나이)로 예측 → **머신러닝**
이미지(CT, MRI, 슬라이드)를 분석 → **딥러닝**
텍스트(의무기록, 논문)를 이해·생성 → **LLM (딥러닝)**

## 주의할 점 ⚠️

### 주의 1: 어떤 AI든 데이터가 핵심
머신러닝이든 딥러닝이든,
학습 데이터가 편향되면 결과도 편향돼요.

예: 특정 인종·성별 데이터로만 학습 → 다른 환자군엔 부정확

✅ 우리 환자 집단을 대표하는 데이터로 학습했는지 확인하세요.

### 주의 2: "AI = 딥러닝 = ChatGPT"가 아니에요
CT 분석 AI와 진료기록 요약 AI는 완전히 다른 기술이에요.
각각 어떤 방식인지 구분해서 이해하세요.

### 주의 3: 성능 수치만 보지 마세요
"정확도 95%"라는 말만으로는 부족해요.
어떤 데이터로, 어떤 상황에서 95%인지가 중요해요.
