## 한줄 요약
약물 목록을 AI에 붙여 넣으면 30초 만에 상호작용 초안을 받을 수 있습니다.
공식 DB 2차 확인은 필수지만, 놓칠 뻔한 조합을 먼저 걸러내는 데 유용합니다.

---

## 당신의 상황
🎯 10가지 약을 복용 중인 환자를 봤는데, 새 약을 추가하려니 상호작용이 걱정됩니다. Micromedex 열기도, 일일이 검색하기도 시간이 촉박합니다.

💡 AI에게 약물 목록을 통째로 주면 주요 상호작용을 심각도별로 정리해줍니다. 검색의 시작점을 단번에 잡을 수 있어요.

## 이렇게 하면 됩니다

### Step 1: 복용 약물 목록 준비하기
EMR에서 현재 복용 중인 약물 전체를 복사합니다. 용량은 몰라도 되지만 약물명은 정확하게 입력하세요. 상품명보다 성분명이 더 잘 인식됩니다.

```
예시 목록:
- 아스피린 100mg
- 와파린 3mg
- 오메프라졸 20mg
- 암로디핀 5mg
- 메트포르민 500mg
- 클로피도그렐 75mg
```

### Step 2: AI에 프롬프트 입력하기
Claude나 ChatGPT 채팅창에 아래 프롬프트를 붙여 넣고 약물 목록을 추가하세요.

```
다음 약물 목록에서 주요 상호작용을 확인해줘.
심각도별(심각 / 주의 / 경미)로 구분하고,
확실하지 않으면 "확인 필요"로 표시해줘.

약물 목록:
- 아스피린 100mg
- 와파린 3mg
- 오메프라졸 20mg
- 암로디핀 5mg
- 메트포르민 500mg
- 클로피도그렐 75mg
```

### Step 3: 결과 확인하기
AI가 심각도별로 정리한 목록을 반환합니다. 예시 결과는 이렇습니다.

```
[심각]
- 아스피린 + 와파린: 출혈 위험 증가. INR 모니터링 필요.
- 클로피도그렐 + 와파린: 3중 항혈소판·항응고 병합으로 출혈 위험 높음.

[주의]
- 아스피린 + 클로피도그렐: 이중 항혈소판 요법, 위장관 보호제 고려.

[확인 필요]
- 암로디핀 + 메트포르민: 드물게 혈당 영향 보고, 공식 DB 확인 권장.
```

### Step 4 (선택): 특정 약물 추가 시 재질문하기
새 약을 추가하려면 이어서 질문하세요.

```
여기에 가바펜틴 300mg을 추가하면 어떤 상호작용이 생기나요?
```

## 더 잘 쓰는 팁
💡 **팁 1**: 성분명으로 입력하면 정확도가 올라갑니다. "노바스크" 대신 "암로디핀"으로 쓰세요.
💡 **팁 2**: 결과 말미에 "불확실한 항목만 따로 목록화해줘"라고 추가하면 검색 우선순위를 잡기 쉽습니다.
💡 **팁 3**: 특이한 조합이나 드문 약물은 AI가 틀릴 가능성이 높습니다. 반드시 Micromedex, Lexicomp 등으로 확인하세요.

## 실수하기 쉬운 부분
⚠️ **실수 1**: AI 결과를 그대로 믿고 공식 DB를 건너뜀 → 해결: AI는 1차 스크리닝 도구입니다. 심각 항목은 반드시 공식 DB에서 2차 확인하세요.
⚠️ **실수 2**: 상품명으로 입력해 AI가 성분을 잘못 파악함 → 해결: 성분명 또는 영문 약물명으로 입력하면 오인식이 줄어듭니다.

## 다른 방법도 있어요
- **Drugs.com Interaction Checker**: 무료로 최대 30개 약물 동시 확인 가능합니다.
- **Micromedex / Lexicomp**: 병원에서 라이선스가 있다면 가장 신뢰할 수 있는 공식 DB입니다. AI 결과 확인용으로 사용하세요.
