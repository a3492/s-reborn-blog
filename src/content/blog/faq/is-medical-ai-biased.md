## 한줄 요약
의료 AI의 편향은 실재한다. 2019년 Optum 알고리즘은 흑인 환자를 체계적으로 과소진료했고, 피부과 AI는 어두운 피부에서 성능이 현저히 낮다. 한국도 학습 데이터 편향에서 자유롭지 않다.

## Q: 의료 AI가 특정 인종이나 성별에게 불공평한가?

## A: 간단 답변
불공평할 수 있고, 실제로 그런 사례가 다수 보고되었다. 편향의 주요 원인은 학습 데이터의 불균형이며, AI가 학습한 인구 집단과 다른 환자에서 성능이 저하된다.

## 상세 설명

### 실제 사례

**사례 1: Optum 의료 자원 배분 알고리즘 (2019)**
Science지에 발표된 연구(Obermeyer et al.)에 따르면, 미국 전역에서 사용된 이 알고리즘은 동일한 건강 상태에서 흑인 환자에게 백인 환자보다 훨씬 낮은 위험 점수를 부여했다. 결과적으로 흑인 환자는 추가 진료 자원 배정에서 체계적으로 배제되었다. 원인은 비용(흑백 간 의료 접근성 차이 반영)을 건강 상태의 대리 지표로 사용한 것.

**사례 2: 피부과 AI 어두운 피부 성능 저하**
2020년 JAMA Network Open에 발표된 연구에 따르면, 피부암 분류 AI 모델들의 학습 데이터 중 피부색이 밝은 샘플(Fitzpatrick Scale I-II)이 전체의 70% 이상을 차지했다. 어두운 피부(V-VI)에서의 민감도는 밝은 피부 대비 15-30% 낮은 경우가 보고됨.

**사례 3: 심전도 AI 성별 편향**
일부 부정맥 감지 AI에서 여성의 QRS 파형 특성이 학습 데이터에서 과소 대표되어 여성 환자에서 성능 저하가 보고됨.

### 편향의 원인 분류

| 편향 유형 | 발생 원인 | 예시 |
|-----------|---------|------|
| 학습 데이터 편향 | 특정 인구 집단 과소 대표 | 백인·남성 위주 학습 데이터 |
| 측정 편향 | 대리 지표의 불평등한 의미 | 의료 비용을 건강 지표로 사용 |
| 레이블 편향 | 레이블 자체의 기존 차별 반영 | 의사 진단 기록의 과거 편향 승계 |
| 피드백 루프 | 편향된 AI가 편향된 데이터 생성 | 과소진료 → 데이터 부재 → 편향 심화 |

### 한국에서의 편향 가능성

한국은 인종적으로 비교적 동질적이지만, 다음 편향 가능성이 있다.

| 영역 | 한국 편향 가능성 | 내용 |
|------|----------------|------|
| 사회경제적 편향 | 높음 | 상급 병원 데이터 편중 — 농촌·저소득층 과소 대표 |
| 성별 편향 | 중간 | 남성 중심 임상 연구 데이터 승계 가능성 |
| 연령 편향 | 중간 | 청소년·고령층 데이터 부족 |
| 외국인 환자 | 높음 | 한국인 데이터로 학습된 AI를 외국인에게 적용 시 성능 미검증 |

### 편향 감지 및 대응 방법

| 단계 | 방법 |
|------|------|
| 도입 전 검토 | 학습 데이터의 인구통계 분포 공개 여부 확인 |
| 성능 검증 | 자체 환자군에서 소집단별(성별·연령·진단명) 성능 분리 평가 |
| 지속 모니터링 | AI 출력과 실제 임상 결과 간 소집단별 격차 추적 |
| 규제 확인 | 식약처 허가 시 성능 검증 대상 인구 범위 확인 |

### 규제 현황
FDA는 2022년 AI/ML 기반 의료기기의 편향 평가를 위한 가이던스를 발표했다. 한국 식약처도 AI 의료기기 허가 심사 시 다양한 인구 집단에서의 성능 검증을 요구하고 있으나, 구체적 기준은 FDA보다 덜 명확하다.

## 핵심 정리
- 의료 AI 편향은 증명된 현실 — Optum 사례, 피부과 AI 사례 등
- 편향의 핵심 원인은 학습 데이터의 불균형
- 한국도 사회경제적·연령 편향에서 자유롭지 않음
- 도입 전 학습 데이터 구성 확인, 도입 후 소집단별 성능 모니터링 필수
