## 한줄 요약
Claude Code 자체는 무료로 설치할 수 있지만, 실행하면 Anthropic API 사용량에 따라 비용이 발생한다. 가볍게 쓰면 월 $5~10, 집중적으로 쓰면 $20~50 이상이 될 수 있다.

## Q: Claude Code 비용은 어떻게 되나요? 무료로 쓸 수 있나요?

## A: 간단 답변
완전 무료는 아니다. Claude Code 도구 설치는 무료지만, AI에 요청할 때마다 API 비용이 발생한다. 얼마나 쓰느냐에 따라 달라지며, 처음에는 소액만 충전해도 충분히 사용할 수 있다.

## 상세 설명

### 비용 구조

Claude Code는 Anthropic API를 사용한다. API는 **토큰(token)** 단위로 과금된다. 토큰은 대략 한국어 기준으로 1~2글자, 영어 기준으로 3~4글자에 해당한다.

**Claude Sonnet 3.5 기준 (2026년 초 기준)**

| 항목 | 단가 |
|------|------|
| 입력 토큰 | $3 / 100만 토큰 |
| 출력 토큰 | $15 / 100만 토큰 |

코드 파일을 읽어서 수정하는 작업은 입력(파일 내용)과 출력(수정된 내용) 토큰이 모두 소비된다.

### 실제 사용 예시로 보는 비용

**간단한 버그 수정 1회**
- 파일 2~3개 읽기 + 수정 코드 출력
- 약 5,000~10,000 토큰
- 비용: $0.05~$0.15 (약 70~200원)

**프로젝트 전체 리팩토링**
- 파일 20개 읽기 + 대화 여러 번
- 약 100,000~300,000 토큰
- 비용: $0.5~$2.5 (약 700~3,500원)

**하루 집중 개발 (4~6시간)**
- 복잡한 기능 구현, 디버깅 반복
- 약 $3~$10 (약 4,000~14,000원)

### 월 비용 예상

| 사용 패턴 | 월 예상 비용 |
|-----------|-------------|
| 가끔 사용 (주 1~2회) | $5 이하 |
| 매일 가벼운 사용 | $10~$20 |
| 업무용 집중 사용 | $30~$100 |

### Cursor와 비교

| 항목 | Claude Code (API) | Cursor Pro |
|------|------------------|------------|
| 기본 요금 | 없음 | $20/월 |
| 과금 방식 | 사용량 기반 | 정액 구독 |
| 적게 쓰면 | 저렴 | 고정 $20 |
| 많이 쓰면 | $20+ 가능 | $20 고정 |
| 예측 가능성 | 어려움 | 쉬움 |

### 비용 절감 방법

**1. Claude Haiku 모델 사용**
간단한 작업은 더 저렴한 모델로 전환한다. 모델 선택은 설정에서 가능하다.

**2. 컨텍스트 관리**
대화가 길어질수록 이전 대화가 입력 토큰으로 쌓인다. 새 작업은 새 대화로 시작하는 것이 효율적이다. `/clear` 명령으로 대화를 초기화할 수 있다.

**3. 한도 설정**
Anthropic Console에서 월 지출 한도를 설정할 수 있다. 한도에 도달하면 자동으로 사용이 멈춘다.

### 처음 시작한다면

$5~$10를 충전하고 시작해보자. 어떤 작업에 얼마나 들지 감을 잡은 후 충전량을 조절하면 된다.

## 핵심 정리
- 설치는 무료, 사용하면 API 비용 발생
- 가볍게 쓰면 월 $10 이하도 충분
- Cursor와 비교: 적게 쓰면 Claude Code, 많이 쓰면 비슷하거나 역전
- Console에서 지출 한도 설정 가능
