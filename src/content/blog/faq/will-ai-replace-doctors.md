## 한줄 요약
현재 AI는 특정 과업에서 의사 수준의 정확도를 달성했지만, 임상 판단·공감·법적 책임 측면에서 의사를 완전히 대체하는 것은 현재 기술로는 불가능하며, 향후 10년 내에도 실현 가능성이 낮다.

## Q: AI가 환자 진료를 완전히 대체할 수 있을까?

## A: 간단 답변
AI는 영상 판독이나 패턴 인식 같은 특정 과업에서 의사에 필적하거나 초월하는 성능을 보이지만, 진단부터 치료 결정까지 전 과정을 법적 책임과 함께 수행하는 것은 현행 기술과 제도 아래서는 불가능하다.

## 상세 설명

### AI가 잘하는 것 vs 못하는 것

| 영역 | AI 성능 | 비고 |
|------|---------|------|
| 흉부 X-ray 판독 | 전문의 수준 (민감도 94%) | Lunit INSIGHT CXR 2023 |
| 안저 사진 당뇨망막병증 탐지 | AUC 0.99 | Google DeepMind |
| 피부 병변 분류 | 피부과 전문의와 동등 | Stanford 2017 |
| 복잡한 증례 감별 진단 | 미숙한 전공의 수준 | GPT-4 USMLE 통과 |
| 환자와의 공감적 의사소통 | 불가 | 언어 모방은 가능, 진정한 공감 불가 |
| 맥락 종합 판단 (사회·심리적 요소) | 매우 제한적 | |
| 법적 의료 행위 수행 | 불가 (의료법상 금지) | 의료법 제27조 |
| 처방·수술 등 침습적 행위 | 불가 | |

### 법적 관점
의료법 제27조는 의료인이 아닌 자의 의료 행위를 금지한다. AI는 법인격이 없으므로 의료 행위의 주체가 될 수 없다. AI가 진단을 내리더라도 이를 환자에게 적용하고 책임지는 주체는 반드시 면허를 가진 의료인이어야 한다.

- **의료법 제27조**: 무면허 의료 행위 금지 — AI는 면허 취득 불가
- **의료법 제21조**: 진료 기록부 작성 의무 — 의사 서명 필수
- **의료법 제15조**: 진료 거부 금지 의무 — AI에게 적용 불가

### 실무 관점: 협력 모델이 현실적 방향

방사선과에서 이미 정착된 'AI-first 판독' 모델이 실질적 대안이다. AI가 1차 스크리닝을 수행하고 의사가 최종 판단을 내리는 구조다.

| 역할 분담 | AI | 의사 |
|-----------|-----|------|
| 이미지 스크리닝 | 전체 | 이상 소견만 검토 |
| 패턴 매칭 | 주도 | 보조 |
| 감별 진단 | 제안 목록 제공 | 최종 결정 |
| 치료 계획 | 프로토콜 제안 | 환자 맞춤 결정 |
| 법적 책임 | 없음 | 전적으로 부담 |

### 10년 전망
- **2026-2030**: AI가 특정 진료과(방사선과, 병리과)에서 의사의 생산성을 2-3배 향상. 의사 수 축소 압력 시작.
- **2030-2035**: 1차 진료에서 AI 트리아지 보편화. 단순 만성질환 모니터링의 AI 주도 가능성.
- **2035 이후**: 법적·윤리적 프레임워크 재정립 논의 본격화. 그러나 '완전 대체'는 사회적 합의와 법 개정 없이는 불가.

## 핵심 정리
- 현재 AI는 특정 진단 과업에서 의사 수준 성능을 달성했지만 전 과정 대체는 불가
- 의료법상 AI는 의료 행위의 법적 주체가 될 수 없음
- 현실적 방향은 AI와 의사의 협력 모델 — AI가 생산성 도구로 기능
- 위협보다 기회로 접근: AI를 잘 쓰는 의사가 AI를 못 쓰는 의사를 대체할 가능성이 더 높음
