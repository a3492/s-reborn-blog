## 한줄 요약
Whisper로 진료 대화를 텍스트로 변환한 뒤, ChatGPT에 SOAP 포맷 프롬프트를 붙이면 5분 안에 구조화된 진료기록이 완성된다.

## 본문

### 왜 AI 진료기록인가

외래 환자 한 명당 진료기록 작성에 평균 4~7분이 소요된다. 하루 40명을 보면 기록에만 3~5시간을 쓰는 셈이다. Whisper + ChatGPT 조합은 이 시간을 1분 이내로 줄여준다.

### 준비물

| 도구 | 용도 | 비용 |
|------|------|------|
| OpenAI Whisper API | 음성 → 텍스트 변환 | $0.006/분 |
| ChatGPT (GPT-4o) | 텍스트 → SOAP 노트 변환 | 월 $20 (Plus) |
| 스마트폰 녹음 앱 | 진료실 음성 녹음 | 무료 |

### 1단계 — 진료 음성 녹음

진료 시작 전 스마트폰 녹음 앱을 켠다. 환자 동의를 반드시 구두로 받고 기록한다.

> "진료 내용 정리를 위해 녹음하겠습니다. 동의하시나요?"

녹음 파일은 m4a 또는 mp3로 저장한다. 한 환자당 보통 3~10분 분량이다.

### 2단계 — Whisper로 텍스트 변환

ChatGPT 앱에서 직접 음성 파일을 업로드하거나, Whisper API를 사용한다.

```python
import openai

client = openai.OpenAI()
audio_file = open("patient_20260329.m4a", "rb")

transcript = client.audio.transcriptions.create(
    model="whisper-1",
    file=audio_file,
    language="ko",
    prompt="내과 외래 진료 대화입니다. 의학 용어가 포함되어 있습니다."
)
print(transcript.text)
```

`prompt` 파라미터에 진료과와 맥락을 넣으면 의학 용어 인식 정확도가 올라간다.

### 3단계 — SOAP 노트 생성 프롬프트

변환된 텍스트를 ChatGPT에 다음 프롬프트와 함께 입력한다.

```
당신은 내과 전문의의 진료기록 작성 보조입니다.
아래 진료 대화 텍스트를 SOAP 형식으로 정리해주세요.

형식:
- S (Subjective): 환자가 호소한 증상, 기간, 양상
- O (Objective): 언급된 활력징후, 검사결과, 신체검진 소견
- A (Assessment): 진단명 또는 감별진단 (ICD-10 코드 포함)
- P (Plan): 처방, 검사 오더, 다음 외래 일정

규칙:
1. 대화에 없는 내용을 추가하지 마세요
2. 의학 용어는 영문 병기하세요 (예: 고혈압(Hypertension))
3. 약물명은 성분명으로 표기하세요

---
[진료 대화 텍스트 붙여넣기]
```

### 실제 출력 예시

```
S: 58세 남성. 2주 전부터 식후 상복부 통증 호소. 쓰린 양상이며
   제산제(antacid) 복용 시 일시적 호전. 체중 감소(-), 흑색변(-),
   구토(-). 흡연력 30갑년, 음주 주 3회 소주 1병.

O: BP 138/88 mmHg, HR 76 bpm, BT 36.5°C
   복부 촉진상 심와부 압통(+), 반발통(-), 장음 정상

A: 1. 기능성 소화불량(Functional dyspepsia) R10.13
   2. R/O 소화성 궤양(Peptic ulcer) K27.9

P: 1. Esomeprazole 40mg qd × 4주
   2. 상부위장관 내시경 예약 (2주 내)
   3. H. pylori 요소호기검사
   4. F/U 4주 후
```

### 이렇게 하세요

- 녹음 전 반드시 환자 동의를 받고 차트에 "녹음 동의 취득"을 기록한다
- Whisper의 `prompt`에 진료과명을 넣어 전문 용어 인식률을 높인다
- 생성된 SOAP 노트를 반드시 의사가 검토·수정 후 EMR에 입력한다
- 민감 정보(주민등록번호 등)가 포함된 경우 API 전송 전 마스킹한다

### 이렇게 하지 마세요

- AI가 생성한 진료기록을 검토 없이 그대로 EMR에 복사하지 않는다
- 환자 동의 없이 녹음하지 않는다 (의료법 위반 소지)
- 대화에 없는 검사 수치를 AI가 "그럴듯하게" 채워 넣은 경우 환각(hallucination)이므로 즉시 삭제한다
- 녹음 파일을 개인 클라우드에 무기한 보관하지 않는다 — 변환 후 즉시 삭제

### 시간 비교

| 단계 | 수기 작성 | AI 활용 |
|------|-----------|---------|
| 진료 중 메모 | 진료와 동시 | 녹음만 |
| 기록 작성 | 5~7분 | 30초 (프롬프트 입력) |
| 검토·수정 | - | 1~2분 |
| 합계 | 5~7분 | 2~3분 |

## 핵심 정리

- Whisper API는 한국어 의학 용어를 90% 이상 정확하게 인식한다 (`prompt` 활용 시)
- SOAP 프롬프트에 "대화에 없는 내용을 추가하지 마세요" 규칙이 핵심이다
- AI 출력물은 반드시 의사가 검토해야 한다 — 최종 책임은 의사에게 있다
- 환자 동의 + 데이터 보안이 법적 전제 조건이다
