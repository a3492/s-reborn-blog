## 한줄 요약
상담 녹음을 Whisper로 텍스트 변환한 뒤, 상담 유형별 구조화 프롬프트를 적용하면 체계적인 상담 기록이 5분 안에 완성된다.

## 본문

### 상담 기록과 진료기록의 차이

진료기록이 SOAP 형식의 객관적 기록이라면, 상담 기록은 환자의 동기·장벽·행동 변화 계획까지 포함해야 한다. 영양상담, 금연상담, 당뇨 자가관리 교육 등은 30~60분이 소요되며, 이를 구조화하여 기록하기란 쉽지 않다.

### 준비물

| 도구 | 용도 |
|------|------|
| 스마트폰 녹음 앱 | 상담 녹음 (환자 동의 필수) |
| Whisper API 또는 Clova Note | 음성 → 텍스트 변환 |
| ChatGPT (GPT-4o) | 텍스트 → 구조화 상담 기록 |

### 범용 상담 기록 프롬프트

```
당신은 의료 상담 기록 전문가입니다.
아래 상담 대화 텍스트를 구조화된 상담 기록으로 정리해주세요.

상담 유형: {영양상담/금연상담/당뇨교육/운동상담}

출력 형식:
1. 상담 요약 (3줄 이내)
2. 현재 상태 평가
   - 현재 습관/행동
   - 문제점
   - 환자의 인식 수준
3. 상담 내용
   - 교육한 내용 (핵심 3~5가지)
   - 환자 반응 및 이해도
4. 행동 변화 단계 (Prochaska 모델)
   - 현재 단계: 무관심/관심/준비/실행/유지
5. 목표 설정
   - 단기 목표 (2주)
   - 장기 목표 (3개월)
6. 다음 상담 계획

규칙:
- 환자가 직접 한 말은 따옴표로 인용
- 대화에 없는 내용을 추가하지 마세요
```

### 실제 적용 — 금연상담

상담 대화 텍스트 (발췌):

```
의사: 하루에 담배 몇 개비 피우세요?
환자: 한 갑 정도요. 20년 됐어요.
의사: 금연 시도해보신 적 있으세요?
환자: 작년에 한번 했는데 2주 만에 다시 폈어요.
      스트레스받으면 참을 수가 없더라고요.
의사: 니코틴 패치나 약물 치료에 대해 들어보셨어요?
환자: 약은 좀 무섭고... 패치는 괜찮을 것 같아요.
```

AI 출력:

```
1. 상담 요약
52세 남성, 20갑년 흡연자. 작년 금연 시도 2주 만에 실패.
스트레스 유발 흡연이 주요 장벽. 니코틴 패치에 긍정적.

2. 현재 상태 평가
- 현재 습관: 1갑/일 × 20년 (20 pack-year)
- 문제점: 스트레스 유발 흡연, 이전 금연 실패 경험으로 자신감 저하
- 인식 수준: 금연 필요성 인지하나 실행 자신감 낮음

3. 상담 내용
- 교육: 니코틴 보조요법(NRT) 종류와 효과, 스트레스 대처법
- 환자 반응: "패치는 괜찮을 것 같아요" (NRT 수용 의향 있음)
- 이해도: 양호

4. 행동 변화 단계: 준비(Preparation)
   - 근거: 금연 의향 있으나 구체적 실행 계획 미수립

5. 목표 설정
- 단기 (2주): 니코틴 패치 시작, 흡연량 50% 감소 (10개비/일)
- 장기 (3개월): 완전 금연 달성

6. 다음 상담: 2주 후, 패치 부착 순응도 + 흡연량 확인
```

### 당뇨 영양상담 프롬프트 변형

```
상담 유형: 당뇨 영양상담

추가 출력:
- 현재 식습관 분석 (탄수화물 비중, 식사 패턴)
- 권장 식단 변경 사항 (환자가 수용 가능한 수준)
- 혈당 자가 측정 패턴 (아침 공복, 식후 2시간)

| 항목 | 현재 | 목표 |
|------|------|------|
| 밥 양 | 1.5공기 | 2/3공기 |
| 간식 | 과일주스 매일 | 생과일 소량 |
| 식사 횟수 | 불규칙 (1~2회) | 규칙적 3회 |
```

### 이렇게 하세요

- 상담 전 환자 동의를 받고 녹음한다
- 상담 유형을 프롬프트에 명시하면 해당 분야 특화 출력이 나온다
- Prochaska 행동 변화 모델을 포함시키면 다음 상담 전략 수립에 유용하다
- 환자의 직접 발언을 따옴표로 인용하면 기록의 신뢰성이 올라간다

### 이렇게 하지 마세요

- 상담에서 나오지 않은 내용을 AI가 추측하여 채워 넣은 경우 삭제한다
- 상담 녹음 파일을 클라우드에 무기한 보관하지 않는다
- AI 기록만으로 상담료 청구의 근거로 삼기 전에 법적 유효성을 확인한다
- 정신건강 상담(자살 사고 등 민감 내용)은 AI 변환 시 각별히 주의한다

## 핵심 정리

- 상담 유형별 구조화 프롬프트로 체계적인 상담 기록을 자동 생성한다
- Prochaska 행동 변화 단계 평가는 다음 상담 전략의 기반이 된다
- 환자 직접 발언 인용은 기록의 객관성과 법적 가치를 높인다
- 녹음 동의와 데이터 보안은 기본 전제 조건이다
