## 한줄 요약
논문 PDF를 Claude나 ChatGPT에 업로드하고 구조화 프롬프트를 쓰면, PICO 기반 요약과 임상 적용 판단까지 5분 안에 끝난다.

## 본문

### 논문 읽기에 AI가 필요한 이유

PubMed에 하루 약 3,000편의 새 논문이 등록된다. 전공 분야만 추려도 주당 수십 편을 읽어야 최신 지견을 유지할 수 있다. AI는 스크리닝 단계를 획기적으로 단축해 정독할 논문을 골라내는 데 도움을 준다.

### 준비물

| 도구 | 장점 |
|------|------|
| Claude 3.5 Sonnet | PDF 직접 업로드 가능, 200K 토큰 컨텍스트로 긴 논문도 처리 |
| ChatGPT (GPT-4o) | PDF 업로드 + 웹 브라우징으로 참고문헌 확인 가능 |
| Gemini 1.5 Pro | 100만 토큰 컨텍스트, 여러 논문 동시 비교 가능 |

### 논문 요약 프롬프트

```
당신은 근거중심의학(EBM) 전문가입니다.
첨부된 논문을 아래 형식으로 요약해주세요.

1. 한줄 요약 (30자 이내)
2. PICO 분석
   - P (Patient): 대상 환자군
   - I (Intervention): 중재/노출
   - C (Comparison): 비교군
   - O (Outcome): 주요 결과변수
3. 연구 디자인: (RCT/코호트/메타분석 등)
4. 핵심 결과:
   - Primary endpoint 결과 (수치 + 95% CI + p-value)
   - NNT 또는 NNH (해당 시)
   - Secondary endpoint 주요 결과
5. 제한점: (저자가 인정한 것 + 내가 볼 때 추가 제한점)
6. 임상 적용:
   - 내 환자에게 적용할 수 있는가? (대상군 일치 여부)
   - 기존 가이드라인과 일치하는가?
   - 진료에 바꿀 것이 있는가?
7. 근거 수준: (GRADE 기준 High/Moderate/Low/Very Low)
```

### 실제 적용 예시

DAPA-HF 연구(Dapagliflozin in Heart Failure)를 넣었을 때의 출력.

```
1. 한줄 요약: HFrEF 환자에서 다파글리플로진이 심부전 악화·심혈관 사망을 줄인다

2. PICO
   - P: EF ≤40% 심부전, NYHA II-IV, NT-proBNP 상승, 당뇨 무관
   - I: Dapagliflozin 10mg qd
   - C: 위약
   - O: 심부전 악화(입원/긴급 방문) + 심혈관 사망 복합

3. 연구 디자인: 다기관 이중맹검 RCT (n=4,744)

4. 핵심 결과:
   - Primary: HR 0.74 (95% CI 0.65-0.85, p<0.001)
   - NNT: 21 (18개월 기준)
   - 심혈관 사망: HR 0.82 (0.69-0.98)
   - 당뇨 유무와 무관하게 효과 일관

5. 제한점:
   - HFpEF는 별도 연구(DELIVER) 필요
   - 아시아인 비율 24% (한국인 데이터 제한적)
   - SGLT2i 간 head-to-head 비교 없음

6. 임상 적용:
   - EF ≤40% 심부전 환자에게 즉시 적용 가능
   - 2023 ESC 가이드라인 Class I 권고와 일치
   - 기존 심부전 약물(ARNi, BB, MRA)에 추가 처방 권장

7. 근거 수준: High (대규모 RCT, 일관된 결과)
```

### 여러 논문 비교 프롬프트

같은 주제의 논문 2~3편을 동시에 분석할 때 사용한다.

```
첨부된 3편의 논문을 비교 분석해주세요.

비교 항목:
| 항목 | 논문 A | 논문 B | 논문 C |
|------|--------|--------|--------|
| 대상군 | | | |
| 중재 | | | |
| Primary endpoint | | | |
| 핵심 결과 | | | |
| 제한점 | | | |
| 근거 수준 | | | |

결론: 세 논문의 결과가 일치하는 부분과 상충하는 부분을 정리하고,
현재 시점에서 임상적 합의(consensus)를 제시하세요.
```

### 이렇게 하세요

- 논문 PDF 원문을 직접 업로드한다 — 초록만 넣으면 핵심 데이터가 누락된다
- PICO + 수치(HR, CI, p-value) 포함을 지시해야 정량적 요약이 나온다
- AI 요약을 읽고 "이 논문을 정독할 가치가 있는가?"를 판단하는 스크리닝 도구로 활용한다
- 중요한 논문은 AI 요약 후 반드시 원문을 직접 읽는다

### 이렇게 하지 마세요

- AI 요약만 읽고 원문을 읽은 것으로 간주하지 않는다
- AI가 제시한 통계 수치를 원문과 대조 없이 인용하지 않는다 (환각 가능)
- 학회 발표·논문 작성에 AI 요약을 직접 복붙하지 않는다
- Preprint(비동료심사)을 동료심사 논문과 동일 수준으로 취급하지 않는다

## 핵심 정리

- PDF 업로드 + PICO 프롬프트로 5분 안에 구조화된 논문 요약을 얻는다
- "임상 적용" 섹션을 포함시키면 논문 읽기가 진료 변화로 이어진다
- 여러 논문을 비교 분석할 때 Gemini의 대용량 컨텍스트가 유리하다
- AI 요약은 스크리닝 도구이지, 원문 읽기를 대체하지 않는다
