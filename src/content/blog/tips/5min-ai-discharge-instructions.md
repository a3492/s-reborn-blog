## 한줄 요약
진단명·수술명·처방 내역을 구조화 프롬프트에 넣으면, ChatGPT가 환자 눈높이의 퇴원 안내문을 자동 생성한다.

## 본문

### 퇴원 안내문이 중요한 이유

퇴원 후 30일 이내 재입원율은 약 10~15%이며, 그 중 상당수가 퇴원 지시 미이해에서 비롯된다. 잘 쓰인 퇴원 안내문은 재입원율을 20% 이상 낮출 수 있다. 그러나 바쁜 병동에서 환자별 맞춤 안내문을 매번 작성하기는 현실적으로 어렵다.

### 준비물

| 도구 | 용도 |
|------|------|
| ChatGPT (GPT-4o) 또는 Claude | 안내문 생성 |
| 병원 퇴원 요약 템플릿 | 진단·처방 정보 확인 |

### 퇴원 안내문 생성 프롬프트

```
당신은 환자 교육 전문 간호사입니다.
아래 퇴원 정보를 바탕으로 환자가 집에서 읽을 퇴원 안내문을 작성해주세요.

퇴원 정보:
- 환자 연령/성별: {연령}세 {성별}
- 진단명: {진단명}
- 수술/시술: {수술명 또는 "해당 없음"}
- 퇴원 처방:
  {약물명 용량 용법}
- 주의사항: {의사가 특별히 당부한 내용}

출력 규칙:
1. 제목은 "퇴원 후 생활 안내"로 시작
2. 6학년이 이해할 수 있는 쉬운 한국어로 작성
3. 의학 용어는 괄호 안에 쉬운 설명 추가
4. 다음 섹션을 포함: 약 복용법, 식이 안내, 활동 제한,
   상처 관리(해당 시), 응급 상황 기준, 다음 외래 일정
5. 응급실에 와야 하는 증상을 빨간 박스처럼 강조
6. 분량: A4 1~2장
```

### 실제 적용 예시

입력:

```
- 환자: 67세 남성
- 진단명: 급성 심근경색(STEMI), LAD 병변
- 수술: 경피적 관상동맥 중재술(PCI), DES 1개 삽입
- 퇴원 처방:
  Aspirin 100mg qd
  Ticagrelor 90mg bid
  Atorvastatin 80mg qd
  Bisoprolol 2.5mg qd
  Ramipril 2.5mg qd
  Nitroglycerin SL 0.6mg prn
- 주의사항: 이중항혈소판요법 최소 12개월 유지 필수,
  자의 중단 시 스텐트 혈전증 위험
```

AI 출력 (요약):

```markdown
# 퇴원 후 생활 안내

## 약 복용 안내

| 약 이름 | 복용법 | 왜 먹나요? |
|---------|--------|-----------|
| 아스피린 100mg | 아침 식후 1알 | 혈관에 넣은 스텐트에 피가 굳지 않게 합니다 |
| 티카그렐러 90mg | 아침·저녁 식후 1알씩 | 아스피린과 함께 스텐트를 보호합니다 |
| 아토르바스타틴 80mg | 저녁 1알 | 콜레스테롤을 낮추고 혈관을 보호합니다 |
| 비소프롤롤 2.5mg | 아침 1알 | 심장이 천천히, 편하게 뛰도록 합니다 |
| 라미프릴 2.5mg | 아침 1알 | 혈압을 낮추고 심장 회복을 돕습니다 |

⚠️ 절대 임의로 약을 중단하지 마세요!
특히 아스피린과 티카그렐러를 갑자기 끊으면 스텐트 안에
혈전(피떡)이 생겨 생명이 위험할 수 있습니다.
최소 12개월은 반드시 두 약을 함께 복용하세요.

## 🚨 응급실에 오셔야 하는 경우

- 가슴 통증이 니트로글리세린 혀 밑 투여 후 5분 이내 호전되지 않을 때
- 갑자기 숨이 심하게 차거나 누우면 숨쉬기 어려울 때
- 의식이 흐려지거나 식은땀이 날 때
- 잇몸 출혈이 멈추지 않거나 검은 변을 볼 때
```

### 이렇게 하세요

- 프롬프트에 "6학년이 이해할 수 있는 수준"을 명시한다 — 의학 용어를 자동으로 풀어 쓴다
- 약물 중단 금지 사항은 별도로 강조 지시를 넣는다
- 생성 후 반드시 담당 의사가 내용을 검토한다
- 환자 특이사항(당뇨 식이, 와파린 복용 등)이 있으면 프롬프트에 추가한다

### 이렇게 하지 마세요

- AI 안내문에 구체적 수치 목표(목표 혈압 등)를 의사 확인 없이 포함하지 않는다
- 모든 환자에게 동일한 템플릿을 돌려 쓰지 않는다 — 개별 처방에 맞춰야 한다
- 환자 이름·등록번호를 프롬프트에 넣지 않는다

### 다국어 안내문 생성

외국인 환자가 있을 때 추가 프롬프트를 사용한다.

```
위 퇴원 안내문을 영어로 번역해주세요.
의학 용어는 원문 한국어를 괄호 안에 병기하세요.
문화적 맥락이 다른 부분(예: 한국 응급실 연락처)은
현지 상황에 맞게 조정하세요.
```

## 핵심 정리

- 진단명·수술·처방을 구조화하여 입력하면 맞춤형 퇴원 안내문이 자동 생성된다
- "쉬운 한국어" + "응급 상황 강조" 규칙이 핵심이다
- 이중항혈소판요법 등 절대 중단 금지 약물은 프롬프트에서 별도 강조한다
- 최종 검토 없는 배포는 의료 사고 위험이 있다
