## 한줄 요약
질환명과 환자 특성(연령, 교육 수준)을 프롬프트에 넣으면 ChatGPT가 환자 맞춤형 설명 자료를 3분 안에 만들어준다.

## 본문

### 왜 환자 설명 자료가 필요한가

건강문해력(health literacy)이 낮은 환자는 치료 순응도가 50% 이하로 떨어진다. 진료실에서 구두로 설명한 내용의 80%는 귀가 후 잊힌다. 인쇄물로 된 설명 자료를 함께 제공하면 이해도와 순응도가 유의하게 올라간다.

### 기본 프롬프트

```
당신은 환자 교육 자료 전문가입니다.
아래 질환에 대해 환자용 설명 자료를 작성해주세요.

질환: {질환명}
환자 특성: {연령}세, {교육 수준 또는 "일반인"}
동반 질환: {있으면 기재}

출력 규칙:
1. 제목은 "○○○ 알아보기"
2. 다음 섹션 포함: 이 병은 무엇인가요?, 왜 생기나요?,
   어떤 증상이 있나요?, 어떻게 치료하나요?,
   일상생활에서 주의할 점, 자주 묻는 질문 3개
3. 의학 용어 사용 시 괄호 안에 쉬운 설명
4. 비유를 활용해 핵심 개념 설명
5. 분량: A4 1장 이내
6. 불필요한 공포 조장 금지
```

### 실제 적용 — 제2형 당뇨병 설명 자료

입력:

```
질환: 제2형 당뇨병
환자 특성: 55세, 고졸, 처음 진단
동반 질환: 고혈압(Amlodipine 5mg 복용 중)
```

AI 출력 (발췌):

```markdown
# 제2형 당뇨병 알아보기

## 이 병은 무엇인가요?

우리 몸은 음식을 먹으면 포도당(혈당)으로 바꿔서 에너지로
씁니다. 이때 인슐린이라는 호르몬이 열쇠처럼 세포 문을 열어
포도당을 들여보냅니다.

제2형 당뇨병은 이 열쇠(인슐린)가 잘 작동하지 않거나,
열쇠가 부족해져서 포도당이 세포 안으로 들어가지 못하고
혈액에 쌓이는 상태입니다.

## 왜 생기나요?

- 유전적 요인 (가족 중 당뇨 환자가 있으면 위험 2~3배)
- 과체중·비만 (특히 배 둘레가 남성 90cm, 여성 85cm 이상)
- 운동 부족
- 나이 (40세 이후 위험 증가)
- 고혈압이 있으면 당뇨도 함께 오는 경우가 많습니다
  → 선생님의 경우 고혈압약을 이미 드시고 계시므로
    두 질환을 함께 관리하는 것이 중요합니다.

## 혈당 목표

| 검사 항목 | 목표 수치 | 의미 |
|-----------|-----------|------|
| 공복혈당 | 80~130 mg/dL | 아침 식전에 재는 혈당 |
| 식후 2시간 혈당 | 180 mg/dL 미만 | 식사 후 2시간 뒤 |
| 당화혈색소(HbA1c) | 6.5~7.0% | 최근 3개월 평균 혈당 |
```

### 질환별 프롬프트 변형

고령 환자용 (큰 글씨, 단순 구조):

```
위 설명 자료를 75세 어르신용으로 수정해주세요.
- 글씨 크기 설명: "(큰 글씨로 인쇄)" 표시
- 문장을 더 짧게 (한 문장 20자 이내)
- 핵심 행동 3가지만 강조 (나머지 생략)
- 가족에게 전달할 내용도 별도 박스로 추가
```

소아 환자 보호자용:

```
질환: 천식(Asthma)
환자 특성: 7세 남아의 부모
동반 질환: 알레르기 비염

추가 규칙:
- 부모가 궁금해할 질문 위주로 구성
- "학교에서 체육 시간에 어떻게 하나요?" 포함
- 흡입기 사용법을 단계별로 설명
```

### 이렇게 하세요

- 환자의 연령과 교육 수준을 프롬프트에 명시하면 눈높이가 맞는 자료가 나온다
- 동반 질환을 넣으면 AI가 연관 설명을 자동으로 포함한다
- 인쇄 전 반드시 의사가 수치·권고사항을 확인한다
- 병원 로고와 연락처를 템플릿에 추가해 공식 자료로 활용한다

### 이렇게 하지 마세요

- AI가 생성한 약 용량·목표 수치를 검증 없이 배포하지 않는다
- "검색하면 나오는 일반 정보"를 그대로 쓰지 않는다 — 환자 개별 상황에 맞춰야 한다
- 설명 자료에 치료비·보험 관련 내용은 넣지 않는다 (변동 가능성)
- 공포를 조장하는 합병증 사진·통계를 남용하지 않는다

## 핵심 정리

- 질환명 + 환자 특성 + 동반 질환을 입력하면 맞춤형 설명 자료가 생성된다
- "비유 활용" 지시가 이해도를 크게 높인다
- 고령·소아·외국인 등 대상별로 프롬프트를 변형한다
- 수치와 권고사항은 반드시 의사가 최종 확인한다
