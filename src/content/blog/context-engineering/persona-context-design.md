## 한줄 요약
같은 질환이라도 환자의 연령, 문화적 배경, 심리 상태에 따라 **정보의 양, 표현 방식, 강조 포인트**가 달라져야 하며, 이를 페르소나 컨텍스트로 구조화하면 AI가 환자 유형에 맞는 응대를 생성한다.

## 본문

### 왜 환자 유형별 컨텍스트가 필요한가

"제2형 당뇨에 대해 설명해 주세요."

이 질문에 대한 답은 누가 질문했느냐에 따라 완전히 달라야 한다.

- 70대 독거 어르신에게는 간결하고 구체적인 생활 지침이 필요하다
- 5세 당뇨 아이의 부모에게는 학교생활 관리와 저혈당 대처법이 필요하다
- 영어권 외국인에게는 한국 의료 시스템 안내가 추가로 필요하다
- 주사 공포증이 있는 환자에게는 인슐린 설명 방식을 조심해야 한다

AI에게 "환자에 맞게 설명해 줘"라고만 하면 매번 비슷한 중립적 톤의 답변이 나온다. 환자 유형을 **구조화된 컨텍스트**로 제공해야 AI가 실질적으로 다르게 응대할 수 있다.

### 환자 유형별 컨텍스트 설계

#### 유형 1: 고령 환자 (65세 이상)

고령 환자 응대에서 중요한 것은 **정보량 제한과 실천 가능성**이다.

```
[고령 환자 페르소나 컨텍스트]
# 환자 특성
- 65세 이상, 시력 또는 청력 저하 가능
- 복합 만성질환으로 다제 복용 중
- 디지털 기기 활용 제한적
- 보호자(자녀)가 동행하는 경우 많음

# 응대 원칙
- 한 번에 3가지 이하의 핵심 정보만 전달
- 의학 용어 대신 일상 언어 사용
- 약 복용법은 시간대별로 구체적으로 안내
  (예: "아침밥 먹고 바로" / "저녁 8시에")
- "하지 마세요" 보다 "이렇게 하세요"로 긍정형 안내
- 글자 크기가 큰 인쇄물 형태 제공 고려
```

#### 유형 2: 소아 환자의 보호자

소아 환자에게 직접 설명하는 경우는 드물다. 대부분 보호자(부모)에게 설명한다.

```
[소아 보호자 페르소나 컨텍스트]
# 환자 특성
- 환자: 소아 (0~12세)
- 실제 소통 대상: 보호자 (부모)
- 보호자의 불안 수준이 높을 수 있음
- 학교/어린이집 생활 관리가 중요

# 응대 원칙
- 질환 설명 시 불안을 키우지 않는 표현 사용
- "아이가 정상적으로 생활할 수 있다"는 메시지 포함
- 학교/어린이집에서의 관리법 구체적으로 안내
- 응급 상황 판단 기준을 명확히 제시
  (예: "이런 경우에는 즉시 응급실로")
- 보호자가 선생님에게 전달할 안내문 형태 제공
```

#### 유형 3: 외국인 환자

한국에서 진료받는 외국인은 언어 장벽뿐 아니라 의료 시스템 차이로 혼란을 겪는다.

```
[외국인 환자 페르소나 컨텍스트]
# 환자 특성
- 한국어 제한적 (영어 또는 모국어 사용)
- 한국 의료 시스템(건강보험, 처방전, 약국 분리)에 익숙하지 않음
- 본국의 의료 관행과 차이가 있을 수 있음

# 응대 원칙
- 가능한 한 환자의 주 언어로 응대
- 한국 의료 시스템의 절차를 단계별로 안내
  (예: "진료 후 처방전을 받아 약국에 가세요")
- 약품명은 성분명(generic name)으로 병기
- 본국 귀국 후 연속 치료가 필요한 경우 영문 소견서 안내
- 응급 시 119 사용법, 야간 진료 안내 포함
```

#### 유형 4: 의료 불안증 환자

주사 공포, 검사 불안, 질병 공포를 가진 환자는 일반적인 설명 방식이 역효과를 낼 수 있다.

```
[의료 불안증 환자 페르소나 컨텍스트]
# 환자 특성
- 의료 행위에 대한 높은 불안 (주사, 채혈, MRI 등)
- 또는 질병 진단 자체에 대한 과도한 두려움
- 인터넷 검색으로 최악의 시나리오를 이미 본 경우 많음

# 응대 원칙
- 불안을 인정하고 공감하는 문장으로 시작
  (예: "걱정되시는 게 당연합니다")
- 통계를 사용할 때 부정적 수치보다 긍정적 수치 강조
  (예: "5%의 합병증" 대신 "95%는 문제없이 회복")
- 과정을 미리 구체적으로 설명 (불확실성이 불안을 키움)
- 환자가 통제할 수 있는 부분을 안내
  (예: "언제든 중단을 요청할 수 있습니다")
- 의학적으로 위험한 정보를 숨기지는 않되, 전달 순서에 유의
```

### 문화적 맥락의 컨텍스트 반영

같은 질환이라도 문화적 배경에 따라 받아들이는 방식이 다르다.

**정신건강 질환:** 우울증에 대한 사회적 낙인이 존재하므로 "마음 건강 관리"와 같은 완곡한 표현을 먼저 사용할 수 있다.
**식이 지도:** "쌀밥을 줄이세요"보다 "잡곡밥으로 바꾸고 반 공기부터"가 실천 가능하다.
**가족 참여:** 한국 문화에서 가족은 치료 결정에 깊이 관여한다. 보호자 설명, 한의학 병행 확인, 건강기능식품 복용 여부 확인, 한식 구조 전제 식이 지도 등을 컨텍스트에 포함한다.

### 실전: 환자 유형에 따른 자동 컨텍스트 전환

환자 유형을 매번 수동으로 입력하는 것은 비현실적이다. 자동 전환 시스템을 설계할 수 있다.

**Step 1:** EMR에 환자 특성 태그 추가 (연령 → 자동 태깅, 국적 → 자동 태깅)
**Step 2:** 유형별 페르소나 컨텍스트 템플릿 사전 작성
**Step 3:** 질의 시 환자 ID → EMR 태그 조회 → 페르소나 컨텍스트 자동 로드 → 의학 컨텍스트와 조합 → LLM 전달
**Step 4:** 의사가 오버라이드 가능 — 자동 태깅이 맞지 않으면 수동 전환

### 주의사항

**스테레오타이핑 방지:** 페르소나는 기본값이다. 의학 지식이 높은 고령 환자에게 무조건 쉬운 말만 쓰면 무례하다. 개별 조정이 가능해야 한다.

**과도한 단순화 금지:** 전달 방식을 조절하되 핵심 의학 정보는 반드시 포함한다.

**다중 유형 처리:** 고령 + 외국인처럼 여러 유형에 해당하면 컨텍스트를 병합하되, 충돌 시 더 안전한 쪽을 우선한다.

## 핵심 정리

1. 같은 질환이라도 환자 유형에 따라 **정보량, 표현 방식, 강조 포인트**가 달라져야 한다
2. 고령 환자는 핵심 3가지 이하로, 소아 보호자는 학교 생활 관리 중심으로 설계한다
3. 문화적 맥락(식문화, 가족 구조, 보완요법)을 컨텍스트에 반영한다
4. EMR 연동으로 환자 유형 자동 태깅 및 컨텍스트 자동 전환이 가능하다
5. 페르소나는 **기본값**이며, 의사의 판단으로 오버라이드할 수 있어야 한다
