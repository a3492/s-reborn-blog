## 한줄 요약
프롬프트는 "지금 이걸 해줘"라는 지시이고, 컨텍스트는 "이 배경을 참고해서"라는 참조 자료다 — 좋은 AI 활용은 둘 다 잘 설계해야 한다.

## 본문

### 진료실 비유

의사가 환자를 진료하는 상황을 생각해 보자.

- 프롬프트 = 의사가 하는 질문: "어디가 불편하세요?"
- 컨텍스트 = 진료 전에 이미 준비된 것들: 과거 차트, 검사 결과지, 복용 약물 목록, 알레르기 정보, 이전 의뢰서

좋은 문진(프롬프트)만으로도 어느 정도 진료가 되지만, 차트 없이 하는 진료(컨텍스트 없이)와 차트를 다 보고 하는 진료의 정확도는 차원이 다르다.

AI도 동일하다.

---

### 구분 정리

| 항목 | 프롬프트 | 컨텍스트 |
|------|----------|----------|
| 정의 | 사용자의 현재 요청/지시 | AI에게 제공되는 모든 배경 정보 |
| 비유 | 문진 질문 | 차트 + 검사지 + 가이드라인 |
| 변동성 | 매 요청마다 바뀜 | 세션/시스템 수준에서 유지 |
| 예시 | "이 환자의 감별진단은?" | 시스템 프롬프트, RAG 문서, 환자 데이터, 대화 이력 |
| 설계 주체 | 사용자 (매번 직접 작성) | 시스템 설계자 (사전 구성) |

---

### 실전에서의 차이

프롬프트만 좋은 경우
```
당신은 심장내과 전문의입니다.
55세 남성, 흉통 30분, 발한 있음. ECG에서 ST 상승.
감별진단과 초기 처치를 단계별로 제시해 주세요.
```
→ 좋은 답변이 나올 수 있다. 하지만 AI는 이 환자의 기존 복용 약물, 알레르기, 이전 PCI 이력을 모른다.

컨텍스트까지 좋은 경우
```
[시스템 프롬프트]
당신은 S병원 심장내과 AI입니다. 한국 보험 기준, 응급 프로토콜 적용.

[환자 데이터 — EMR 추출]
이름: (익명) / 55세 남성
기저질환: DM2, 고지혈증, 2023년 LAD PCI 이력
복용 약물: Aspirin 100mg, Clopidogrel 75mg, Atorvastatin 20mg, Metformin 1000mg
알레르기: Penicillin (발진)
최근 검사: eGFR 68, HbA1c 7.2, EF 55%

[현재 상황]
흉통 30분, 발한, BP 160/100, HR 92, ECG V2-V5 ST↑ 3mm

[프롬프트]
STEMI 프로토콜에 따른 초기 처치와 PCI 재시행 여부를 판단해 주세요.
```
→ 이전 PCI 이력, 이중항혈소판 이미 복용 중인 점, 메트포르민 조영제 관련 주의 등을 모두 반영한 구체적 답변.

---

### 왜 둘 다 필요한가

프롬프트만 있으면:
- 일반론적 답변
- 환자 특이적 정보 반영 불가
- 매번 긴 배경 설명을 반복해야 함

컨텍스트만 있으면:
- AI가 무엇을 해야 할지 모름
- 정보는 있는데 방향이 없음
- 출력 형식이 제멋대로

둘 다 잘 설계하면:
- 환자별 맞춤 답변
- 일관된 형식과 품질
- 반복 작업 없이 효율적 사용

---

### 역할 분담

| 누가 | 무엇을 설계하는가 |
|------|---------------------|
| 시스템 설계자 (개발자/IT) | 시스템 프롬프트, RAG 파이프라인, EMR 연동, 도구 연결 |
| 사용자 (의사) | 프롬프트 — 구체적 질문과 출력 형식 지시 |
| 자동화 | 환자 데이터 자동 추출, 관련 가이드라인 자동 검색 |

이상적인 의료 AI 시스템은 컨텍스트는 자동으로 구성되고, 의사는 프롬프트만 입력하면 되는 구조다.

---

### 현실적 접근

아직 EMR 연동 AI가 없는 상황에서도 컨텍스트 엔지니어링을 적용할 수 있다.

1. ChatGPT Custom Instructions에 시스템 프롬프트 설정
2. 긴 대화 시작 시 배경 정보를 먼저 입력 (환자 정보, 관련 가이드라인 복붙)
3. 자주 쓰는 참조 자료를 템플릿으로 저장해 매번 붙여넣기
4. Claude Projects에 자주 참조하는 문서 업로드

## 핵심 정리
- 프롬프트 = 지시 (무엇을 해달라), 컨텍스트 = 배경 (이걸 참고해서)
- 프롬프트만으로는 일반론, 컨텍스트까지 설계하면 환자별 맞춤 답변
- 이상적 구조: 컨텍스트는 자동 구성, 의사는 프롬프트만 입력
