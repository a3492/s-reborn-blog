## 한줄 요약

68세 간세포암 환자의 다학제 회의에서 AI는 Barcelona 병기 분류와 치료 알고리즘을 즉시 제공했지만, 간경변 중증도에 따른 수술 가능성 판단은 의료진의 종합 토의가 필요했다.

## 환자 정보

| 항목 | 내용 |
|------|------|
| 나이/성별 | 68세 남성 |
| 진단 | 간세포암 (HCC), B형 간염 간경변 |
| 발견 경위 | 정기 복부 초음파에서 2.8cm 간 종괴 발견 |
| 기저질환 | B형 간염 (entecavir 복용 중), 당뇨, 고혈압 |

## 검사 결과

| 검사 | 결과 | 의미 |
|------|------|------|
| AFP | 340 ng/mL | 상승 (정상 < 10) |
| Liver CT (동맥기 조영증강, 문맥기 washout) | S6 2.8cm 단일 종괴 | 전형적 HCC 소견 |
| Child-Pugh score | B7 (albumin 3.0, bilirubin 2.1, PT INR 1.4, 복수 경도, 뇌병증 없음) | 중등도 간기능 저하 |
| MELD score | 12 | 중등도 |
| ICG R15 | 28% | 간 예비능 저하 |
| Platelet | 82,000/uL | 감소 (문맥압 항진 시사) |
| Chest CT | 이상 없음 | 폐 전이 배제 |

## 다학제 회의 구성

| 진료과 | 역할 |
|--------|------|
| 간담췌외과 | 수술적 절제 가능성 평가 |
| 영상의학과 | 영상 판독, 경동맥화학색전술(TACE) 시행 |
| 종양내과 | 전신 항암 요법 검토 |
| 방사선치료과 | SBRT(체부정위방사선치료) 가능성 |
| 소화기내과 | 간기능 평가, 간이식 적응증 검토 |

## AI 활용 — 회의 전 자료 정리

```
68세 남성 HCC 환자의 다학제 회의 자료를 정리하라.

[환자 데이터]
- HCC: S6 단일 종괴 2.8cm, 혈관 침범 없음, 원격 전이 없음
- AFP: 340 ng/mL
- Child-Pugh: B7 (Alb 3.0, Bil 2.1, INR 1.4, 경도 복수)
- MELD: 12
- ICG R15: 28%
- 기저: HBV cirrhosis (entecavir), DM, HTN

다음을 제시하라:
1. BCLC 병기 분류
2. 각 병기별 권장 치료 알고리즘
3. 각 치료 옵션의 5년 생존율 데이터
4. 이 환자에서 고려할 수 있는 치료 옵션 목록
```

## AI 분석 결과

### BCLC 병기 분류

AI는 이 환자를 BCLC Stage A (조기)로 분류했다.
- 단일 종괴 < 3cm, ECOG PS 0, Child-Pugh B

### 치료 옵션 비교

| 치료법 | 적응증 | 5년 생존율 | 이 환자 적용 가능성 |
|--------|--------|-----------|-------------------|
| 간 절제 | Child-Pugh A, ICG R15 < 15% | 50-70% | :red_circle: Child B7, ICG 28% |
| 간 이식 | Milan criteria 충족 | 70-80% | :yellow_circle: 단일 2.8cm → 충족 |
| RFA (고주파열치료) | < 3cm, 3개 이하 | 40-60% | :green_circle: 기술적 가능 |
| TACE | BCLC B, 다발성 | 20-30% | :yellow_circle: 단일 종괴엔 차선 |
| SBRT | 수술/RFA 불가 시 | 30-50% | :yellow_circle: 대안 옵션 |
| Atezolizumab + Bevacizumab | BCLC C, 전신 치료 | 20% (2년 OS 40%) | :red_circle: 현 병기에 과잉 |

## 각 과의 의견과 AI 비교

### 간담췌외과

> "AI가 간 절제를 적응증 외로 분류한 것은 적절합니다. ICG R15 28%에서 우엽 절제는 간부전 위험이 높습니다."

### 영상의학과

> "종괴 위치가 S6 표면이라 RFA 접근이 용이합니다. TACE 단독보다는 RFA가 우선입니다."

### 소화기내과

> "Milan criteria 충족하므로 간이식 대기 등록을 병행하면서, 브릿지 요법으로 RFA를 먼저 시행하는 것을 제안합니다."

### 종양내과

> "현 병기에서 전신 항암 요법은 불필요합니다. 재발 시 고려 사항으로 남겨둡니다."

### 방사선치료과

> "RFA가 기술적으로 가능하면 SBRT보다 RFA가 우선입니다. RFA 실패 시 SBRT를 고려합니다."

## 최종 치료 계획

| 순서 | 치료 | 시기 |
|------|------|------|
| 1 | RFA (브릿지 요법) | 즉시 |
| 2 | 간이식 대기 등록 | RFA 후 4주 내 |
| 3 | 3개월 간격 AFP + CT 추적 | 지속 |
| 4 | 재발 시 TACE 또는 전신 치료 검토 | 필요 시 |

## AI가 다학제 회의에서 실제로 도운 부분

1. 병기 분류 즉시 산출 -- BCLC, Child-Pugh, MELD 점수를 입력 데이터에서 자동 계산
2. 치료 알고리즘 가시화 -- 각 옵션의 근거 논문과 생존율을 테이블로 정리
3. 시간 절약 -- 회의 전 자료 정리에 30분 → 5분

## AI가 하지 못한 부분

1. ICG R15 28%의 임상적 의미 해석 -- 수치를 제시했지만 "우엽 절제 시 간부전 위험"이라는 외과적 판단은 의사 몫
2. 환자의 사회경제적 상황 -- 간이식 비용, 생체 공여자 가능성 등은 AI 분석 범위 밖
3. 각 과 의견 간 우선순위 조율 -- 최종 합의는 의료진 토의로만 가능

## 핵심 정리

- AI는 다학제 회의의 자료 정리와 근거 검색에 효과적이다
- 병기 분류, 치료 알고리즘 같은 정형화된 판단은 AI가 빠르고 정확하게 수행
- 최종 치료 결정은 환자 개별 상황과 각 과 전문 의견의 종합으로 이루어진다
