## 한줄 요약

자가주사를 처음 시작하는 환자에게 구두 설명만으로는 부족하다 — 가져갈 수 있는 맞춤 안내문을 AI로 3분 만에 만든다.

## 자가주사 안내문이 필요한 이유

주사 교육은 진료실에서 5-10분 구두로 진행되지만, 환자가 집에서 혼자 처음 주사를 맞을 때 기억나는 내용은 30% 미만이다 (JDCE 2022). 맞춤 인쇄 안내문이 있으면 순응도가 22% 향상된다.

## 프롬프트 1: 인슐린 자가주사 안내

```
다음 조건에 맞는 인슐린 자가주사 안내문을 작성해줘.
처음 인슐린을 시작하는 환자가 혼자 집에서 읽고 따라할 수 있도록
단계별로 구체적으로 설명해줘. 그림 설명이 필요한 부분은 [그림: 설명] 형식으로 표시해줘.

인슐린 종류: 글라진(Lantus) 10단위
투여 시간: 취침 전 (오후 10시 고정 권장)
투여 부위: 복부 (배꼽 주변 5cm 제외)
대상: 65세 여성, 제2형 당뇨, 처음 인슐린 시작
사용 기기: FlexPen (프리필드 펜형)
저장 온도: 미개봉 냉장 / 개봉 후 실온 28일

포함해야 할 내용:
1. 주사 전 준비물 (체크리스트)
2. 단계별 주사 방법 (7단계 이내)
3. 주사 부위 돌려가기 방법
4. 주사 후 확인 사항
5. 저혈당 발생 시 대처법 (혈당 < 70 mg/dL)
6. 즉시 병원 연락이 필요한 상황

형식: A4 1페이지, 큰 글씨, 번호 목록, 경고 박스 포함
```

실제 출력 예시 (요약):
```
[인슐린 자가주사 안내문]
환자: ○○○님 / 인슐린: 글라진(Lantus) 10단위 / 시간: 매일 밤 10시

준비물 체크리스트:
□ 란투스 FlexPen  □ 주사 바늘 (4mm 또는 6mm)
□ 알코올 솜        □ 소독된 손

7단계 주사 방법:
1. 손을 비누로 20초 씻기
2. 펜에서 캡 제거 → 새 바늘 돌려 끼우기
3. 2단위 공기 주입 테스트 (펜 위로 향하게, 방울 확인)
4. 용량 다이얼: 10단위로 맞추기 [그림: 다이얼 10]
5. 배꼽에서 5cm 이상 떨어진 복부 집어 올리기
6. 90도로 바늘 삽입 → 버튼 눌러 천천히 주입
7. 10초 기다린 후 바늘 제거 → 바늘 캡 씌워 분리 폐기

⚠ 저혈당 대처 (혈당 < 70 mg/dL 또는 손 떨림, 식은땀):
즉시 → 설탕 15g (주스 150mL 또는 사탕 3개)
15분 후 혈당 재측정 → 70 미만이면 반복
혼자라면 119 호출

즉시 연락: 혈당 > 300, 구역/구토로 식사 불가, 주사 부위 심한 통증
병원 전화: ○○○-○○○○
```

## 프롬프트 2: GLP-1 주사제 안내 (오젬픽 / 위고비)

```
다음 조건에 맞는 GLP-1 수용체 작용제 자가주사 안내문을 작성해줘.

약물: 오젬픽(Ozempic, 세마글루타이드) 0.25mg 주 1회
적응증: 제2형 당뇨 혈당 조절
용량 증량 계획: 4주 0.25mg → 4주 0.5mg → 유지 1mg
투여 부위: 복부, 허벅지, 상박 외측 중 선택
대상: 48세 남성, 비만(BMI 31), 당뇨 3년차
사용 기기: 오젬픽 프리필드 펜

포함 내용:
1. 주 1회 주사 — 요일 고정 방법 (예: 매주 월요일)
2. 식사와 무관하게 투여 가능 (오젬픽 특성)
3. 흔한 부작용: 오심, 구토 — 언제 줄어드는지
4. 절대 금지: 식이 교육 없이 과도한 식이 제한 금지
5. 요일 바꿀 경우 최소 2일 이상 간격 유지
6. 냉장 보관 / 이동 시 냉장백 사용

형식: A4 반 페이지, 환자 친화적 언어, 부작용 경고 박스
```

## 프롬프트 3: 생물학적 제제 주사 안내 (에타너셉트 / 아달리무맙)

```
다음 생물학적 제제 자가주사 안내문을 작성해줘.
면역 억제 상태임을 고려해 감염 예방 주의사항을 반드시 포함해줘.

약물: 에타너셉트(Enbrel) 50mg 주 1회 또는 아달리무맙(Humira) 40mg 격주
적응증: 류마티스 관절염
대상: 52세 여성, 류마티스 관절염 2년, 메트렉세이트 병용 중
보관: 냉장 (2-8°C), 투여 30분 전 실온 꺼내기
주사 부위: 복부 또는 허벅지 (관절염 주변 부위 피할 것)

필수 포함 내용:
1. 주사 전 감염 징후 자가 확인 (발열, 기침, 상처 등)
2. 결핵 검사 / B형 간염 검사 완료 여부 확인 (병원 확인사항)
3. 주사 부위 반응 (발적, 부종) — 정상 vs 이상 반응 구분
4. 생백신 접종 금지 (대상포진, 홍역 등)
5. 응급 상황: 고열, 급격한 쇠약감 → 즉시 응급실

형식: 2페이지 이내, 감염 예방 항목은 빨간 박스로 강조
```

## 환자 개별 맞춤화 필수 항목

AI 출력 후 반드시 의사/간호사가 개인화해야 할 내용:
- 실제 처방 용량 (AI 출력의 예시 용량과 다를 수 있음)
- 투여 시간 (환자 생활 패턴에 맞게 조정)
- 보험급여 조건에 따른 용량 증량 일정
- 저혈당 대처 — 기저 신기능, 다른 약물과 상호작용 고려
- 병원 연락처 (AI 출력에 빈칸으로 처리됨)

## 주의사항

- AI 출력을 그대로 인쇄하지 말 것 — 용량·적응증 반드시 확인
- 한국어 말하기 어렵거나 시력 약한 고령 환자는 보호자 동석 교육 추가
- 주사 기기 종류(펜형 vs 사전충전 주사기 vs 자동주사기)에 따라 단계가 다름

## 핵심 정리

- 프롬프트 1: 인슐린 글라진 — 처음 시작 고령 환자, 7단계 방법+저혈당 대처 포함
- 프롬프트 2: 오젬픽/위고비 — 주 1회 고정, 오심 부작용 관리, 용량 증량 계획
- 프롬프트 3: 에타너셉트/아달리무맙 — 감염 예방 강조, 생백신 금기 명시
- 맞춤 안내문으로 자가주사 순응도 22% 향상 (JDCE 2022)
- 용량·연락처·시간은 AI 출력 후 반드시 수동으로 개인화
