## 한줄 요약

다제약물 고령 환자 한 명의 처방을 AI에 넣었더니 5분 만에 심각한 상호작용 2건과 Beers 기준 위반 약물 2건이 나왔다 — 이후 Lexi-Interact로 교차확인까지 10분.

## 다제약물 처방 검토의 현실

65세 이상 환자의 40% 이상이 5가지 이상 약물을 복용한다(다제약물, polypharmacy). 약물 수가 늘수록 상호작용 가능한 조합은 기하급수적으로 증가한다:
- 5가지 약물: 10가지 조합
- 10가지 약물: 45가지 조합
- 15가지 약물: 105가지 조합

AI는 이 모든 조합을 순식간에 검토하는 데 강점이 있다. 단, 최종 판단은 반드시 의사와 임상 약사가 수행해야 한다.

## 프롬프트 1: 다제약물 상호작용 확인

```
다음 처방에서 주요 약물 상호작용, 금기사항, 주의사항을 확인해줘.
심각도 순서(심각 > 중등도 > 경미)로 정리하고, 각 항목마다 임상적 결과와
대처 방안을 제시해줘.

환자 정보:
- 연령/성별: 68세 여성
- 체중: 52kg
- 신기능: eGFR 45 mL/min/1.73m² (CKD stage 3b)
- 간기능: 정상
- 과거력: 심방세동, 제2형 당뇨, 골다공증, 불면증

현재 처방:
1. 메트포르민 500mg BID (당뇨)
2. 와파린 5mg QD (심방세동, 현재 INR 2.5)
3. 아미오다론 200mg QD (심방세동 리듬 조절)
4. 아토르바스타틴 40mg QN (이상지질혈증)
5. 에소메프라졸 20mg QD (위장보호)
6. 졸피뎀 10mg QHS (불면증)
7. 칼시트리올 0.25mcg QD (골다공증)
8. 세티리진 10mg QD (알레르기 비염)

요청:
- 심각한 상호작용 먼저
- 신기능 저하로 용량 조정 필요한 약물
- 고령 환자에서 주의가 필요한 약물
- 대체 약물 추천 (있는 경우)
```

AI 출력 예시 (요약):
```
[처방 검토 결과]

⚠ 심각 등급 상호작용:

1. 아미오다론 ↔ 와파린 (심각)
기전: 아미오다론이 CYP2C9 억제 → 와파린 대사 감소 → INR 급상승
위험: 출혈 위험 3-5배 증가 (뇌출혈, 위장관 출혈)
현황: INR 2.5 — 아미오다론 시작 후 2-4주 내 INR 4-6으로 상승 가능
권고: 와파린 용량 30-50% 감량 + 주 1-2회 INR 모니터링
대안: 다비가트란 150mg BID 고려 (단, eGFR 45이면 110mg BID)

2. 메트포르민 (신기능 저하, eGFR 45) (중등도-심각)
기준: eGFR 30-45 → 계속 사용 가능하나 용량 주의
       eGFR < 30 → 금기
현재 500mg BID: 허용 범위이나 조영제 검사 전 반드시 중단
권고: eGFR 분기별 모니터링, 구역/구토/복통 시 즉시 중단

⚠ 중등도 상호작용:

3. 졸피뎀 10mg — Beers Criteria 위반 (65세 이상 금기)
위험: 낙상 위험 2.3배, 인지 기능 저하, 다음날 진정
권고: 10mg → 5mg으로 감량 후 비약물 치료(수면 위생, CBT-I) 병행
대안: 멜라토닌 서방형 2mg (Circadin) 고려

4. 아미오다론 ↔ 아토르바스타틴
기전: 아미오다론이 CYP3A4 억제 → 스타틴 농도 상승
위험: 근육병증(근육통, CK 상승), 횡문근융해증
권고: 아토르바스타틴 최대 20mg으로 감량 또는 로수바스타틴 10mg으로 교체

⚠ 모니터링 필요:

5. 세티리진 (신기능 저하)
권고: eGFR 45에서 용량 절반(5mg) 권장
```

## 프롬프트 2: 신기능 저하 환자 용량 조정 확인

```
다음 처방에서 신기능 저하(eGFR [수치])를 고려한
용량 조정 필요 약물을 확인해줘.

환자 eGFR: [수치] mL/min (CKD stage [단계])
처방 목록:
- [약물명 용량 횟수]
- [약물명 용량 횟수]
- ...

출력 형식:
| 약물명 | 현재 용량 | eGFR 기준 권장 용량 | 조정 필요 여부 | 근거 가이드라인 |

주의사항: 투석 중인 경우 별도 표기, ESRD 시 완전 금기 약물 강조
```

## 프롬프트 3: 고령 환자 Beers Criteria 확인

```
다음 처방에서 2023 Beers Criteria (미국노인의학회)에 해당하는
약물을 확인하고, 각 약물의 위험과 대체 옵션을 제시해줘.

환자: [나이]세 [성별], 체중 [kg]
처방 목록:
- [약물 목록]

출력 형식:
- Beers 해당 약물 목록 (있는 경우)
- 각 약물의 위험 (낙상, 인지 저하, 부정맥 등)
- 한국에서 처방 가능한 대체 약물
- Beers 해당 없음 시 "확인한 처방 목록에서 Beers 기준 해당 약물 없음" 명시

참고: 2023 AGS Beers Criteria 기준 사용
```

## 중요: AI는 참고용, 교차확인 필수

AI 처방 검토는 첫 번째 스캔 단계다. 임상 의사 결정 전에 반드시 다음을 수행한다:

| 단계 | 도구 | 목적 |
|------|------|------|
| 1차 스캔 | AI (ChatGPT/Claude) | 전체 상호작용 빠른 탐색 |
| 2차 확인 | Lexi-Interact / Micromedex | 근거 문헌 기반 심각도 확인 |
| 3차 판단 | 임상 약사 협의 | 복잡한 케이스 최종 결정 |
| 4차 기록 | EMR 입력 | 검토 사실 문서화 |

AI가 자주 놓치는 것:
- 환자별 개인 반응 이력 (예: 과거 스타틴 근육통 경험)
- 한국 미허가 약물의 최신 상호작용 데이터
- 허가 외 사용(off-label) 약물의 상호작용

## 핵심 정리

- 프롬프트 1: 전체 다제약물 스캔 — 심각도순 정렬, 신기능·고령 고려
- 프롬프트 2: 신기능 저하 환자 용량 조정 전용 — CKD 단계별 표 출력
- 프롬프트 3: Beers Criteria 확인 — 65세 이상 처방 필수 점검
- AI 출력은 1차 스캔 — Lexi-Interact/Micromedex로 반드시 교차확인
- 아미오다론+와파린처럼 실제 임상에서 자주 놓치는 심각한 조합을 AI가 빠르게 발견
