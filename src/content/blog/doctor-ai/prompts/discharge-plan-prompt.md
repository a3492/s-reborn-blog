## 한줄 요약

퇴원 당일 가장 바쁜 10분 — 퇴원 요약, 환자용 안내문, 의원 회신 서한 세 문서를 AI 프롬프트로 30분에서 5분으로 줄인다.

## 퇴원 계획 문서의 종류

퇴원 시 생성이 필요한 문서:
1. **퇴원 요약 (Discharge Summary)**: 의사-의사 간 커뮤니케이션, 의학 용어
2. **환자용 퇴원 안내문**: 환자·보호자 이해 수준, 초등학생도 이해 가능한 언어
3. **의원 회신 서한 (Referral Letter)**: 지역 의원 주치의에게 인계 내용 전달

## 탈식별화 방법

입력 전: 환자명 → [환자] 또는 가명 / 등록번호 → 제거 / 날짜 → 연도만 또는 유지
출력 후: 실제 환자 정보로 교체 후 의사 검토

## 프롬프트 1: 퇴원 요약 (SOAP 형식)

```
다음 임상 정보를 바탕으로 퇴원 요약(Discharge Summary)을 SOAP 형식으로 작성해줘.
수신자는 전문의이므로 의학 용어를 그대로 사용해도 됩니다.

기본 정보:
- 환자: [나이/성별], 입원 기간: [YYYY-MM-DD ~ YYYY-MM-DD] (○일)
- 주진단(KCD 코드): [진단명]
- 부진단: [동반질환 목록]

S (Subjective):
- 입원 시 주호소: [증상]
- 관련 과거력: [기저질환, 수술력, 약물 알레르기]

O (Objective):
- 활력징후: [BP, HR, RR, BT, SpO2]
- 주요 검사 결과: [혈액, 영상, 특수검사]
- 시술/수술: [날짜, 명칭]

A (Assessment):
- 입원 경과 요약: [호전/악화/합병증]

P (Plan):
- 퇴원 약물: [약물명, 용량, 기간]
- 추적 관찰: [○주 후 외래, 검사 항목]
- 응급 귀원 기준: [증상 목록]
- 활동 제한: [있으면 기재]

출력 형식: 항목별 소제목, 전문 의학 용어 사용, A4 1페이지 분량
```

**실제 케이스 (심근경색 후 퇴원)**:

입력:
```
환자: 58세 남성, 입원: 2026-03-20 ~ 2026-03-25 (5일)
주진단: 급성 전벽 ST분절 상승 심근경색(I21.0)
부진단: 고혈압(I10), 제2형 당뇨(E11.9), 흡연(30갑년)
입원 시 주호소: 흉통 2시간, NRS 8/10
처치: PCI(스텐트 삽입, LAD 근위부), 성공적
퇴원 시 EF: 45%
퇴원 약물: 아스피린 100mg, 티카그렐러 90mg BID, 아토르바스타틴 40mg, 메트로프롤롤 25mg BID, 람이프릴 5mg
외래: 2주 후 심장내과
```

AI 출력 예시:
```
[퇴원 요약]

환자: 58세 남성 | 입원: 2026-03-20 ~ 2026-03-25 (6일)
주진단: 급성 전벽 STEMI (I21.0)
부진단: 고혈압 (I10), 2형 당뇨 (E11.9), 흡연력 30갑년

S: 흉통 2시간 지속(NRS 8) 후 응급실 내원. 과거 심혈관 이벤트 없음.
   약물 알레르기 없음.

O: 입원 시 BP 152/94, HR 88, SpO2 97%. ECG: II, III, aVF ST 하강,
   V1-V4 ST 상승 2-4mm. Troponin I 최고치 48.3 ng/mL.
   긴급 PCI 시행: LAD 근위부 99% 폐쇄, DES 삽입 성공.
   퇴원 전 심초음파: EF 45%, 전벽 운동저하.

A: PCI 후 안정적 경과. Killip class I 유지. 합병증 없음.

P:
약물: 아스피린 100mg QD + 티카그렐러 90mg BID (12개월),
      아토르바스타틴 40mg QN, 메트로프롤롤 25mg BID,
      람이프릴 5mg QD
외래: 2026-04-08 심장내과 (운동부하검사 예정)
귀원 기준: 흉통 재발, 호흡곤란, 실신
제한: 2주간 운전 금지, 무거운 물건(>5kg) 들기 금지
```

## 프롬프트 2: 환자용 퇴원 안내문 (초등학생 수준)

```
다음 의학적 내용을 환자와 보호자가 쉽게 이해할 수 있는 퇴원 안내문으로 바꿔줘.
초등학교 5-6학년 수준의 한국어를 사용하고, 의학 용어는 괄호 안에 쉬운 말로 설명해줘.

진단: [진단명]
퇴원 후 복용할 약: [약물 목록]
조심해야 할 증상 (이런 증상이 생기면 즉시 응급실로): [목록]
해도 되는 것 / 하면 안 되는 것: [활동 제한]
다음 진료 날짜: [날짜, 병원, 진료과]

형식:
- 제목: 퇴원하시는 [환자명]님께
- 소제목 4개: 1) 어떤 병이었나요? 2) 집에서 드실 약 3) 이런 증상이 생기면 즉시 응급실로!
  4) 다음 진료 일정
- 마지막에 병원 전화번호 기재란 추가
- 전체 A4 반 페이지 이내
```

## 프롬프트 3: 의원 회신 서한 (지역 주치의용)

```
다음 입원 케이스에 대해 지역 의원 주치의에게 보내는 회신 서한을 작성해줘.
수신 의원이 이후 관리를 이어받을 수 있도록 핵심 임상 정보를 간결하게 전달해줘.

수신: [의원명] [의사명] 선생님
발신: [병원명] [과명] [의사명]
환자: [이름/나이/성별]
입원 요약: [핵심 3줄]
퇴원 약물: [전체 목록]
향후 모니터링 요청 사항: [예: 혈압, 혈당, INR 등]
다음 대학병원 외래: [날짜]
연락처: [담당 의사 또는 과 연락처]

형식: 공식 서한체, A4 반 페이지, 인사말로 시작
```

## 개인정보 탈식별화 체크리스트

AI 입력 전 확인:
- [ ] 환자 실명 제거 또는 가명 처리
- [ ] 주민등록번호 제거
- [ ] 병원 등록번호 제거
- [ ] 날짜는 연도 수준만 남기거나 유지 (기관 정책 따름)
- [ ] 희귀 질환 + 소수 집단 조합으로 재식별 가능한 경우 추가 수정

## 핵심 정리

- 프롬프트 1: 의사-의사 SOAP 퇴원 요약 — 전문 의학 용어 사용
- 프롬프트 2: 환자용 안내문 — 초등학생 수준, 응급 귀원 기준 강조
- 프롬프트 3: 지역 의원 회신 서한 — 인계 포인트 중심 간결 작성
- 심근경색 퇴원처럼 복잡한 케이스도 3개 문서 총 5분 내 초안 완성 가능
- 모든 AI 출력은 담당 의사 최종 검토 후 환자에게 제공
