## 한줄 요약

"수치가 조금 높네요"가 환자에게 전달되지 않는 이유 — AI 프롬프트로 수치를 맥락과 행동 지침이 담긴 설명으로 바꾼다.

## 건강검진 결과 설명의 핵심 원칙

효과적인 검진 결과 설명은 세 가지를 담아야 한다:
1. 현재 수치의 의미: 정상 범위와 비교해 어떤 상태인지
2. 위험도 맥락화: 단순 '높다'가 아닌, 실제 건강에 어떤 영향인지
3. 행동 지침: 다음 진료 전까지 환자가 할 수 있는 것

---

## 프롬프트 1: 혈액검사 해석 (환자용)

```
다음 혈액검사 수치를 40대 남성 환자에게 이해하기 쉽게 설명해줘.
의학 용어는 피하고, 실생활 비유를 사용해서 중학생도 이해할 수 있도록 해줘.
각 수치에 대해: 지금 상태 → 의미 → 오늘부터 할 수 있는 것 순서로 작성해줘.

공복혈당: 110 mg/dL (정상: 70-100 mg/dL)
당화혈색소(HbA1c): 6.1% (정상: < 5.7%)
LDL 콜레스테롤: 148 mg/dL (정상: < 100 mg/dL)
HDL 콜레스테롤: 38 mg/dL (정상: > 40 mg/dL)
중성지방: 195 mg/dL (정상: < 150 mg/dL)
AST: 42 IU/L (정상: < 40 IU/L)
ALT: 58 IU/L (정상: < 40 IU/L)
감마GTP: 72 IU/L (정상: < 60 IU/L)
요산: 8.2 mg/dL (정상: < 7.0 mg/dL)

형식:
- 전체 도입 문단 (전반적 건강 상태 요약)
- 각 항목별 소제목 + 3문장 이내 설명
- 마지막에 "의사에게 꼭 물어보세요" 질문 목록 3가지
```

AI 출력 예시:
```
[건강검진 결과 해석]

전반적으로 당·지방·간 수치가 정상보다 약간 높습니다.
아직 '병'이 있는 상태는 아니지만, 지금 관리하지 않으면
3-5년 안에 당뇨, 고지혈증, 지방간으로 진행될 수 있는
'경계 단계'입니다. 지금이 가장 좋은 예방 타이밍입니다.

혈당 (110 mg/dL) — 공복혈당 장애
혈액 속 설탕이 정상보다 약간 많습니다. 당뇨는 아니지만
'당뇨 전 단계'입니다. 하얀 쌀밥, 빵, 과자 섭취를 줄이고
걷기 운동 주 5회 30분을 시작하면 6개월 내 정상으로 돌아올
확률이 58%입니다.

LDL 콜레스테롤 (148 mg/dL) — 경계 높음
혈관 벽에 기름때가 끼는 '나쁜 콜레스테롤'입니다.
당뇨 위험과 합쳐지면 심장병 위험이 2배 이상 올라갑니다.
삼겹살, 버터, 튀김 대신 생선, 올리브오일, 견과류로 바꿔보세요.

간 수치 (ALT 58) — 경미한 상승
간이 약간 피로한 상태입니다. 술을 마신다면 2주간 완전 금주 후
재검사를 권합니다. 복부 초음파 검사로 지방간 여부를 확인해보세요.

요산 (8.2 mg/dL) — 높음
관절에 모래알 같은 결정이 쌓이는 '통풍' 위험이 있습니다.
맥주와 내장류(간, 곱창)를 줄이고 물을 하루 2리터 이상 마시세요.

의사에게 꼭 물어보세요:
1. 당뇨 전 단계 — 생활 습관 교정만으로 충분한가요, 약이 필요한가요?
2. LDL이 이 수준이면 지금 당장 콜레스테롤 약을 시작해야 하나요?
3. 간 수치 때문에 복부 초음파를 찍어봐야 할까요?
```

## 프롬프트 2: 검진 요약 (의사용)

```
다음 건강검진 결과를 내과 의사가 빠르게 검토할 수 있도록
임상적 의미 중심으로 요약해줘. 의학 용어를 사용해도 됩니다.

환자: [나이/성별/과거력]
주요 이상 소견: [수치 목록]
기저질환: [있으면 기재]
현재 복용 약: [있으면 기재]

출력 형식:
- 주요 이상 소견 요약 (bullet, 3줄 이내)
- 가장 시급한 확인/추가 검사 항목
- 3개월 내 생활 습관 목표
- 약물 시작 기준 근접 여부 (예: LDL >190이면 스타틴 무조건 시작)
```

## 프롬프트 3: 이상 소견 후속 조치 안내

```
건강검진에서 다음 이상 소견이 나왔습니다.
환자에게 후속 조치를 단계별로 안내하는 문자 메시지를 작성해줘.
문자는 400자 이내, 친절하고 간결하게.

이상 소견: [예: 폐 결절 8mm, 우상엽]
권고 사항: [예: 3개월 후 흉부 CT 추적 검사]
예약 방법: [예: 호흡기내과 외래 예약 또는 검진 센터 재예약]
긴급도: [높음/중간/낮음]

추가 조건:
- 불필요한 걱정을 일으키지 않되, 추적 검사의 중요성을 명확히 전달
- 병원 이름, 전화번호 기재란 포함
```

실제 출력 예시 (폐 결절 문자):
```
[건강검진 결과 안내 - ○○의원]

안녕하세요. 이번 건강검진에서 폐에 작은 결절(8mm)이
확인되었습니다. 대부분은 양성이지만, 안전을 위해
3개월 후 흉부 CT 추적 검사를 받으시길 권장합니다.

예약: 호흡기내과 외래 또는 검진센터 재예약
전화: ○○○-○○○○ (평일 09:00-17:00)

불안하신 점은 언제든 전화 주세요.
```

## 활용 팁

- 당뇨·고혈압 등 만성질환 환자의 정기 검진 결과에 프롬프트 1 활용 — 상담 시간 3분 절감
- 이상 소견 문자 발송 시 프롬프트 3으로 초안 후 간호사 확인 → 발송 워크플로
- 복잡한 케이스(여러 이상 소견 동시)는 프롬프트 2로 의사 내부 검토 후 프롬프트 1로 환자 설명

## 핵심 정리

- 프롬프트 1: 환자용 — 비유+행동 지침 포함, 중학생 수준 언어
- 프롬프트 2: 의사용 — 임상 요약, 추가 검사·약물 시작 기준 확인
- 프롬프트 3: 후속 조치 문자 — 400자 이내, 불안 최소화하며 중요성 전달
- "의사에게 꼭 물어보세요" 질문 목록이 진료 효율을 높이는 핵심 요소
- AI 출력은 의사 검토 후 환자에게 제공 — 수치 오기 여부 반드시 확인
