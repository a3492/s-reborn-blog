## 한줄 요약
AI는 패턴 인식에서 의사를 이길 수 있지만, 의료는 패턴 인식만으로 되지 않는다.

## 본문

### "AI가 방사선과 의사를 이겼다" — 뉴스의 맥락

2020년 Nature에서 Google AI가 유방암 스크리닝에서 방사선과 전문의와 동등하거나 우월한 성능을 보였다는 연구가 나왔다. 2019년에는 AI가 당뇨망막병증을 전문의 수준으로 진단했다는 논문이 Lancet에 실렸다.

이런 연구를 볼 때마다 "의사도 AI로 대체되는가"라는 질문이 나온다. 결론부터 말하면: 특정 작업에서 AI가 의사를 이길 수 있다, 그러나 의사의 역할은 그 특정 작업 하나가 아니다.

### AI 강점 vs 의사 강점: 비교표

| 능력 | AI | 의사 |
|------|----|----|
| 대규모 영상 패턴 인식 | ✅ 우월 | △ 피로, 집중력 한계 |
| 수천 개 변수 동시 처리 | ✅ 우월 | △ 제한적 |
| 24시간 일관된 성능 | ✅ 우월 | ✗ 피로에 취약 |
| 학습 데이터 외 상황 대응 | ✗ 취약 | ✅ 적응 가능 |
| 환자 감정·사회적 맥락 이해 | ✗ 불가 | ✅ 핵심 역량 |
| 치료 결정의 윤리적 책임 | ✗ 불가 | ✅ 법적 의무 |
| 비언어적 소통 (표정, 자세) | ✗ 불가 | ✅ 진단 단서 |
| 응급 상황 즉흥 대응 | ✗ 학습 범위 밖 | ✅ 필수 역량 |
| 희귀 증례·드문 조합 대처 | ✗ 데이터 부족 | ✅ 추론 가능 |

### 의사가 대체불가한 5가지 이유

#### 1. 전인적 판단 (Holistic Judgment)

환자의 질환은 이미지와 수치만으로 구성되지 않는다. 75세 폐암 환자에게 수술을 권고할지 결정할 때 AI가 계산하는 것:

```
AI가 처리하는 정보:
  CT 영상 → 종양 크기 2.3cm, 위치, 형태
  PET → 대사 활성도
  혈액 검사 → 종양표지자
  → 출력: "수술 가능성 71%"
```

그러나 의사가 고려하는 것:
- 환자 본인의 치료 의지와 두려움
- 가족 상황 (혼자 사는 독거노인인지, 돌봄 여력)
- 이전 항암 치료 후 삶의 질 변화에 대한 환자의 경험
- "나는 수술보다 남은 시간을 잘 살고 싶다"는 환자 가치관

이 판단을 AI가 대신할 수 없다. 수치화할 수 없는 인간적 맥락이 치료 결정의 핵심이기 때문이다.

#### 2. 임상 맥락 이해 — AI 영상 판독이 놓친 것

실제 사례(보고된 유형):

```
상황:
  68세 남성, 흉부 X-ray AI 판독
  AI 결과: "좌하엽 음영 — 폐렴 가능성 78%"
  AI 권고: "항생제 치료 고려"

방사선과 전문의가 추가로 확인한 것:
  - 환자가 같은 날 아침 달리기 중 넘어져 갈비뼈를 다침
  - 환자 기저질환: 심부전 (BNP 1,240)
  - 당일 체중 3kg 증가
  - 청진: 양측 기저부 수포음

실제 진단: 심부전 급성 악화 (폐부종)
치료: 항생제가 아닌 이뇨제

AI는 영상 패턴만 봤다. 의사는 영상 + 병력 + 신체 검진 + 검사 결과를 통합했다.
```

AI가 "폐렴"이라고 할 때 항생제를 시작했다면, 실제 진단인 심부전은 악화되었을 것이다.

#### 3. 윤리적 책임과 법적 의무

한국 의료법 제2조는 의료행위의 주체를 의료인으로 규정한다. AI는 법적 주체가 아니다. 진단과 치료의 결과에 대한 책임은 최종 결정을 내린 의사에게 있다.

이것은 단순한 법적 형식이 아니다. 책임이 있기 때문에 의사는 신중하게 결정하고, 결정의 근거를 의무기록에 남기며, 환자에게 설명한다. 이 책임 구조가 없으면 의료는 성립하지 않는다.

#### 4. 비언어적 소통과 신체 검진

```
의사가 진찰실에서 얻는 정보 (AI 접근 불가):
  - 환자가 들어올 때 걸음걸이 (파킨슨 가능성)
  - 손의 떨림 (갑상선 기능 항진 vs 파킨슨)
  - 말할 때 호흡 패턴 (호흡 곤란 정도)
  - 눈의 황달 여부 (간 질환)
  - 복부 촉진 시 환자 반응 (복막 자극 징후)
  - "선생님, 사실 요즘 너무 힘들어요"라는 말 (우울증 스크리닝)
```

이 정보들은 AI가 텍스트나 이미지로 받을 수 없는 실시간 물리적 상호작용이다.

#### 5. 응급 상황과 불확실성 대응

표준 알고리즘 밖의 상황에서 AI는 취약하다.

```
응급실 상황:
  40세 여성, 갑작스러운 실신 후 내원
  초기 검사: 정상 ECG, 정상 CT, 정상 혈액 검사
  AI 판독: "이상 소견 없음"

응급의학과 전문의가 추가로:
  - 환자 어머니가 40대에 급사했다는 병력 청취
  - 운동 중 실신 여부 재확인 → 운동 직후 실신
  - 추가 검사: 운동부하 심초음파 → 비후성 심근병증 발견

AI가 "이상 없음"이라 했을 때 퇴원시켰다면 급사 위험
```

학습 데이터에 없는 조합, 드문 질환, 환자 병력의 숨은 맥락 — 이런 상황에서 의사의 임상 추론이 생명을 구한다.

### 협력 모델: AI + 의사 > 의사 단독 > AI 단독

여러 연구에서 일관된 결론:

```
피부암 진단 (Tschandl et al., Nature Medicine 2019):
  AI 단독: AUC 0.82
  피부과 전문의 단독: AUC 0.86
  AI + 전문의 (XAI 포함): AUC 0.91

유방암 스크리닝 (McKinney et al., Nature 2020):
  AI 단독 vs 방사선과 2명: 비슷한 성능
  AI + 방사선과 의사: 위양성 5.7% 감소, 위음성 9.4% 감소
```

이것이 의미하는 협력 모델:

| 역할 | AI | 의사 |
|------|----|----|
| 1차 스크리닝 | 대량 처리, 일관성 | - |
| 패턴 인식 지원 | 놓치기 쉬운 소견 flagging | - |
| 임상 맥락 통합 | - | 최종 판단 |
| 환자 소통 | - | 설명, 의사결정 공유 |
| 법적 책임 | - | 최종 책임 |

### 10년 후 전망

2034년의 의료는 어떨까? AI가 더 강해지더라도 의사의 역할은 없어지지 않는다 — 변형된다.

의사의 역할 변화 예측:
- 단순 반복적 영상 판독 → AI가 1차 처리, 의사는 검토·확인
- 데이터 기반 진단 지원 → AI가 제안, 의사가 맥락 통합
- 환자 소통, 치료 결정, 윤리적 판단 → 여전히 의사의 핵심 역할
- 새로운 역할: AI 아웃풋을 해석하고 검증하는 역량

"AI를 잘 다루는 의사"가 "AI만 쓰는 의사"를 이긴다. 그리고 "AI를 전혀 안 쓰는 의사"는 경쟁에서 뒤처질 것이다.

## 핵심 정리
- AI는 패턴 인식, 대규모 데이터 처리, 일관성에서 의사보다 우월하다
- 전인적 판단, 임상 맥락, 윤리적 책임, 비언어적 소통, 응급 대응은 의사만 할 수 있다
- 최선의 결과는 AI + 의사 협력 모델에서 나온다 — AI 단독이나 의사 단독보다 낫다
- 10년 후 의사의 역할은 사라지지 않고, AI를 활용하고 검증하는 방향으로 진화한다
