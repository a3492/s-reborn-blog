## 한줄 요약
AI는 학습 데이터에 내재된 편향을 그대로 반영한다 — 백인 위주 데이터로 학습한 피부과 AI는 어두운 피부톤에서 정확도가 최대 30% 하락한다.

## 개요

2019년 Science에 발표된 연구는 미국 의료 시스템에서 널리 사용되는 Optum의 건강 위험 예측 알고리즘이 흑인 환자를 체계적으로 과소평가했음을 밝혔다. 같은 중증도의 흑인 환자가 백인 환자보다 추가 진료를 받을 확률이 현저히 낮았다. 알고리즘은 "과거 의료비 지출"을 건강 상태의 대리 변수로 사용했는데, 사회경제적 이유로 의료 접근성이 낮은 흑인 환자는 의료비가 적어 "건강하다"고 판정된 것이다.

이 사례는 AI 편향이 단순한 기술적 결함이 아니라, 구조적 불평등을 자동화하고 확대하는 메커니즘임을 보여준다.

## 의료 AI 편향의 유형

### 편향 분류

| 편향 유형 | 발생 단계 | 의료 사례 |
|-----------|-----------|-----------|
| 선택 편향 | 데이터 수집 | 3차 병원 데이터만 사용 → 경증 환자 과소 대표 |
| 측정 편향 | 데이터 라벨링 | 판독의 간 일치도 낮은 데이터로 학습 |
| 인종 편향 | 학습 데이터 구성 | 피부과 AI 학습 데이터의 85%가 백인 피부 |
| 성별 편향 | 학습 데이터 구성 | 심혈관 AI가 남성 증상 패턴 위주로 학습 |
| 사회경제적 편향 | 대리 변수 사용 | 의료비 지출 = 건강 상태로 간주 (Optum 사례) |
| 자동화 편향 | 사용 단계 | 의사가 AI 결과를 무비판적으로 수용 |

### 실제 사례

피부과 AI — 피부톤 편향

2021년 JAMA Dermatology 연구에 따르면, 피부암 진단 AI의 정확도는 피부톤에 따라 유의미한 차이를 보였다:

| Fitzpatrick 피부 유형 | AI 민감도 | AI 특이도 |
|-----------------------|-----------|-----------|
| I-II (밝은 피부) | 91% | 88% |
| III-IV (중간 피부) | 85% | 83% |
| V-VI (어두운 피부) | 63% | 72% |

학습 데이터에서 Fitzpatrick V-VI 유형이 전체의 5% 미만이었던 것이 원인이다.

심혈관 AI — 성별 편향

여성의 심근경색 증상은 남성과 다르다. 흉통 대신 피로, 호흡곤란, 상복부 불편감이 주 증상일 수 있다. 남성 환자 위주로 학습한 AI는 여성 환자의 비전형적 증상을 놓칠 가능성이 높다.

## 편향 탐지 방법

### 정량적 지표

| 지표 | 정의 | 기준 |
|------|------|------|
| 인구통계적 동등성 | 각 하위집단의 양성 판정 비율이 동일 | 비율 차이 < 0.1 |
| 균등 기회 | 각 하위집단의 진양성률(TPR)이 동일 | TPR 차이 < 0.05 |
| 예측 동등성 | 각 하위집단의 양성예측도(PPV)가 동일 | PPV 차이 < 0.05 |
| 보정 | 예측 확률과 실제 발생률이 하위집단별로 일치 | 보정 곡선 검토 |

### 편향 감사(Audit) 절차

1. 데이터 감사: 학습 데이터의 인구통계학적 분포 확인 (성별, 연령, 인종, 지역)
2. 성능 분해: 하위집단별 민감도, 특이도, AUC를 개별 산출
3. 오류 분석: 오분류 사례가 특정 집단에 집중되는지 확인
4. 교차 검증: 외부 데이터셋(다른 병원, 다른 인구)으로 성능 재검증

## 편향 완화 전략

### 기술적 접근

| 단계 | 전략 | 방법 |
|------|------|------|
| 전처리 | 데이터 균형화 | 과소 대표 집단 오버샘플링, 합성 데이터 생성 |
| 학습 중 | 공정성 제약 추가 | 하위집단 간 성능 차이를 손실 함수에 반영 |
| 후처리 | 임계값 조정 | 하위집단별 판정 임계값을 다르게 설정 |

### 임상적 접근

- AI 결과를 해석할 때 환자의 인구통계학적 특성을 고려한다
- AI가 학습한 데이터의 인구 구성과 해당 환자의 특성이 일치하는지 확인한다
- 과소 대표 집단 환자에서는 AI 결과에 대한 독립 검토를 강화한다

## 의료진 체크리스트

- [ ] 도입하려는 AI의 학습 데이터 인구통계 분포를 제조사에 요청
- [ ] 하위집단별(성별, 연령대) 성능 지표를 확인
- [ ] 우리 병원 환자 구성과 AI 학습 데이터 구성이 유사한지 비교
- [ ] 과소 대표 집단 환자에 대한 추가 검토 프로토콜 수립
- [ ] AI 오류 사례 수집 시 인구통계 정보를 함께 기록

## 핵심 정리

1. AI 편향은 학습 데이터의 구조적 불균형에서 비롯된다 — 기술이 아니라 데이터의 문제다
2. 피부과, 심혈관, 안과 등 다양한 분야에서 인종·성별 편향이 실증적으로 확인되었다
3. 편향 탐지는 하위집단별 성능 분해(subgroup analysis)로 시작한다
4. 의료진은 AI 제조사에 학습 데이터 구성과 하위집단별 성능을 반드시 요구해야 한다
