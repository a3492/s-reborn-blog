## 한줄 요약
의료 특화 AI 앱 중 Glass Health는 감별진단 생성에, Isabel은 진단 정확도에, Ada Health는 환자 대면 트리아지에 각각 강점을 보인다.

## 개요

ChatGPT, Gemini 같은 범용 LLM과 달리, 의료 특화 AI 앱은 의학 데이터로 파인튜닝되어 환각 위험이 낮고, 임상 워크플로우에 최적화되어 있다. 2024년 BMJ Digital Health 리뷰에서 의료 특화 AI의 감별진단 정확도가 범용 LLM보다 평균 12% 높았다.

의사용(Glass Health, Isabel, DXplain)과 환자용(Ada Health, Buoy Health)으로 나뉘며, 용도와 예산에 따라 선택이 달라진다.

## Glass Health

### 기본 정보
| 항목 | 내용 |
|------|------|
| 개발사 | Glass Health (San Francisco) |
| 출시 | 2023년 |
| 가격 | 무료 베타 (2026년 기준) |
| 모델 | GPT 기반 + 의학 교과서/가이드라인 학습 |

### 강점
- 증상/소견 입력 시 감별진단 목록 + 각 진단별 근거를 생성
- 선택한 진단에 대한 치료 계획(Plan)까지 초안 작성
- 의사가 실제 쓰는 SOAP note 형식에 최적화
- 범용 ChatGPT보다 환각이 적고, 근거 기반 설명 제공

### 약점
- 무료 베타이므로 장기 가격 정책 불확실
- 한국어 미지원, 한국 보험 기준 치료 계획 미반영
- 드문 질환에서는 여전히 부정확할 수 있음

## Isabel Healthcare

### 강점
- 1999년부터 운영된 감별진단 도구의 원조 — 20년 이상 임상 검증
- 10,000개 이상 질환 DB로 희귀 질환 커버리지 최고
- Red flag 증상 자동 경고 기능
- 여러 연구에서 진단 정확도 96% 이상 보고 (Top-20 감별진단 기준)

### 약점
- 연간 $700 비용
- 치료 계획 생성 기능 없음 — 감별진단 전용
- 인터페이스가 구식, 모바일 최적화 부족

## DXplain

### 강점
- MIT/MGH 개발, 의학 교육 목적으로 설계
- 각 감별진단에 대한 교과서 수준 설명 제공
- 의대생, 전공의 교육에 최적

### 약점
- 기관 구독만 가능, 개인 사용 불가
- 2,400개 질환으로 DB 규모가 상대적으로 작음
- 실시간 가이드라인 업데이트 느림

## Ada Health

### 강점
- 환자가 직접 증상을 입력하면 가능한 질환과 긴급도를 안내
- 12개 언어 지원 (한국어 일부)
- 의사 버전(Ada for Doctors)도 제공
- 2024년 연구에서 증상 체커 중 가장 높은 정확도 기록

### 약점
- 환자용 도구이므로 의사의 임상 워크플로우에 최적화되지 않음
- 심각한 질환을 과소평가하는 보수적 경향
- B2B 모델로 전환 중이라 무료 접근 제한 가능성

## Buoy Health

### 강점
- 대화형 인터페이스로 환자 친화적
- 증상 입력 후 적절한 진료과/긴급도 안내
- 미국 여러 의료 시스템과 연동

### 약점
- 진단 DB 규모가 1,700개로 가장 작음
- 의사용 도구로는 부적합
- 한국어 미지원

## 실제 임상 사용 시나리오

**권장하는 사용법:**
```
시나리오: 전공의, 드문 증상 조합의 감별진단이 떠오르지 않을 때
입력: Glass Health에 "23세 여성, 반복 구강 궤양, 외음부 궤양,
포도막염"
→ 베체트병, 크론병, HSV, SLE 등 감별진단 목록 생성
→ 각 진단별 추가 확인 검사 제안
→ UpToDate에서 교차 확인 후 검사 오더
```

**피해야 하는 사용법:**
```
X Ada Health 결과를 환자에게 "진단"으로 전달
  → 증상 체커는 트리아지 도구, 확정 진단이 아님
X Glass Health 치료 계획을 검증 없이 그대로 사용
  → 한국 보험 기준, 약가, 사용 가능 약물 확인 필요
```

## 핵심 정리
- **Glass Health** — 감별진단 + 치료 계획 초안이 필요한 전공의/주치의에게 추천
- **Isabel** — 희귀 질환까지 포괄하는 최고 정확도의 감별진단 도구
- **Ada Health** — 환자 대면 트리아지, 외래 사전 문진에 유용
- 모든 도구의 결과는 반드시 **임상 판단과 교차 확인** 후 적용
- 한국어 지원 도구가 제한적이므로 영어 입력에 익숙해질 필요
