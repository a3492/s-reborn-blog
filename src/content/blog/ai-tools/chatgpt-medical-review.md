## 한줄 요약
ChatGPT는 의료 교육, 문서 초안 작성, 감별진단 브레인스토밍에는 유용하지만, 환자 진료에 직접 활용하려면 환각과 최신 정보 부재 문제를 반드시 인지해야 한다.

## 개요

OpenAI의 ChatGPT는 의료진이 가장 많이 사용하는 범용 AI 모델 중 하나다. 2023년 JAMA, NEJM 등에서 의사 국가시험(USMLE) 합격 수준의 성능이 보고된 이후, 임상 현장 도입이 빠르게 늘었다. 하지만 "의료에 쓸 수 있다"와 "의료에 안전하게 쓸 수 있다"는 다른 문제다.

의사, 전공의, 의대생이 실제로 ChatGPT를 어떻게 활용하고, 어디서 멈춰야 하는지 정리한다.

## 기본 정보

## 강점 — 의료 현장에서 실제로 잘하는 것

### 감별진단 브레인스토밍
"37세 여성, 발열 3일, 우상복부 통증, Murphy sign 양성, 백혈구 14,000"이라고 입력하면 담낭염, 담관염, 간농양, 우측 신우신염 등을 체계적으로 나열한다. 누락된 진단을 빠르게 확인하는 용도로 유용하다. 단, 최종 감별진단은 반드시 임상의가 판단한다.

### 의학 교육 및 설명
복잡한 병태생리를 환자 수준으로 쉽게 설명하거나, 전공의에게 기전을 설명하는 초안을 만드는 데 탁월하다. "당뇨병을 초등학생에게 설명해줘"부터 "GLP-1 수용체 작용제의 심혈관 보호 기전을 전공의에게 설명해줘"까지 수준 조절이 잘 된다.

### 문서 초안 작성
퇴원요약, 회신 서한, 진료 안내문 초안을 빠르게 만든다. 2024년 UCSF 연구에서 ChatGPT가 작성한 퇴원 요약이 의사 작성본과 유사한 수준임을 확인했다(단, 개인정보 입력 주의).

### 논문 요약 및 번역
영문 논문 초록을 한국어로 요약하거나, Methods 섹션의 통계 분석 방법을 설명하는 데 유용하다. RAG 기반 도구(Consensus, Elicit)보다 정확도는 낮지만 속도가 빠르다.

### 코딩 및 데이터 분석
R, Python을 이용한 임상 데이터 분석 코드 작성을 도와준다. SPSS 대신 R을 배우려는 의사들에게 유용한 코딩 튜터가 된다.

## 약점 — 의료 맥락에서 반드시 알아야 할 한계

### 환각(Hallucination)
존재하지 않는 논문을 그럴듯하게 인용하고, 잘못된 약물 용량을 자신 있게 제시한다. 2023년 JAMA 연구에서 ChatGPT의 의학 참고문헌 22%가 가짜였다. **약물 용량, 가이드라인 수치는 반드시 교차 확인 필수.**

### 최신 정보 부재
학습 데이터 컷오프 이후 승인된 신약, 변경된 가이드라인을 모른다. 2024년 이후 승인된 약물(예: tarlatamab, tebentafusp)은 답변하지 못하거나 틀리게 답한다.

### 개인정보 보호 위험
기본 ChatGPT(무료/Plus)에 실제 환자 정보를 입력하면 개인정보보호법 위반 소지. Enterprise 플랜에서 HIPAA BAA 체결 후 사용하거나, 탈식별화 후 입력해야 한다.

### 근거 없는 확신
"가능성이 높습니다", "대부분의 경우"라는 표현을 자주 쓰지만 실제 확률 근거가 없다. 틀린 답변도 맞는 답변과 동일한 자신감으로 표현된다.

## 실제 임상 사용 시나리오

**권장하는 사용법:**
```
시나리오: 당직 중 드문 발진 패턴을 만났을 때
입력: "성인 남성, 갑작스러운 발열과 함께 손바닥/발바닥에
집중된 구리색 반점성 발진. 감별진단 목록 만들어줘"
→ 2기 매독, Rocky Mountain Spotted Fever, 수족구 등
   감별진단 아이디어를 빠르게 얻음
→ UpToDate에서 확인 후 최종 판단
```

**피해야 하는 사용법:**
```
X "이 환자에게 메트포르민 용량을 얼마나 줘야 해?"
  → 용량 계산은 약전/전문 도구로
X "이 약 상호작용 안전해?"
  → Lexi-Interact, Drugs.com 등 전용 DB 사용
X 실제 환자 이름, 생년월일, 병록번호를 포함한 입력
  → 개인정보 위반
```

## ChatGPT vs 의료 특화 AI 비교

| 항목 | ChatGPT (일반) | ChatGPT Enterprise | 의료 특화 AI (예: Glass.health) |
|------|---------------|--------------------|---------------------------------|
| 의학 지식 수준 | USMLE 합격 수준 | 동일 | 의료 특화 훈련 |
| 최신 가이드라인 | 컷오프 이후 미반영 | 동일 | 일부 업데이트 |
| HIPAA | 미지원 | BAA 가능 | 기본 지원 |
| 환각 위험 | 높음 | 동일 | 상대적으로 낮음 |
| 가격 | 무료/$20 | $30+/월 | $50–200+/월 |
| 사용 편의성 | 매우 높음 | 높음 | 중간 |

## 핵심 정리
- ChatGPT는 **의사의 사고를 보조하는 도구**이지, 임상 결정을 내리는 도구가 아니다
- 감별진단 아이디어, 의학 교육, 문서 초안 작성에서 실질적 가치가 있다
- 약물 용량, 가이드라인 수치, 논문 인용은 반드시 원본 소스로 교차 확인
- 실제 환자 정보 입력 시 개인정보 보호 설정 필수 (Enterprise 또는 탈식별화)
- 2024년 이후 신약이나 변경된 가이드라인은 모를 수 있다
