## 한줄 요약
Claude는 긴 임상 문서 처리, 신중한 답변 태도, 환자 설명문 작성에서 두각을 나타내며, 의료진이 "믿고 쓸 수 있는" AI로 평가받고 있다.

## 개요

Anthropic이 개발한 Claude는 "안전하고 유용하며 정직한 AI"를 목표로 Constitutional AI 방법론으로 훈련된 모델이다. ChatGPT와 Gemini에 비해 대중적 인지도는 낮지만, 의료계에서는 오히려 긴 문서 처리 능력과 신중한 답변 태도로 높은 평가를 받는다.

특히 임상 기록 요약, 진료 가이드라인 분석, 환자 동의서 초안 작성 등 정확성이 중요한 업무에서 선호도가 높다.

## 강점 — 의료 현장에서 Claude가 잘하는 것

### 긴 문서의 정확한 처리
200,000 토큰의 컨텍스트 윈도우는 실제로 임상에서 차이를 만든다. 50페이지 분량의 진료 기록, 전체 임상시험 프로토콜, 복수의 영문 가이드라인 원문을 한 번에 입력하고 요약이나 비교 분석을 요청할 수 있다.

```
실제 활용 예시:
- 전원 환자의 수십 장 의무기록을 붙여넣고 "주요 과거력과 현재 투약 목록 추출"
- ACC/AHA 심부전 가이드라인 전문 입력 → "LVEF 35% 환자에 대한 권고사항 정리"
- 다기관 임상시험 프로토콜 분석 → "주요 배제 기준 목록화"
```

### 신중하고 정직한 답변 태도
Claude는 불확실한 내용에 대해 "제가 확신하기 어렵습니다" 또는 "최신 가이드라인을 직접 확인하시길 권합니다"라는 표현을 ChatGPT보다 훨씬 자주 사용한다. 이는 Constitutional AI 훈련의 결과로, 의료처럼 오류 비용이 높은 분야에서 실질적인 안전장치가 된다.

### 환자 설명문/동의서 초안 작성
복잡한 의학 용어를 환자 눈높이에 맞게 바꾸는 작업에서 탁월하다. "다발성경화증 진단을 받은 40대 환자에게 나탈리주맙 치료를 시작하기 전 설명할 내용을 중학교 수준의 언어로 작성해줘"라는 요청에 대해, Claude는 의학적 정확성을 유지하면서도 이해하기 쉬운 문장을 구성한다.

### 논리적 추론과 근거 제시
복잡한 임상 시나리오에서 감별진단 추론 과정을 단계별로 명시한다. "왜 그렇게 생각하는가"에 대한 논리적 설명이 포함되어 있어, 단순한 답변 목록보다 교육적 가치가 높다.

### 윤리적 민감 사안 처리
자살 위험 평가, 치료 중단 결정, 취약 계층 환자 상담 등 윤리적 복잡성이 높은 주제에서 적절한 경계를 지키며 답변한다. 섣불리 처방을 권유하거나 진단을 단언하지 않는다.

## 약점 — 반드시 알아야 할 한계

### 멀티모달 기능 제한
Claude 3의 비전 기능은 개선되었지만, 의료 영상(CT, MRI, X-ray, 병리 슬라이드) 분석에서는 전문 의료 AI(Aidoc, Viz.ai 등)에 비해 크게 부족하다. 영상 분석 목적으로는 적합하지 않다.

### 국내 의료 정보 약세
한국 의료 환경 특유의 정보(국내 보험 급여 기준, 국내 신약 허가 현황, 한국 가이드라인)에서 오류나 누락이 발생한다. 한국 의료 현실에 맞는 답변이 필요할 때는 추가 교차 확인이 필요하다.

### 실시간 의학 정보 없음
ChatGPT와 동일하게 학습 데이터 컷오프 이후 정보를 모른다. 최신 임상시험 결과나 2024년 이후 신규 허가 약물에 대해서는 오답을 낼 수 있다.

### 웹 검색 기능 기본 미포함
기본 Claude.ai는 웹 검색이 제한적이다. ChatGPT는 기본적으로 Bing 검색을 연동하므로 최신 정보 접근에서 유리한 경우가 있다.

## 실제 임상 사용 시나리오

**긴 문서 분석 (Claude의 강점):**
```
입력: [전원 의뢰서 전문 + 수십 항의 이전 검사 결과]
요청: "이 환자의 지난 2년간 항생제 사용 이력과
      내성균 배양 결과를 시간 순서대로 정리해줘"
→ 정확하고 구조적인 요약 생성
```

**환자 설명문 작성:**
```
요청: "신장 이식 후 타크로리무스를 복용 중인 환자에게
      자몽 주스를 피해야 하는 이유를 쉽게 설명하는
      반 페이지 분량의 안내문 작성해줘"
→ CYP3A4 억제 기전을 환자 언어로 정확하게 설명
```

## ChatGPT vs Claude vs Gemini 비교

| 항목 | ChatGPT (GPT-4o) | Claude 3.7 Sonnet | Gemini 1.5 Pro |
|------|------------------|-------------------|----------------|
| 긴 문서 처리 | 128K 토큰 | 200K 토큰 | 1M 토큰 |
| 의학 지식 정확도 | 높음 | 높음 | 높음 |
| 신중한 답변 태도 | 보통 | 높음 | 보통 |
| 멀티모달(이미지) | 있음 | 있음(제한적) | 있음(강함) |
| 한국어 품질 | 높음 | 높음 | 높음 |
| HIPAA 지원 | Enterprise | Enterprise | Workspace |
| 가격(월) | $20 | $20 | $20~무료 |

## 핵심 정리
- Claude의 핵심 강점은 **긴 문서 처리 + 신중한 답변 태도 + 정직한 불확실성 표현**
- 50페이지 이상의 의무기록 요약, 복잡한 가이드라인 분석에서 차별화된 성능
- 불확실한 내용을 솔직히 인정하는 태도가 의료 맥락에서 오히려 안전
- 의료 영상 분석이나 실시간 최신 정보가 필요한 작업에는 적합하지 않다
- 환자 동의서, 퇴원 안내문 등 환자용 문서 작성에 특히 추천
