## 한줄 요약
AI 소프트웨어가 질병을 진단하거나 치료를 권고하면 의료기기다 — 식약처 허가 없이 임상에 사용하면 의료기기법 위반이다.

## 개요

병원에서 AI 흉부 X-ray 판독 소프트웨어를 도입하려 한다. 이 소프트웨어는 폐결절을 검출하고 악성 확률을 수치로 제시한다. 이것이 의료기기인가? 허가가 필요한가? 허가를 받으려면 무엇이 필요한가?

식품의약품안전처(식약처)는 2020년부터 AI 기반 소프트웨어를 SaMD(Software as a Medical Device)로 분류하고, 위험도에 따라 1~4등급으로 나누어 허가·인증·신고 절차를 적용하고 있다.

## 의료기기 분류 기준

### SaMD 해당 여부 판단

| 기능 | 의료기기 해당 | 비의료기기 |
|------|:---:|:---:|
| CT 영상에서 폐결절 검출 | O | |
| 심전도 부정맥 자동 분류 | O | |
| 당뇨 환자 혈당 추세 예측 | O | |
| 병원 예약 일정 관리 AI | | O |
| 의료 문헌 검색 보조 | | O |
| 환자 증상 기록 음성 변환 | | O |

식약처 기준: 질병의 진단, 치료, 예방, 모니터링에 직접 관여하는 소프트웨어는 의료기기다. 단순 행정·기록·검색 기능은 해당하지 않는다.

### 등급 분류

| 등급 | 위험도 | 허가 절차 | 예시 |
|------|--------|-----------|------|
| 1등급 | 최저 | 신고 | 건강 기록 관리 SW |
| 2등급 | 낮음 | 인증 (식약처 지정 기관) | 의료 영상 저장·전송(PACS) |
| 3등급 | 보통 | 허가 (식약처 직접 심사) | AI 흉부 X-ray 판독 보조 |
| 4등급 | 높음 | 허가 + 임상시험 필수 | AI 기반 방사선 치료 계획 |

대부분의 AI 진단 보조 소프트웨어는 3등급에 해당한다. 환자에게 직접 치료를 제공하거나 생명 유지에 관여하면 4등급이다.

## 허가 절차

### 3등급 AI 의료기기 허가 흐름

```
1. 사전 상담 (식약처)
   ↓
2. 등급 분류 신청
   ↓
3. 기술문서 작성 (성능시험, 안전성, SW 검증)
   ↓
4. 임상 성능 시험 (다기관 전향적/후향적)
   ↓
5. 허가 신청서 제출
   ↓
6. 식약처 심사 (90~180일)
   ↓
7. 허가증 발급
   ↓
8. 시판 후 관리 (변경 허가, 이상사례 보고)
```

### 제출 서류

| 서류 | 내용 | 비고 |
|------|------|------|
| 기술문서 | 알고리즘 설명, 학습 데이터, SW 아키텍처 | IEC 62304 준수 |
| 성능시험 자료 | 민감도, 특이도, AUC, 검증 데이터셋 | 독립 테스트셋 필수 |
| 임상시험 자료 | 다기관 검증, 비열등성/우월성 설계 | 3등급 이상 |
| 사이버보안 자료 | 데이터 암호화, 접근 제어, 취약점 분석 | 2020년 가이드라인 |
| 생물학적 안전성 | 해당 시 (하드웨어 접촉) | SW 단독은 면제 |

## 실제 허가 사례

### 국내 허가된 AI 의료기기

| 제품 | 기능 | 등급 | 허가 연도 |
|------|------|------|-----------|
| Lunit INSIGHT CXR | 흉부 X-ray 판독 보조 | 3등급 | 2020 |
| VUNO Med-BoneAge | 골연령 판독 보조 | 3등급 | 2019 |
| JLK AI-RAD | 뇌 MRI 출혈 검출 | 3등급 | 2020 |
| SELVAS AI | 골밀도 예측 | 2등급 | 2021 |

2024년 기준 식약처에 허가·인증된 AI 의료기기는 200건 이상이다. 그 중 약 70%가 영상 판독 보조 분야에 집중되어 있다.

## 의료진이 확인할 사항

### 도입 전 체크리스트

- [ ] 해당 AI 소프트웨어의 식약처 허가번호 확인
- [ ] 허가된 적응증(사용 목적) 범위 확인
- [ ] 허가 조건(예: 특정 장비와만 호환) 확인
- [ ] 허가 범위 외 사용(off-label use) 시 법적 위험 인지
- [ ] 제조사의 시판 후 안전성 보고 의무 이행 여부 확인

### 미허가 AI 사용의 법적 위험

의료기기법 제5조는 허가·인증·신고 없이 의료기기를 제조·수입·판매하는 행위를 금지한다. 의료인이 미허가 AI를 임상에 사용해 환자 피해가 발생하면:

- 의료기기법 위반: 3년 이하의 징역 또는 3천만 원 이하의 벌금
- 의료법 위반: 의료인의 주의 의무 위반으로 과실 인정
- 민사 책임: 민법 제750조 불법행위 손해배상

## 모델 업데이트와 변경 허가

AI 모델이 업데이트되면 성능이 변할 수 있다. 식약처는 의도된 사용 목적 변경, 알고리즘 구조 변경, 학습 데이터 변경 시 변경 허가를 요구한다. 단순 버그 수정이나 UI 변경은 경미 변경으로 신고만 하면 된다.

| 변경 유형 | 절차 | 예시 |
|-----------|------|------|
| 중대 변경 | 변경 허가 | 새로운 질환 추가, 알고리즘 아키텍처 변경 |
| 보통 변경 | 변경 인증 | 학습 데이터 추가, 성능 개선 |
| 경미 변경 | 변경 신고 | UI 변경, 버그 수정 |

## 핵심 정리

1. AI가 질병 진단·치료·모니터링에 관여하면 의료기기이며, 식약처 허가가 필요하다
2. 대부분의 AI 진단 보조 SW는 3등급으로 분류되어 임상시험과 식약처 직접 심사를 거친다
3. 미허가 AI 임상 사용은 의료기기법 위반이며, 환자 피해 시 형사·민사 책임이 발생한다
4. AI 모델 업데이트 시 변경 허가 여부를 반드시 확인해야 한다
