## 한줄 요약
감별진단 프롬프트는 환자 정보를 구조화해서 입력하고, AI에게 likelihood 순으로 정리하게 하는 것이 핵심이다 — "흉통"만 입력하면 안 되고, 나이·성별·동반증상을 함께 줘야 한다.

## 본문

### AI가 감별진단에 강한 이유

AI는 방대한 의학 문헌을 학습했기 때문에 희귀 질환을 포함한 넓은 감별진단을 빠르게 제시할 수 있다. 특히:
- 놓치기 쉬운 드문 질환을 상기시켜준다
- cross-specialty 감별을 동시에 다룬다 (예: 흉통 → 심장 + 폐 + 소화기 + 근골격계)
- 학년·경험과 무관하게 일관된 체크리스트 역할을 한다

단, AI는 환자를 직접 보지 못한다. 신체검진 소견, 임상 직관은 의사가 직접 더해야 한다.

---

### 기본 감별진단 프롬프트 템플릿

```
당신은 [전문과] 전문의입니다.
다음 환자의 감별진단을 가능성 높은 순으로 5가지 제시해 주세요.
각 진단에 대해: (1) 이 환자에게 해당하는 근거, (2) 배제하거나 확인하는 검사를 명시해 주세요.

환자:
- 나이/성별: [예: 45세 여성]
- 주소(Chief Complaint): [예: 3일간 지속된 우상복부 통증]
- 통증 특성: [예: 식후 악화, 방사통 없음, VAS 6/10]
- 동반증상: [예: 오심, 미열 37.8도]
- 관련 병력: [예: 비만, 고콜레스테롤혈증]
- 복용 약물: [예: Statin 복용 중]
- 신체검진 주요 소견: [예: Murphy's sign 양성]
- 시행한 검사: [예: 아직 없음]
```

### 증상별 특화 템플릿

흉통
```
당신은 내과/응급의학과 전문의입니다.
흉통 환자의 감별진단을 아래 카테고리로 나눠 제시해 주세요.
각 항목에 "이 환자에게 시사하는 소견"과 "배제 검사"를 포함해 주세요.

카테고리:
1. 즉각적 생명 위협 (STEMI, 대동맥박리, 폐색전증, 긴장성 기흉)
2. 급성 (불안정 협심증, 심낭염, 식도 파열)
3. 비응급 (안정 협심증, GERD, 근골격계, 불안장애)

환자: [나이/성별/흉통 특성/동반증상/활력징후/ECG 소견]
```

두통
```
당신은 신경과 전문의입니다.
두통 환자를 1차 두통(편두통/긴장형/군발성)과 2차 두통(기질적 원인)으로 나누어
감별진단해 주세요.

Red flag 증상이 있는지 먼저 확인해 주세요.
Red flag 있으면 즉시 응급 평가 필요 여부를 먼저 명시해 주세요.

환자: [정보]
```

복통
```
당신은 외과/내과 전문의입니다.
아래 위치별로 감별진단을 정리해 주세요.

위치: [우상복/좌상복/우하복/좌하복/상복부/하복부/전복부]
각 감별진단에 해당 위치에서의 특징적 소견을 포함해 주세요.

환자: [나이/성별/통증 위치·양상/동반증상/병력]
```

발열
```
당신은 감염내과 전문의입니다.
발열 환자의 감염원을 장기계통별로 우선순위를 정해 제시해 주세요.

계통: 호흡기 / 비뇨기 / 복부 / 신경계 / 피부연조직 / 혈액/전신 / 기타

각 감염에 대해 이 환자에게 시사하는 소견과 확인 검사를 명시해 주세요.
입원 여부 기준(qSOFA 등)도 함께 평가해 주세요.

환자: [나이/성별/발열 기간·패턴/동반증상/면역상태/최근 병력]
```

### 결과를 더 유용하게 만드는 추가 지시

위험도 층화 추가
```
...각 진단에 대해 "즉각 처치 필요 / 오늘 검사 필요 / 외래 추적 가능"으로 분류해 주세요.
```

환자별 특수 상황 반영
```
...이 환자는 면역억제 상태(고용량 스테로이드 복용 중)임을 반드시 고려해 주세요.
기회감염 가능성도 포함해 주세요.
```

검사 우선순위 지정
```
...검사를 "즉시(응급) / 당일 / 비응급" 순서로 나눠 주세요.
비용 효율이 높은 검사를 우선 배치해 주세요.
```

### 한계와 주의사항

- AI는 신체검진을 못 한다. "압통 있음"이라고 입력해도 실제 반발통·근긴장도 등은 의사가 직접 평가해야 한다
- 희귀 질환을 과도하게 포함하는 경우가 있다 — 임상 맥락에서 현실적인 가능성으로 필터링할 것
- 지역적 특성(풍토병, 국내 발생률)을 반영하지 못할 수 있다

## 핵심 정리
- "흉통"만 입력하면 교과서 나열. 나이·성별·동반증상·검사까지 입력하면 임상 수준 감별진단
- 증상별 특화 템플릿을 즐겨찾기에 저장하고 [괄호]만 채워 쓰면 된다
- 결과는 항상 임상 판단으로 검토 — AI는 보조, 의사가 결정
