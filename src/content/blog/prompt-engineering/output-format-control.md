## 한줄 요약
형식을 지정하는 것은 출력 품질을 높이는 가장 빠른 방법이다 — "표로 정리해줘" 한 마디가 산문 5단락을 한눈에 보이는 비교표로 바꾼다.

## 본문

### 형식 지정이 왜 중요한가

AI 기본 출력은 산문(prose)이다. 읽기는 쉽지만:
- 정보 밀도가 낮다
- 비교가 어렵다
- 복사해서 EMR에 붙여넣기 불편하다

형식을 명시적으로 지정하면 같은 내용이 훨씬 쓸 만한 형태로 나온다.

---

### 주요 출력 형식과 사용 상황

**1. 마크다운 표**
```
...를 표로 정리해 주세요. 열: [항목1 | 항목2 | 항목3]
```
적합한 상황: 약물 비교, 가이드라인 비교, 감별진단 요약

**2. Bullet 리스트**
```
...를 bullet point로 5개 이내로 정리해 주세요.
```
적합한 상황: 핵심 요약, 처치 순서, 환자 교육 항목

**3. SOAP 노트**
```
다음 정보를 SOAP 형식으로 작성해 주세요.
S(Subjective), O(Objective), A(Assessment), P(Plan) 각 항목을 명확히 구분해 주세요.
```
적합한 상황: 진료 기록, 케이스 정리

**4. 번호 목록 (Step-by-step)**
```
...를 단계별 순서로 번호를 붙여 설명해 주세요.
```
적합한 상황: 시술 순서, 프로토콜, 환자 복약 지도

**5. Q&A 형식**
```
이 내용을 환자가 자주 묻는 질문 5개와 답변 형식으로 정리해 주세요.
```
적합한 상황: 환자 FAQ 시트, 설명문 초안

**6. JSON**
```
결과를 JSON 형식으로 출력해 주세요.
필드: name, diagnosis, medications, follow_up
```
적합한 상황: 개발·자동화 통합, 데이터 파싱

---

### 실전 예시: 약물 비교표

**프롬프트**
```
당신은 내과 전문의입니다.
다음 표 형식으로 ARB 3종을 비교해 주세요.

| 약물명 | 용량 | 반감기 | CKD 사용 | 당뇨 적응증 | 비용(30일) |
|--------|------|--------|----------|------------|------------|
| ... | ... | ... | ... | ... | ... |

대상 약물: Losartan, Valsartan, Telmisartan
```

→ 정확히 그 형식으로 채워진 비교표가 나온다. 열 이름을 직접 지정하면 AI가 빠뜨리지 않는다.

---

### 실전 예시: SOAP 노트

**프롬프트**
```
아래 진료 내용을 SOAP 노트로 정리해 주세요.
- S: 환자 주訴와 현재 증상
- O: 활력징후, 신체검진, 검사 수치
- A: 주진단 및 감별진단
- P: 처방, 검사, 교육, 추적 계획

진료 내용:
45세 남성, 혈압 높다고 내원. 두통 2주. 복용 중 약 없음.
BP 158/96, HR 78, 체중 82kg. 말초부종 없음. 소변검사 정상.
고혈압 1기로 진단, 생활습관 교정 1달 시도 후 재방문 예정.
```

---

### 길이 제어

출력 길이도 형식의 일부다.

```
3문장 이내로 요약해 주세요.
A4 반 페이지 분량으로 작성해 주세요.
핵심만 1~2줄로 말해 주세요.
200자 이내로 요약해 주세요.
```

**주의**: 한국어는 "200자"로 지정하면 글자 수 기준이 되고, "200 words"는 영어 단어 기준이 된다.

---

### 형식 지정 조합 예시

```
당신은 가정의학과 전문의입니다.
다음 환자의 고혈압 치료 계획을 작성해 주세요.

형식:
1. 진단 요약 (2문장)
2. 치료 목표 (bullet 3개)
3. 처방 계획 (표: 약물명 | 용량 | 주의사항)
4. 추적 일정 (step-by-step)
5. 환자 교육 포인트 (bullet 5개 이내)

환자: [환자 정보]
```

각 섹션에 다른 형식을 지정하면 리포트처럼 구조화된 결과물이 나온다.

## 핵심 정리
- 형식 미지정 → 장황한 산문. 형식 지정 → 즉시 쓸 수 있는 결과물
- 표는 비교할 때, bullet은 요약할 때, SOAP는 기록할 때
- 열 이름·항목 수·길이를 직접 지정하면 AI가 빠뜨리지 않는다
