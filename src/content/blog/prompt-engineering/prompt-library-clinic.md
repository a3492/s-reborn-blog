## 한줄 요약
프롬프트를 매번 새로 쓸 필요 없다 — 검증된 템플릿 30개를 즐겨찾기에 저장하고 [괄호]만 채워 쓰면 외래 효율이 달라진다.

## 본문

### 사용법

1. 아래 템플릿을 메모장/노션에 저장
2. ChatGPT/Claude를 열고 복사·붙여넣기
3. `[대괄호]` 안의 내용만 실제 환자 정보로 교체
4. 전송

---

## 🩺 진단 영역

**1. 기본 감별진단**
```
당신은 [내과/가정의학과] 전문의입니다.
다음 환자의 감별진단 상위 5가지를 likelihood 순으로 정리해 주세요.
각각: 이 환자에서의 근거 + 확인/배제 검사

환자: [나이/성별], 주소: [증상], 기간: [일/주]
동반증상: [나열], 병력: [관련 과거력], 검사: [있다면]
```

**2. Red flag 확인**
```
다음 증상이 있는 환자에서 즉각적인 처치나 응급 평가가 필요한
Red flag 소견을 확인해 주세요.

증상: [증상 및 소견]
환자: [기본 정보]

Red flag가 있다면 즉시 할 조치도 함께 제시해 주세요.
```

**3. 희귀질환 고려**
```
흔한 진단을 배제했는데도 원인을 모를 때입니다.
다음 소견이 있는 환자에서 희귀하지만 가능한 진단을 추가로 제시해 주세요.
각 진단에 확인 검사를 포함해 주세요.

소견: [증상 및 검사 결과 요약]
```

---

## 💊 처방 영역

**4. 1차 약제 선택**
```
당신은 임상약리학 전문의입니다.
다음 환자에게 [질환명] 치료를 위한 1차 약제를 추천해 주세요.

표 형식: 약물명 | 시작 용량 | 최대 용량 | 이 환자에 유리한 점 | 주의사항

환자: [나이/성별/기저질환/현재 복용 약물/알레르기]
기준: 대한[해당학회] 가이드라인 / 건강보험 급여 약제 우선
```

**5. 다제약물 검토**
```
다음 환자의 복용 약물 목록을 검토해 주세요.

확인 항목:
1. 주요 약물 상호작용 (특히 CYP450 관련)
2. 중복 계열 약물
3. 이 환자의 신기능([eGFR])을 고려한 용량 조절 필요 여부
4. 중단 또는 변경이 권장되는 약물

약물 목록: [복용 중인 약물 나열]
환자: [나이/기저질환/eGFR/체중]
```

**6. 항생제 선택**
```
당신은 감염내과 전문의입니다.
다음 감염에 대한 경험적 항생제를 선택해 주세요.

감염 부위: [부위]
환자: [나이/성별/알레르기/기저질환/최근 항생제 사용]
세팅: [외래/입원/응급]

표: 1차 선택 | 용량 | 기간 | 페니실린 알레르기 대안 | 주의사항
```

**7. 소아 용량 확인**
```
다음 약물의 소아 용량을 확인해 주세요.

약물: [약물명]
환자: [나이], 체중: [kg]
적응증: [사용 목적]

체중 기반 용량 (mg/kg) + 최대 용량을 함께 제시해 주세요.
국내 허가 용량 기준으로. 확실하지 않으면 "확인 필요" 표시.
```

## 📋 기록·문서 영역

**8. SOAP 노트 작성**
```
다음 진료 내용을 SOAP 노트로 정리해 주세요.

S: 환자 주訴, 현 병력
O: 활력징후, 신체검진, 검사 수치
A: 주진단 (r/o 포함)
P: 처방 / 검사 의뢰 / 교육 / 추적 계획

진료 내용: [진료 요약 또는 대화 내용 붙여넣기]
```

**9. 의뢰서 초안**
```
다음 환자의 [전문과] 의뢰서를 작성해 주세요.

포함 항목: 의뢰 이유, 현 상태 요약, 시행한 검사 결과, 요청 사항

어조: 공식적, 간결하게
길이: A4 반 페이지 이내

환자 정보: [나이/성별/진단/경과/검사 결과]
```

**10. 퇴원 요약**
```
다음 입원 경과를 바탕으로 퇴원 요약(discharge summary)을 작성해 주세요.

포함: 입원 진단, 주요 처치, 검사 결과 요약, 퇴원 시 상태, 퇴원 처방, 추적 계획

입원 경과: [경과 요약]
```

## 👨‍⚕️ 환자 커뮤니케이션 영역

**11. 환자 설명문 (일반)**
```
[질환명]에 대한 환자 교육 자료를 작성해 주세요.
대상: [나이대, 예: 60대 이상 초진 환자]
수준: 의학 지식 없음, 중학생 언어
어조: 친근하고 따뜻하게
길이: A4 1장 이내
포함: 질환 설명 / 합병증 / 생활습관 / 복약 / 응급 상황
```

**12. 나쁜 소식 전달 스크립트**
```
다음 상황에서 환자에게 나쁜 소식을 전달하는 대화 스크립트를 작성해 주세요.
SPIKES 프로토콜을 따라 주세요.

상황: [진단명 및 상황]
환자 배경: [나이/가족 동반 여부/예상 반응]

각 단계(Setting / Perception / Invitation / Knowledge / Emotion / Strategy)를
명시하고, 실제 말할 수 있는 문장으로 작성해 주세요.
```

**13. 검사 결과 설명**
```
다음 검사 결과를 환자에게 설명하는 문장을 작성해 주세요.

결과: [검사명 및 수치]
정상 범위: [범위]
의미: [임상적 의미]

조건: 의학 용어 없이, 2~3문장, 걱정하지 않도록 하되 중요성은 전달
```

## 📚 학습·연구 영역

**14. 논문 Abstract 분석**
```
다음 논문 Abstract를 분석해 주세요.

형식:
- 연구 목적 (1문장)
- 연구 설계 (RCT/코호트/메타분석 등)
- 주요 결과 (수치 포함)
- 통계적 유의성 vs 임상적 유의성
- 한계점 (2개)
- 내 진료에 적용 가능성 (1문장)

[Abstract 붙여넣기]
```

**15. 가이드라인 요약**
```
[학회명] [연도] [질환명] 가이드라인의 핵심을 정리해 주세요.

포함:
- 진단 기준
- 치료 목표 (수치)
- 1차 / 2차 치료
- 이전 가이드라인 대비 주요 변경사항
- 자주 간과하는 포인트

확실하지 않은 내용은 "확인 필요" 표시.
```

**16. 케이스 발표 준비**
```
다음 케이스를 학술 발표용으로 정리해 주세요.

구조: Chief Complaint → 병력 → 신체검진 → 초기 감별진단 →
검사 결과 → 최종 진단 → 치료 → 경과 → 토론 포인트

케이스: [케이스 내용]
발표 시간: [분]
청중: [전공의/전문의/학부생]
```

## 🔬 분석·판단 영역

**17. 두 치료법 비교**
```
[치료A] vs [치료B]를 다음 기준으로 비교해 주세요.

표 형식:
| 항목 | [치료A] | [치료B] |
비교 항목: 효능 | 안전성 | 비용 | 투여 편의성 | 특수 집단 고려 | 근거 수준

이 환자([환자 특성])에게 어느 것이 더 적합한지 결론도 제시해 주세요.
```

**18. 입원 기준 판단**
```
다음 환자의 입원 여부를 판단하는 데 도움을 주세요.

평가 도구를 적용해 주세요:
- [폐렴이면 PSI/CURB-65]
- [패혈증이면 qSOFA/SOFA]
- [기타: 관련 scoring system]

점수 계산 + 점수에 따른 권고 + 이 환자에서의 임상적 판단 포인트

환자: [정보]
```

**19. 비용 효율 분석**
```
다음 두 검사/치료 옵션의 비용 효율을 비교해 주세요.

옵션A: [이름, 비용]
옵션B: [이름, 비용]

비교 항목: 민감도·특이도 / 비용 / 환자 불편 / 추가 정보 획득 여부 /
이 환자의 사전 확률([%]) 기준 권고

베이즈 추론(사전확률 → 사후확률)도 적용 가능하면 포함해 주세요.
```

**20. 다학제 케이스 요약**
```
다음 케이스를 다학제 회의(MDT)에 제출할 형식으로 요약해 주세요.

포함: 현재 상태 요약 / 각 과에서 고려해야 할 사항 /
결정해야 할 핵심 질문 / 가능한 치료 방향 2~3가지

케이스: [케이스 내용]
참여 과: [내과/외과/영상의학/병리 등]
```

## ⚡ 빠른 참조 영역

**21~25. 초간단 원라이너**
```
# 21. 약물 단일 확인
[약물명]의 성인 표준 용량, 주요 부작용, 금기를 3줄로.

# 22. 검사 해석
[검사명] [수치]의 임상적 의미를 2문장으로.

# 23. 증상 설명
[증상]이 있는 환자에게 할 수 있는 첫 마디 3가지 옵션.

# 24. 처치 순서
[상황]에서의 초기 처치 순서를 번호로 5단계.

# 25. 한 줄 요약
[복잡한 내용]을 전공의에게 설명하듯 1문장으로 요약.
```

## 📝 문서·행정 영역

**26. 소견서 초안**
```
다음 목적의 소견서를 작성해 주세요.
목적: [예: 직장 병가, 운전 가능 여부, 장애 진단서]
환자 정보: [나이/진단/현재 상태/기능 제한]
발행 기관: [어디에 제출]
어조: 공식적, 의학적으로 정확하게
```

**27. 임상시험 적격성 확인**
```
다음 환자가 아래 임상시험에 참여 가능한지 적격성을 확인해 주세요.

임상시험 포함/배제 기준: [붙여넣기]
환자 정보: [나이/진단/병력/검사 결과/복용 약물]

각 기준을 충족/미충족/확인필요로 표시해 주세요.
```

**28. 보험 청구 코드 확인**
```
다음 진단/처치의 한국 건강보험 코드를 확인해 주세요.
KCD 코드 + 주요 처치 코드 + 자주 누락되는 추가 청구 항목

진단: [진단명]
처치: [처치 내용]

확실하지 않으면 "심평원 확인 필요"로 표시해 주세요.
```

## 🎓 교육 영역

**29. 레지던트 교육 피드백**
```
다음 전공의의 케이스 발표/처치에 대한 피드백을 작성해 주세요.

평가 항목: 임상 추론 / 근거 제시 / 안전 고려 / 환자 커뮤니케이션
형식: 잘한 점 2개 + 개선점 2개 + 핵심 학습 포인트 1개

발표/처치 내용: [내용 요약]
```

**30. 의대생 설명**
```
다음 개념을 의대 [학년] 학생에게 설명해 주세요.
비유를 1~2개 사용하고, 임상과 연결해서 마무리해 주세요.

개념: [개념명]
선수 지식: [이미 알고 있는 것]
```

## 핵심 정리
- 30개 템플릿을 메모장/노션에 저장하고 [괄호]만 채워 쓰면 된다
- 진단 / 처방 / 기록 / 교육 / 연구 / 행정 전 영역 커버
- 반복 쓰다 보면 나만의 최적화 버전으로 발전시킬 수 있다
