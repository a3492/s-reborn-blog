## 한줄 요약
좋은 프롬프트는 원하는 것을 말하는 동시에 원하지 않는 것도 명시한다 — 제약 조건이 많을수록 AI의 출력이 현실에 더 가까워진다.

## 본문

### 제약 조건이 필요한 이유

AI에게 "고혈압 치료법을 알려줘"라고 하면 교과서 전체를 요약한다. 제약이 없으면 AI는 포괄적인 답변을 생성한다. 하지만 임상에서 필요한 건 **이 환자, 이 상황, 이 기준**에 맞는 답이다.

제약 조건은 AI의 출력 공간을 좁혀서 더 유용한 답변을 만든다.

---

### 제약 조건의 4가지 유형

**1. 금지 조건 (Do NOT)**
```
- 수술적 치료는 포함하지 마세요
- 비급여 약물은 제외해 주세요
- 논문을 인용하지 마세요 (출처 날조 방지)
- 5가지 이상 나열하지 마세요
```

**2. 범위 제한 조건**
```
- 한국 건강보험 급여 기준 내에서만
- 1차 의료(의원급) 세팅에서 가능한 것으로 한정
- 입원 없이 외래에서 처치 가능한 범위로
- 소아 제외, 성인(18세 이상)만 고려
```

**3. 조건부 지시**
```
- eGFR 30 미만이면 용량 조절 방법을 반드시 포함
- 알레르기 병력이 있는 경우 대체 약물을 병기
- 임산부 가능 여부를 항상 명시
```

**4. 우선순위 조건**
```
- 비용 효율을 최우선으로 고려
- 안전성을 효능보다 우선
- 환자 순응도를 고려해 복잡한 처방보다 단순한 것 우선
```

---

### 의료 실전 예시

**외래 처방 제약 조건 조합**
```
당신은 가정의학과 전문의입니다. 다음 환자의 고혈압 처방을 제안해 주세요.

제약 조건:
1. 한국 건강보험 급여 약물만 사용
2. 1일 1회 복용 가능한 약제 우선
3. 이 환자의 CKD stage 3 (eGFR 42) 반드시 고려
4. ARB 계열에 알레르기 — 사용 금지
5. 비용은 월 5만원 이하로
6. 2제 이상 조합이 필요하면 이유를 설명

표 형식: 약물명 | 용량 | 근거 | 주의사항

환자: 62세 여성, BP 158/96 (2회 측정), DM2, CKD3, ARB 알레르기
```

**가이드라인 검색 제약 조건**
```
다음 주제에 대해 설명해 주세요.

제약:
- 대한고혈압학회 2022 가이드라인 기준으로만 (다른 학회 기준 혼용 금지)
- 수치(목표 혈압, 약물 용량)는 반드시 해당 가이드라인 페이지 또는 섹션 명시
- 확신하지 못하는 내용은 "확인 필요" 표시

주제: 노인 고혈압 치료 목표 혈압
```

### 제약 조건 작성 팁

**구체적 숫자 > 모호한 표현**
```
❌ "간단하게 설명해줘"
✅ "3문장 이내로 설명해줘"

❌ "짧게 나열해줘"
✅ "bullet point 5개 이내로"
```

**긍정형과 부정형 조합**
원하는 것(긍정)과 원하지 않는 것(부정)을 함께 명시하면 AI가 혼동하지 않는다.
```
✅ 포함: 약물 요법, 생활습관 교정
❌ 제외: 수술, 시술, 입원 치료
```

**제약 조건 우선순위 명시**
여러 제약이 충돌할 때 AI가 어느 것을 따를지 명시한다.
```
우선순위: 안전성 > 급여 여부 > 비용
만약 충돌이 있으면 안전성을 최우선으로 합니다.
```

---

### 지나친 제약의 역효과

제약 조건이 너무 많거나 서로 충돌하면 AI가 아무 것도 못 하거나 엉뚱한 답변을 한다.

```
❌ 과도한 제약 예시:
- 3문장 이내로
- 하지만 모든 감별진단을 포함하고
- 각각에 대한 검사도 제시하고
- 치료 계획까지
- 위험도 층화까지
```

→ 이 경우 제약 조건을 줄이거나 멀티턴으로 분리해서 진행하는 것이 낫다.

## 핵심 정리
- 제약 조건은 AI 출력의 범위를 좁혀 더 유용한 답을 만든다
- 금지 / 범위 제한 / 조건부 / 우선순위 — 4가지 유형을 조합해서 사용
- 제약이 지나치면 역효과. 핵심 제약 3~5개가 적당하다
