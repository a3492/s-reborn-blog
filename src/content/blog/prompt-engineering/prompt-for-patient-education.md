## 한줄 요약
환자 교육 자료는 AI가 즉시 도움이 되는 영역이다 — 질환명·대상·길이·어조를 지정하면 10분 안에 맞춤 설명문이 완성된다.

## 본문

### 왜 환자 교육 자료인가

의사가 진료 중 직접 만드는 환자 교육 자료는 드물다. 시간이 없고, 만들어도 재사용이 어렵다. AI는 이 문제를 해결한다:

- **대상 맞춤화**: 60대 어르신용, 부모 교육용, 영어 사용 환자용
- **질환별 특화**: 고혈압, 당뇨, 치료 전 주의사항 등
- **형식 선택**: A4 리플릿, 1페이지 요약, 카카오톡 메시지 형식
- **업데이트 용이**: 가이드라인 변경 시 재생성만 하면 된다

---

### 기본 환자 교육 자료 템플릿

```
당신은 환자 교육 전문 가정의학과 의사입니다.
다음 조건으로 환자 교육 자료를 작성해 주세요.

[대상 질환]: [예: 제2형 당뇨병]
[대상 환자]: [예: 60대 이상, 처음 진단받은 환자]
[이해 수준]: [예: 의학 지식 없음, 중학생 수준의 언어]
[어조]: [예: 친근하고 따뜻하게, "~하세요" 존칭]
[길이]: [예: A4 1장 이내]
[포함 항목]:
1. 이 병이 뭔가요? (2~3문장)
2. 왜 치료해야 하나요? (합병증 2~3가지)
3. 오늘부터 할 수 있는 것 (생활습관 bullet 5개)
4. 약 복용 안내
5. 언제 병원에 빨리 와야 하나요?
```

### 대상별 맞춤 프롬프트

**어르신 환자 (65세 이상)**
```
조건:
- 짧은 문장 (한 문장 15자 이내)
- 한자어·외래어 최소화
- 중요 정보는 굵게 강조
- 큰 글씨로 인쇄할 수 있게 내용 간결하게
- "어르신", "드세요", "하세요" 어투
```

**소아 부모 교육**
```
조건:
- 부모(보호자)를 대상으로 작성
- 아이 연령 [몇 세] 반영한 기준치 포함
- 응급 상황 판단 기준 명확하게 (예: 체온 몇 도 이상이면)
- 학교/어린이집 알림 여부 포함
```

**외국인 환자 (영어)**
```
Write a patient education sheet in simple English (SMOG grade 6 or below).
Topic: [disease]
Target: [age/background]
Include: diagnosis explanation, lifestyle, medication, when to return to clinic.
Use bullet points and short sentences.
```

**문해력 낮은 환자**
```
조건:
- 그림으로 설명할 수 있는 내용은 설명 후 "(그림 삽입)" 표시
- 단계별 번호로 순서 명확하게
- 핵심 메시지는 박스로 강조
- "지금 당장 해야 할 것" 섹션을 맨 앞에 배치
```

### 상황별 특수 교육 자료

**시술/검사 전 안내문**
```
[시술명: 위내시경]을 받기 전 환자에게 줄 안내문을 작성해 주세요.

포함 항목:
- 검사 목적 (1문장)
- 검사 전날 해야 할 것 (금식 포함)
- 검사 당일 주의사항
- 검사 중 느낄 수 있는 것 (불안감 완화)
- 검사 후 주의사항
- 결과 확인 방법

어조: 불안감을 줄이는 따뜻한 어투
```

**퇴원 교육**
```
다음 환자의 퇴원 교육 자료를 작성해 주세요.

환자 정보: [나이/성별/진단명/치료 내용]

포함 항목:
1. 퇴원 진단 요약 (환자가 이해할 수 있는 언어로)
2. 집에서 주의할 증상 (이게 보이면 응급실)
3. 복용 약물 (약물명 | 용도 | 복용 방법 | 주의사항) — 표 형식
4. 다음 외래 일정
5. 금기 활동 (목욕, 운동, 음주 등)
```

**만성질환 관리 카드**
```
[고혈압] 환자가 지갑에 넣고 다닐 수 있는 관리 카드를 만들어 주세요.

크기: 명함 2배 크기
포함: 목표 혈압 수치, 복용 약물, 응급 연락처, 증상 나타나면 해야 할 것
형식: 간결, 핵심만, 번호 목록
```

### 품질 체크리스트

AI가 만든 환자 교육 자료를 배포 전 확인할 항목:

- [ ] 의학 용어를 과도하게 사용하지 않았는가?
- [ ] 수치(혈압 목표치, 혈당 범위, 약물 용량)가 최신 가이드라인과 맞는가?
- [ ] 응급 상황 기준이 명확하게 표시됐는가?
- [ ] 대상 환자의 언어·문화에 적합한가?
- [ ] 병원 연락처가 포함됐는가?
- [ ] 개인정보(환자명)가 포함되지 않았는가? (템플릿으로 쓸 경우)

## 핵심 정리
- 대상(나이·문해력)·길이·어조·포함 항목을 지정하면 맞춤 교육 자료가 10분 안에 완성
- 시술 전 안내문, 퇴원 교육, 만성질환 관리 카드 등 형식별 특화 템플릿 활용
- 배포 전 수치와 가이드라인은 반드시 교차 확인
