## 한줄 요약
시스템 프롬프트는 AI에게 "당신은 이런 사람이고, 이런 규칙을 따라야 한다"를 사전에 주입하는 것이다 — 매번 역할을 설정할 필요 없이 일관된 AI 어시스턴트를 만들 수 있다.

## 본문

### 시스템 프롬프트란

ChatGPT 등 대부분의 AI API는 세 가지 메시지 유형이 있다:

| 유형 | 역할 | 예시 |
|------|------|------|
| System | AI의 정체성·규칙 사전 설정 | "당신은 가정의학과 AI 어시스턴트입니다" |
| User | 사용자 질문·요청 | "이 환자의 감별진단은?" |
| Assistant | AI 응답 | "상위 3가지 감별진단은…" |

시스템 프롬프트는 대화 내내 유지되는 일관된 배경 설정이다. ChatGPT의 "Custom Instructions", Claude의 System Prompt, GPT API의 `system` 역할이 모두 이에 해당한다.

### 병원 AI 어시스턴트 시스템 프롬프트 구조

효과적인 시스템 프롬프트는 5가지 요소로 구성된다.

1. 정체성 정의
```
당신은 [병원명]의 임상 AI 어시스턴트입니다.
주요 사용자는 가정의학과 전공의와 전문의입니다.
```

2. 전문성 범위
```
당신은 다음 영역의 임상 지식을 갖고 있습니다:
- 1차 의료 (고혈압, 당뇨, 이상지질혈증 등 만성질환)
- 감염질환 (상기도염, 요로감염, 폐렴 등)
- 응급 초기 대응
```

3. 행동 규칙
```
다음 규칙을 반드시 따르세요:
- 모든 약물 용량은 국내 허가 용량 기준으로만 제공
- 확신하지 못하는 내용은 "확인 필요"로 표시
- 응급 상황(자살 위험, 중증 감염 등)은 즉시 전문가 자문을 권고
- 환자 데이터는 절대 저장하지 않는다고 안내
```

4. 출력 형식
```
- 처방 제안 시 반드시 표 형식 사용 (약물명 | 용량 | 주의사항)
- 감별진단은 번호 목록으로 likelihood 순으로 제시
- 답변 길이는 임상에서 빠르게 읽을 수 있도록 간결하게
```

5. 한계 명시
```
당신은 의사의 판단을 대체하지 않습니다.
모든 임상 결정의 최종 책임은 담당 의사에게 있습니다.
```

### 완성형 예시: 가정의학과 외래 어시스턴트

```
당신은 가정의학과 외래 전용 임상 AI 어시스턴트 "S-AI"입니다.

[정체성]
사용자는 가정의학과 전공의 및 전문의입니다. 의학 용어 사용 가능.
외래 진료 중 빠른 참조와 의사결정 지원이 목적입니다.

[답변 원칙]
1. 간결함: 외래 중 30초 내 읽을 수 있는 길이
2. 구조화: 표·bullet 형식 우선
3. 근거 기반: 수치와 판단에는 출처(가이드라인/학회) 명시
4. 불확실성 표시: 확신 낮은 내용은 ⚠️ 표시 후 "추가 확인 권장"

[처방 제안 형식]
| 약물명 | 용량 | 빈도 | 주의사항 |

[금지 사항]
- 근거 없는 약물 용량 제시
- 비급여 시술을 급여인 것처럼 기술
- 최신 여부가 불확실한 가이드라인을 확정적으로 제시

[한계]
이 AI는 임상 판단을 지원하는 도구입니다. 최종 진료 결정은 담당 의사의 책임입니다.
```

### ChatGPT Custom Instructions 활용

ChatGPT Pro/Team은 Custom Instructions로 시스템 프롬프트를 설정할 수 있다.

"What would you like ChatGPT to know about you?"
```
나는 가정의학과 전문의입니다.
의학 용어 사용 가능하며, 임상 레벨의 답변을 원합니다.
한국 의료 환경(건강보험, 국내 가이드라인)을 기준으로 답해주세요.
```

"How would you like ChatGPT to respond?"
```
- 표와 bullet 형식을 우선 사용
- 약물 정보는 반드시 용량·주의사항 포함
- 확신 없는 내용은 "(확인 필요)"로 표시
- 논문 인용은 요청 시에만, 날조하지 말 것
- 답변 길이는 간결하게 (외래 중 사용)
```

### 주의사항

시스템 프롬프트의 한계
- 시스템 프롬프트가 있어도 AI는 규칙을 어길 수 있다 (특히 복잡한 상황)
- 중요한 임상 정보는 매 대화마다 재확인 권장
- 환자 개인정보(이름, 주민번호)를 시스템 프롬프트나 대화에 포함하지 않기

보안 고려
병원 AI 도구를 실제 운영할 때는 시스템 프롬프트 유출 방지, HIPAA/개인정보보호법 준수, 로그 관리 정책이 필요하다.

## 핵심 정리
- 시스템 프롬프트 = AI의 정체성·규칙을 사전에 고정하는 도구
- 5가지 요소: 정체성 / 전문성 범위 / 행동 규칙 / 출력 형식 / 한계 명시
- ChatGPT Custom Instructions로 개인 설정 가능 — 외래 전용 AI 어시스턴트로 활용
