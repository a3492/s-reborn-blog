## 한줄 요약
역할 설정은 AI에게 답변의 관점·깊이·어조를 지정하는 것이다 — "전문의처럼" vs "일반인용"은 완전히 다른 답변을 만든다.

## 본문

### 역할 설정이 중요한 이유

AI는 기본 상태에서 모든 사람을 위한 '평균적인' 답변을 생성한다. 고혈압에 대해 물으면 환자 교육 수준의 답변이 나온다. 의사가 원하는 답변 — 구체적인 약제 선택 근거, JNC 8 기준, 특수 상황 고려 — 은 기본적으로 나오지 않는다.

역할 설정은 AI가 어떤 **관점**에서, 어떤 **깊이**로, 어떤 **어조**로 답할지를 지정한다.

---

### 역할 설정 전·후 비교

**질문**: 당뇨 환자의 혈당 목표를 알려줘.

**역할 없음**
```
혈당 목표에 대해 알려줘.
```
→ "공복혈당 126 이상이면 당뇨입니다. 식이요법과 운동이 중요합니다…" (일반인 대상)

**역할 설정 후**
```
당신은 내분비내과 전문의입니다. 다음 환자의 혈당 관리 목표를 설정해 주세요.

환자: 72세 남성, T2DM 10년, eGFR 45, 저혈당 인식장애 있음.
ADA 2024 가이드라인 기준으로 HbA1c 목표와 이유를 설명해 주세요.
```
→ "저혈당 인식장애 + CKD stage 3b를 고려할 때 HbA1c 목표는 8% 미만이 적절합니다. 이는 ADA 2024의 고위험 환자 기준에 해당하며…" (전문의 수준)

---

### 역할 설정 문장 패턴

**기본형**
```
당신은 [전문과] 전문의입니다.
```

**심화형** — 경험과 컨텍스트 추가
```
당신은 3차 병원의 [전문과] 교수입니다.
레지던트를 교육하듯 근거와 함께 설명해 주세요.
```

**독자 설정형** — 대상에 맞는 설명
```
당신은 환자 교육 전문 간호사입니다.
중학생이 이해할 수 있는 언어로 설명해 주세요.
```

**멀티 역할형** — 다른 관점 비교
```
가정의학과 의사와 심장내과 의사의 관점에서
각각 어떻게 이 환자를 접근할지 비교해 주세요.
```

---

### 전문과별 역할 설정 예시

| 목적 | 역할 설정 |
|------|-----------|
| 감별진단 | "당신은 진단검사의학과 전문의입니다" |
| 약물 처방 | "당신은 임상약리학 전문의입니다" |
| 수술 여부 결정 | "당신은 외과 교수입니다" |
| 환자 설명 | "당신은 환자 커뮤니케이션 전문 의사입니다" |
| 논문 비평 | "당신은 근거중심의학 전문가입니다" |
| 법적 판단 | "당신은 의료법 전문 변호사입니다" |
| 건강보험 기준 | "당신은 건강보험심사평가원 심사위원입니다" |

---

### 역할 설정의 흔한 실수

**너무 광범위한 역할**
```
❌ "당신은 의사입니다."
✅ "당신은 응급의학과 전문의이며, 응급실 교수입니다."
```

**역할과 요청이 불일치**
```
❌ "당신은 소아과 전문의입니다. 노인 당뇨 치료법을 알려줘."
✅ "당신은 내분비내과 전문의입니다. 노인 당뇨 치료법을 알려줘."
```

**역할 설정 후 확인 안 함**
AI가 설정한 역할을 벗어난 답변을 줄 때가 있다. 중요한 내용이라면 "지금 답변이 [전문과] 전문의 관점에서 맞나요?" 로 교차 확인할 것.

---

### 고급: 역할 + 제약 조건 조합

역할만 설정하면 부족할 때가 있다. 역할에 제약 조건을 더하면 더 정확한 답변이 나온다.

```
당신은 가정의학과 전문의입니다.
다음 제약 조건을 반드시 지켜주세요:
- 한국 건강보험 급여 기준 내에서 처방
- 최신 대한고혈압학회(2022) 가이드라인 기준
- 이 환자의 신기능 저하(eGFR 40)를 고려
- 처방 시 용량 조절 필요 여부를 반드시 명시

환자: [정보]
```

## 핵심 정리
- 역할 설정 = AI에게 답변의 관점·깊이·어조를 지정하는 것
- "당신은 [전문과] 전문의입니다" 한 줄이 답변 품질을 결정한다
- 역할은 구체적일수록 좋다. 역할 + 제약 조건 조합으로 더 정밀하게
