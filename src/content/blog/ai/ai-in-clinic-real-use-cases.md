## 한줄 요약
AI는 진단을 대체하지 않지만, 진료의 모든 단계에서 생산성을 2배로 높여준다.

## 본문

### 왜 진료실에서 AI를 쓰는가

하루 외래 환자 30~50명. 한 환자당 5~10분. 이 시간 안에 문진, 진찰, 검사 해석, 처방, 교육을 모두 해야 한다.

AI는 이 과정에서 **반복적이고 시간이 걸리는 작업**을 빠르게 처리해준다. 아래는 실제로 진료실에서 쓰고 있는 7가지 방법이다.

### 1. 감별진단 체크리스트

환자의 증상이 여러 질환과 겹칠 때, AI에게 감별 목록을 요청한다.

```
48세 여성, 2주간 피로감 + 체중 증가(3kg) + 변비
TSH 정상, CBC 정상
→ 가능성 높은 감별진단 5개와 각각의 추천 검사
```

**핵심**: AI 결과를 **체크리스트**로 사용한다. "내가 놓친 게 있나?" 확인용이지, AI의 순위를 맹신하지 않는다.

### 2. 검사 결과 해석 보조

복잡한 혈액 검사 결과를 패턴으로 분석할 때:

```
[검사 결과]
AST 78, ALT 125, GGT 45, ALP 정상
TB 1.2, Albumin 3.8
HBsAg(-), Anti-HCV(-)
→ 간수치 상승 패턴 분석 및 추가 검사 제안
```

**핵심**: 특히 여러 수치가 동시에 변할 때 패턴 인식에 유용하다. 단, 수치의 정확성은 반드시 원본과 대조한다.

### 3. 환자 교육 자료 즉석 제작

"이 병이 뭐예요?"라는 질문에 설명하느라 5분을 쓰는 대신:

```
제2형 당뇨병 환자 교육 자료.
조건: 60대가 이해할 수준, A4 반 장,
포함: 식이요법 핵심 3가지, 운동 권고, 저혈당 대처법
```

→ 생성된 자료를 검토 후 인쇄해서 환자에게 제공. 설명 시간이 절반으로 줄어든다.

### 4. 의학 영어 논문 빠른 요약

최신 논문을 읽을 시간이 부족할 때:

```
아래 Abstract를 한국어로 요약해 줘.
형식: 목적/방법/결과/결론, 각 2문장 이내.
임상적 의의를 한 줄로 추가.
[Abstract 붙여넣기]
```

**핵심**: 전문을 읽기 전 "이 논문이 내 진료에 관련이 있는가?"를 1분 안에 판단할 수 있다.

### 5. 처방 상호작용 더블체크

다제 복용 환자의 약물 상호작용을 확인할 때:

```
현재 복용약:
- Warfarin 3mg
- Amiodarone 200mg
- Omeprazole 20mg
새 처방 예정: Fluconazole 150mg
→ 약물 상호작용 확인 및 주의사항
```

**주의**: AI의 답변은 **1차 스크리닝**이다. 최종 확인은 반드시 약전이나 Lexicomp/UpToDate에서 한다. 약물 상호작용은 환각이 발생하면 위험하다.

### 6. 진료 기록 초안 작성

진료 후 SOAP 노트 초안을 빠르게 만들 때:

```
아래 내용으로 SOAP 노트를 작성해 줘.
- 45세 남성, 3일 전부터 우측 무릎 통증, 계단 내려갈 때 악화
- 부종(+), 열감(-), ROM 제한 없음
- X-ray: 경미한 퇴행성 변화
- 소견: OA R/O meniscal tear
- 계획: NSAIDs 처방, MRI 의뢰 고려
```

**핵심**: 초안을 AI가 쓰고, 의사가 검토·수정한다. 기록 시간이 70% 줄어든다.

### 7. 컨퍼런스·발표 준비

학회 발표나 원내 컨퍼런스 슬라이드 구성:

```
"비만 환자의 GLP-1 수용체 작용제 사용" 주제로
15분 발표 슬라이드 구성안을 만들어 줘.
대상: 가정의학과 전공의
포함: 기전, 적응증, 주요 임상시험 3개, 부작용, 실전 팁
```

→ 구성안을 받아서 내용을 채우면 준비 시간이 절반 이하로 줄어든다.

### 사용 시 반드시 지켜야 할 원칙

| 원칙 | 이유 |
|------|------|
| **환자 정보 비식별화** | 이름, 등록번호 절대 입력 금지. "48세 여성"으로 충분 |
| **AI 답변 = 초안** | 최종 판단은 항상 의사가. AI는 보조 도구 |
| **숫자 교차 확인** | 약물 용량, 검사 기준값은 반드시 원전 대조 |
| **환각 경계** | 특히 약물 상호작용, 논문 인용에서 환각 빈번 |
| **기록 남기기** | AI 활용 사실을 진료 기록에 명시 (법적 보호) |

## 핵심 정리
- AI는 감별진단, 검사 해석, 교육자료, 논문 요약, 처방 체크, 기록 작성, 발표 준비에 활용 가능
- 모든 출력은 "초안"으로 취급하고 의사가 최종 검증
- 환자 개인정보는 절대 AI에 입력하지 않는다
